Настройка обработки оформления излишков и недостач товаров

Перед тем, как начинать работу с обработкой, необходимо проверить выполнение следующих условий.

  • Пересчет товаров в рамках приказа на инвентаризацию должен быть закончен. Во всех документах пересчета должен быть установлен статус "Выполнен".
  • По результатам пересчета на ордерном складе должен быть оформлен документ "Ордер на отражение результатов пересчета товаров".
  • Приказ на инвентаризацию должен быть закрыт. Дата закрытия приказа должна быть меньше, чем дата оформляемых складских актов.

Как указать список организаций, по которым должны быть оформлены финансовые документы по результатам регистрации излишков и недостач товаров?

Список организаций, по которым должны быть распределены выявленные излишки и недостачи товаров и оформлены финансовые документы (складские акты) указывается в поле "Организации". При открытии обработки этот список заполняется списком всех организаций, зарегистрированных в программе. Для изменения списка организаций выполните следующие действия.

  • Нажмите на кнопку выбора в поле "Организации".
  • Появится диалоговое окно "Список организаций".
  • Если Вы хотите удалить организацию из списка, то установите курсор на организацию, которую Вы хотите удалить и нажмите на кнопку Del (или на кнопку удаления в списке организаций).
  • Для добавления новой организации используйте кнопку "Добавить".
  • После редактирования списка организаций нажмите на кнопку "ОК".
  • Отредактированный список организаций будет перенесен в обработку "Помощник оформления складских актов".

Можно ли ограничить список товаров, по которым надо оформлять излишки и недостачи товаров, например отобрать только товары, принадлежащие к одной группе?
  • Нажмите на кнопку выбора рядом с полем "Отбор по номенклатуре".
  • В появившемся диалоговом окне "Редактирование отбора" введите нужные параметры отбора. Например укажите следующие параметры "Номенклатура В группе "Бытовая техника"". После установки параметров отбора нажмите на кнопку "ОК".
  • Параметры отбора будут перенесены в обработку.
  • Нажмите на кнопку "Заполнить".
  • Список товаров для оформления излишков и недостач товаров будет уточнен в соттветствии с указанными параметрами отбора.

Распределение излишков и недостач товаров по организациям

Перед распределением излишков и недостач товаров по организациям необходимо выявить список тех товаров, которые можно зачесть, как пересортицу товаров. После этого необходимо произвести распределение пересортиц товаров, а также излишков и недостач товаров по организациям.

Оформление порчи товаров

Как в помощнике оформить бракованый товар обнаруженный в результате пересчета?

Для этого необходимо воспользоваться командой "Зачесть по порче (все)". Далее программа автоматически зачитывает недостачи качественных товаров излишками некачественных. Результат этой операции будет отражен на странице "Порча".

Важно. Автоматический зачет по порче не сработает если:
- связи между товарами различного качества не установлены в регистре "Товары другого качества".
- если у качественного товара не ведется учет по характеристикам, а у некачественного ведется (в обратной ситуации автоматический зачет сработает).
- если товары разного качества относятся к разным видам номенклатуры и при этом у этих видов номенклатуры назначено ведение общих характеристик.
Если товары разного качества относятся к одному виду номенклатуры, или если у разных видов номенклатуры назначено ведение индивидуальных характеристик, то программа попытается сопоставить характеристики по наименованиям.

Можно ли выполнить зачет бракованных товаров в ручном режиме?

Зачет товаров можно произвести и вручную, анализируя список товаров к списанию и оприходованию. Для этого выполните следующую последовательность действий:

  • В списке товаров "Товары к списанию" установите курсор на тот товар, который необходимо списать по порче.
  • Аналогичным образом установите курсор на бракованный товар в списке "Товары к оприходованию".
  • Воспользуйтесь командой "Зачесть" - "По порче".

Оформление пересортицы товаров

Может ли программа подсказать список тех товаров, по которым вероятнее всего необходимо зафиксировать факт пересортицы?

При анализе излишков и недостач программа анализирует список товаров и формирует список тех товаров, которые могут быть оформлены в качестве пересортицы. Критерием обнаружения пересортицы товаров, является то, что один и тот же товар, имеющий различные характеристики попадает одновременно в в два списка: "Товары к списанию" и "Товары к оприходованию".

Пример. В результате проведения инвентаризации товаров обнаружилось, что в излишках ("Товары к оприходованию" ) обнаружен товар "Ботинки" с характеристикой "41 размер, цвет белый". Одновременно с этим зарегистрирована недостача товара "Ботинки" с характеристикой "41 размер, цвет красный". Товар "Ботинки" будет указан в качестве того товара, по которому можно оформить документ "Пересортица товаров".

Список тех товаров, которые можно зачесть по пересортице показывается в поле "Показать номенклатуру".

Как произвести зачет товаров по пересортице?
  • В списке "Показать номенклатуру" выберите тот товар, который необходимо зачесть по пересортице.
  • В списках "Товары к оприходованию" и "Товары к списанию" будет показан только тот товар, который можно зачесть по пересортице.
  • Нажмите на кнопку "Зачесть по пересортице" .
  • Будет показано отдельное диалоговое окно, в котором необходимо указать то количество товаров, которое можно зачесть по пересортице.
  • Уточните количество товаров, которое можно зачесть по пересортице, в поле "К зачету всего".
  • Нажмите на кнопку "ОК".

На странице "Пересортица" появится информация о тех товарах, которые можно зачесть по пересортице. Аналогичным образом оформите операцию пересортицы по остальным товарам.

Зачет товаров можно произвести и вручную, анализируя список товаров к списанию и оприходованию. Если в процессе анализа списка товаров к списанию и оприходованию были выявлены товары, которые можно зачесть по пересортице, то выполните следующую последовательность действий.

  • В списке товаров "Товары к списанию" установите курсор на тот товар, который необходимо зачесть по пересортице.
  • Аналогичным образом установите курсор на товар, который необходимо зачесть по пересортице в списке "Товары к оприходованию".
  • Нажмите на кнопку "Зачесть" - "По пересортице".
  • Укажите то количество, которое необходимо зачесть по пересортице.

Совет. При оформлении пересортицы товаров обращайте внимание на цену товаров, которые зачитываются по пересортице. Цена товаров показывается для каждой организации. В качестве цены товаров показывается значение той цены, которая определена в качестве учетной цены для данного склада. Данные об учетной цене берутся на момент оформления операции пересортицы.

Важно. Нельзя зачесть по пересортице товары, для которых указаны различные ставки НДС.

Как распределить недостачи товаров по организациям?
  • В обработке "Помощник оформления складских актов" перейдите на страницу "К оформлению". В списке "Товары к списанию" будут показаны те товары , по которым была зарегистрирована недостача. В нижней части показывается список тех организаций, которые были указаны в настройке обработки.

Установив курсор на строку с товаром, по которым регистрируется списание можно посмотреть остаток товаров к списанию, который зарегистрирован по каждой организации и цену, по которой будет списан данный товар. В качестве цены товара показывается расчитанная на момент оформления документа списания себестоимость товара по данной организации. Распределение товаров по организациям можно произвести автоматически или вручную.

Для автоматического распределения товаров по организациям выполните следующие действия.

  • Выделите в списке "Товары к списанию" те записи о товарах, по которым зарегистрирована недостача, и которые необходимо распределить по организациям. Для выделения всех записей в списке используйте сочетание клавиш CTRL+A.
  • Нажмите на кнопку "Распределить по организациям".
  • В появившемся диалоговом окне укажите тот порядок, по которому должны формироваться документы по организациям.
  • Распределение товаров по организациям будет производиться последовательно в соответствии с зарегистрированными остатками товаров организаций.
  • Информация о распределении товаров по организациям будет показана в списке организаций в поле "К списанию". Также для каждого товара будет показана информация об остатках списанного товара по организациям.

Пример. В результате проведения пересчета товаров была зарегистрирована недостача товара "Ботинки" в количестве 25 пар. В программе зарегистрированы остатки товара "Ботинки" по всем организациям (Торговый дом, ИЧП и Стройкомплекс) в количестве 10 пар. Необходимо в первую очередь списать товар с ИЧП, потом по организации Торговый дом и только затем по организации Стройкомплекс. В этом случае при распределении остатков по организациям указываем следующие приоритеты (порядок) списания: 1 -ИЧП, 2-Торговый дом, 3 -Стройкомплекс. В результате проведения операции распределения товаров по организациям будет зарегистрировано распределение списания товара "Ботинки" с организации ИЧП в количестве 10 пар, с организации Торговый дом в количестве 10 пар, с организации Стройкомплекс в количестве 5 пар.

Важно. Информация в поле "К списанию" может быть откорректирована вручную. Корректировка этого поля возможна только для тех организаций, у которых имеются положительные остатки товара, который необходимо списать при оформлении недостач товаров.

Информация о результатах распределения товаров по организациям будет зафиксирована на странице "Документы". На основании этой информации в дальнейшем будут регистрироваться финансовые документы по списанию товаров.

Как отнести выявленные излишки товаров на конкретные организации?

В обработке "Помощник оформления складских актов" перейдите на страницу "К оформлению". В списке "Товары к оприходованию" будут показаны те товары, по которым были зарегистрирована излишки. В нижней части показывается список тех организаций, которые были указаны в настройке обработки.

Установив курсор на строку с товаром, по которым регистрируется оприходование можно посмотреть остаток к оприходованию, который зарегистрирован по каждой организации. Распределение излишков товаров можно произвести автоматически или вручную.

Для автоматического отнесения излишков товаров пна конкретную организацию выполните следующие действия.

  • Выделите в верхнем списке те записи о товарах, по которым зарегистрирован излишек и которые необходимо отнести на конкретную организацию.
  • Нажмите на кнопку "Распределить по организациям".
  • В появившемся диалоговом окне выберите ту организацию, на которую необходимо отнести излишек товаров и нажмите на кнопку "ОК".
  • Аналогичным образом отнесите излишки товаров на другую организацию
  • Информация о распределении товаров по организациям будет показана в списке организаций в поле "К оприходованию". Также для каждого товара будет показана информация об остатках оприходованного товара после распределения товаров по организациям.

Совет. Если все излишки товаров необходимо отнести на одну организацию, то для выделения всех товаров в списке используйте сочетание клавиш CTRL+A. Если излишек одного и того же товара должен быть отнесен к разным организациям в определенных количествах, то используйте возможность ручного распределения. Для каждого товара укажите соответствующее количество по каждой организации в поле "К оприходованию".

Информация о результатах распределения излишков товаров по организациям будет зафиксирована на странице "Документы". На основании этой информации в дальнейшем будут регистрироваться финансовые документы по оприходованию товаров.

Как распределить товары, по которым зарегистрирована пересортица по организациям?

В обработке "Помощник оформления складских актов" перейдите на страницу "Пересортица". В списке будут показаны те товары , по которым была зарегистрирована пересортица. В списке "Товар к списанию" будет показан тот товар, который должен быть списан в результате пересортицы, а в списке "Товар к оприходованию" будет показан тот товар, который необходимо оприходовать при оформлении пересортицы. В нижней части показывается список тех организаций, которые были указаны в настройке обработки.

Установив курсор на строку с товарами, по которым регистрируется переоценка можно посмотреть остаток товаров к списанию и к оприходованию который зарегистрирован по каждой организации. Распределение товаров по организациям можно произвести автоматически или вручную.

Для автоматического распределения товаров по организациям выполните следующие действия.

  • Выделите в верхнем списке те записи о товарах, по которым зарегистрирована пересортица, и которые необходимо распределить по организациям. Для выделения всех записей в списке используйте сочетание клавиш CTRL+A.
  • Нажмите на кнопку "Распределить по организациям".
  • В появившемся диалоговом окне укажите тот порядок, по которому должны формироваться документы по организациям.
  • Распределение товаров по организациям будет производиться в соответствии с зарегистрированными остатками товаров организаций. То есть недостача товаров, которая выявлена в результате пересортицы, будет отнесена на ту организацию, у которой есть остаток данного товара.
  • Информация о распределении товаров по организациям будет показана в списке организаций в поле "К зачету". Также для каждого товара будет показана информация об остатках списанного товара и оприходованного товара после распределения товаров по организациям.

Важно. Информация в поле "К зачету" может быть откорректирована вручную. Корректировка этого поля возможна только для тех организаций, у которых имеются положительные остатки товара, который необходимо списать при оформлении операции пересортицы.

Информация о результатах распределения товаров по организациям будет зафиксирована на странице "Документы". На основании этой информации в дальнейшем будут регистрироваться финансовые документы по пересортице товаров.

Как распределить товары, по которым зарегистрирована порча по организациям?

В обработке "Помощник оформления складских актов" перейдите на страницу "Порча". В списке будут показаны те товары , по которым была зарегистрирована порча. В списке в поле"Товар исходного качества" будет показан тот товар, который должен быть списан в результате порчи, а в поле "Испорченный товар" будет показан тот товар, который необходимо оприходовать при оформлении порчи. В нижней части показывается список тех организаций, которые были указаны в настройке обработки.

Установив курсор на строку с товарами, по которым регистрируется порча можно посмотреть остаток товаров к списанию и к оприходованию который зарегистрирован по каждой организации. Распределение товаров по организациям можно произвести автоматически или вручную.

Для автоматического распределения товаров по организациям выполните следующие действия.

  • Выделите в верхнем списке те записи о товарах, по которым зарегистрирована порча, и которые необходимо распределить по организациям. Для выделения всех записей в списке используйте сочетание клавиш CTRL+A.
  • Нажмите на кнопку "Распределить по организациям".
  • В появившемся диалоговом окне укажите тот порядок, по которому должны формироваться документы по организациям.
  • Распределение товаров по организациям будет производиться в соответствии с зарегистрированными остатками товаров организаций. То есть недостача товаров, которая выявлена в результате порчи, будет отнесена на ту организацию, у которой есть остаток данного товара.
  • Информация о распределении товаров по организациям будет показана в списке организаций в поле "К зачету". Также для каждого товара будет показана информация об остатках списанного товара и оприходованного товара после распределения товаров по организациям.

Важно. Информация в поле "К зачету" может быть откорректирована вручную. Корректировка этого поля возможна только для тех организаций, у которых имеются положительные остатки товара, который необходимо списать при оформлении операции порчи.

Информация о результатах распределения товаров по организациям будет зафиксирована на странице "Документы". На основании этой информации в дальнейшем будут регистрироваться финансовые документы по порче товаров.

Оформление финансовых документов

Финансовые документы списания, оприходывания и пересортицы товаров оформляются по результатам пересчетов товаров. Документы должны быть оформлены после регистрации всех результатов пересчета. Если инвентаризация оформляется на ордерном складе, то должны быть оформлены ордера на оприходование излишков и недостач товаров. Финансовые документы должны быть оформлены датой, большей, чем дата окончания периода, указанная в приказе на проведение инвентаризации.

Важно. Если склад разделен на помещения, то финансовые документы оформляются отдельно по каждому помещению.

Как оформить документы списания товаров после распределения недостач товаров по организациям?
  • В обработке "Помощник оформления складских актов" перейдите на страницу "Документы".
  • В списке оформления операций установите курсор на строку, в которой указан тип операции "Списание".
  • Для данной операции будет показана оценочная стоимость списанных товаров ("Стоимость (оценка)"). Оценочная стоимость формируется в соответствии с тем видом цен, который указан в карточке склада в качестве учетного вида цен.
  • В правой части диалогового окна обработки будет показан список списываемых товаров и оценочная стоимость по каждому списываемому товару.
  • Нажмите на кнопку "Оформить операцию".

Будет оформлен новый документ "Списание недостач товаров". В документе необходимо заполнить информацию о статье расходов и подразделении, на которое будут отнесены расходы по списанию товаров. Аналитика расходов заполняется в соответствии с тем типом аналитики, которая определена для статьи расходов.

Проведите документ и зарегистрируйте результаты списания товаров.

Важно. Товары, зарегистрированные в документе "Списание недостач товаров" будут списаны по себестоимости, рассчитанной в соответствии с алгоритмом расчета себестоимости, установленной для организации. При печати документа в качестве цены списания будет указана та цена, которая зарегистрирована в карточке склада (помещения) в качестве учетной цены для оценки товаров на складе. Информация об этом виде цен должна быть зарегистрирована документом "Установка цен номенклатуры".

Как оформить документы оприходования товаров после распределения выявленных излишков товаров по организациям?
  • В обработке "Помощник оформления складских актов" перейдите на страницу "Документы".
  • В списке оформления операций установите курсор на строку, в которой указан тип операции "Оприходование".
  • Для данной операции будет показана оценочная стоимость оприходываемых товаров ("Стоимость (оценка)"). Для расчета оценочной стоимости используется та цена, которая зарегистрирована в карточке склада (помещения) в качестве учетной цены для оценки товаров на складе.
  • В правой части диалогового окна обработки будет показан список оприходываемых товаров и оценочная стоимость по каждому каждого оприходываемого товара.
  • Нажмите на кнопку "Оформить операцию".

Будет оформлен новый документ "Оприходование излишков товаров". В документе необходимо заполнить информацию о статье доходов и подразделении, на которое будут отнесены доходы, возникшие при оприходовании товаров. Аналитика доходов заполняется в соответствии с тем типом аналитики, которая определена для статьи доходов. При создании документа оприходования, цены в нем заполнятся по виду цен, указанному в документе в поле "Вид цены".

Проведите документ и зарегистрируйте результаты оприходования товаров.

Как оформить документы по пересортице товаров после распределения выявленных пересортиц товаров по организациям?
  • В обработке "Помощник оформления складских актов" перейдите на страницу "Документы".
  • В списке оформления операций установите курсор на строку, в которой указан тип операции "Пересортица".
  • В правой части диалогового окна обработки будет показан список оприходываемых и списываемых товаров и оценочная стоимость для каждого товара.
  • Нажмите на кнопку "Оформить операцию".

Будет оформлен новый документ "Пересортица товаров". В документе будет заполнена информация о списываемых товарах (Списываемый товар ) и приходуемых товарах (Приходуемый товар)), в том числе и цены приходуемых товара.
В результате оформления этой операции один товаров будет списан по текущей себестоимости, а второй товар будет оприходован по указанной цене. После выполнения операции может возникнуть или доход или расход. Получившийся доход или расход будет отнесен на статью дохода или расхода которые необходимо указать на странице "Дополнительно". Там же необходимо указать (если требуется) аналитику статей.
Если в документе установить флажок "Приходовать товары по себестоимости списания", то в результате оформления этой операции один из товаров будет списан по текущей себестоимости, а второй будет оприходован по той же цене. Никаких доходов и расходов при этом не возникнет.

Проведите документ и зарегистрируйте результаты пересортицы товаров.

Как оформить документы по порче товаров после распределения выявленных бракованных товаров по организациям?
  • В обработке "Помощник оформления складских актов" перейдите на страницу "Документы".
  • В списке оформления операций установите курсор на строку, в которой указан тип операции "Порча".
  • В правой части диалогового окна обработки будет показан список оприходываемых и списываемых товаров и оценочная стоимость для каждого товара.
  • Нажмите на кнопку "Оформить операцию".

Будет оформлен новый документ "Порча товаров". В результате оформления документа один из товаров будет списан по текущей себестоимости, а второй товар будет оприходован по указанной цене. После проведения документа может возникнуть расход. Получившийся расход будет отнесен на статью расхода которую необходимо указать в поле "Статья расходов". В поле "Аналитика расходов" (если требуется) необходимо указать аналитику статьи.

Важно. В программе не предусмотрено возникновение дохода в процессе оформления брака, но при этом документ "Порча товаров" позволяет указать цену бракованного товара выше, цены качественного. При оформлении такой операции сформированный доход будет записан как расход со знаком минус.
