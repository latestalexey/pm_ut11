Форма рабочего места

Использование мобильного рабочего места работника склада позволяет быстро оформлять операции по отбору, размещению, перемещению и пересчету товаров с использованием терминала сбора данных. В качестве терминала сбора данных могут использоваться те модели, которые могут быть подключены в программе.

Модель работы следующая. С терминала сбора данных запускается терминальная сессия на сервере под управлением ОС Windows (XP, Vista, Windows 7, Server 2000/2003/2008), в этой терминальной сессии открывается сеанс работы с информационной базой 1С:Предприятия, и при начале работы с информационной базой запускается обработка – мобильное рабочее место работника склада. Соответственно, у той учетной записи под которой выполняется подключение к терминалу должны быть права на запуск 1С:Предприятия на терминальном сервере. Для лучшего отображения формы обработки рекомендуется установить опцию Автоматически скрывать у панели задач на терминальном сервере. Используется модель «подключенного» рабочего места, т.е. без наличия связи с информационной базой рабочее место использовать невозможно. Терминал сбора данных должен иметь установленный клиент RDP (Remote Desktop Connection). Рабочее место запускается с дополнительным параметром запуска. Причем параметр запуска должен указываться с ключом «/C». Т.е. в поле Дополнительный параметр запуска должно быть установлено значение:/CWarehouseMobileWorkspace.

Мобильное рабочее место работника склада может использоваться на складе (помещении) с адресным хранением товаров. Для работника склада оформляется соответствующее задание на отбор или размещение товаров (документ Отбор (размещение) товаров) или задание на пересчет товаров в рамках проводимой инвентаризации (документ Пересчет товаров). В процессе работы программа показывает список тех заданий, которые необходимо выполнить (отбор, размещение, перемещение или пересчет). В процессе выполнения задания программа последовательно указывает какое действие необходимо выполнить – отсканировать ячейку, отсканировать товар, отсканировать серию товара и и показывает какой товар необходимо отсканировать и в какую ячейку его необходимо положить (переместить, отобрать, пересчитать). Перед началом использования мобильного рабочего места работника склада необходимо правильно заполнить всю информацию для правильного формирования заданий на адресном складе. Кроме этого необходимо провести дополнительные настройки для подключения пользователей, которые будут использовать в работе мобильное рабочее место.

Важно. Пользователи, которые будут использовать мобильное рабочее место работника склада должны быть включены в группу доступа с профилем доступа Работники склада.
