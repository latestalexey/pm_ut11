Основная форма

Обработка предназначена для загрузки в информационную базу данных по операциям эквайринга, направляемых банком-эквайером в виде отчета торговому предприятию.

Сценарий работы.

  • Выбрать XML файл с данными, содержащий отчет банка по операциям эквайринга.
  • Выбрать статью расходов, на которую будет списана сумма эквайринговой комиссии, и при необходимости аналитику расходов.
  • В случае, когда в организации используются подразделения для учета доходов и расходов, выбрать подразделение.
  • Установить или снять флажок Проводить документы при загрузке.
  • Загрузить отчет банка по эквайрингу.
