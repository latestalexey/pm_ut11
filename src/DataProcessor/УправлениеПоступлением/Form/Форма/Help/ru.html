Какие документы отображаются в качестве распоряжений в обработке "Приемка товаров на склад"?

В качестве распоряжений в обработке отображаются как документы от внешних поставщиков, так и документы по внутренним предприятиям торгового предприятия. В списке показываются только те распоряжения, по которым поставка товаров не соответствует распоряжениям: поступление товаров на склад оформлено не полностью или оформлена поставка большего количества товаров, чем было зарегистрировано в распоряжении.

Распоряжениями на поставку товаров могут быть следующие документы.

  • Заказ поставщику. В заказе поставщику должен быть установлен статус "К поступлению".
  • Поступление товаров и услуг. Распоряжениями являются только те документы, которые оформлены без указания заказа поставщику.
  • Заказ на сборку/разборку. Документ будет показан в качестве распоряжения только в том случае, если на основании него оформлен и проведен документ "Сборка (разборка) товаров". Количество принимаемых товаров соответствует тому количеству, который указан в документе Документ будет показан в качестве распоряжения только в том случае, если на основании него оформлен и проведен документ "Сборка (разборка) товаров".
  • Сборка (разборка) товаров. Документ будет показан в качестве распоряжения только в том случае, если он оформлен без указания заказа. В зависимости от вида оформляемой операции (сборка или разборка) в списке принимаемых товаров будет показан комлект или комплектующие.
  • Заказ на перемещение. Документ будет показан в качестве распоряжения только в том случае, если на основании него оформлен и проведен документ "Перемещение товаров".
  • Перемещение товаров. Документ будет показан в качестве распоряжения только в том случае, если он оформлен без указания заказа.

Важно. Предусмотрена возможность оформления приемки товаров на склад без оформления конкретных документов распоряжения. В этом случае в качестве распоряжения на поставку используется "Соглашение с поставщиком". В соглашении с поставщиком должен быть установлен флажок "Приемка товаров может происходить без заказов и накладных".

Как ввести новый документ поставки товаров на склад?
  • В обработке "Приемка товаров на склад" укажите склад, на который оформляется поставка товаров.
  • В табличной части "Распоряжения на поступления" установите курсор на тот документ распоряжения, по которому оформляется поставка товаров.
  • Нажмите на кнопку "Создать ордер".

Будет создан документ "Приходный ордер на товары" в соответствии с указанным распоряжением. В документе будут заполнены те товары, по которым еще не оформлено фактическое поступление товаров на склад.

Можно ли распечатать несколько документов "Приходный ордер на товары"?
  • В обработке "Приемка товаров на склад" в табличной части "Приходные ордера" снимите флаг "Только принятые".
  • Выделите в списке те приходные ордера, которые необходимо распечатать, используя возможности "1С:Предприятие".
  • Нажмите на кнопку "Печать".
  • Выберите нужную форму печати.

Будет распечатан список выделенных документов. Каждый документ будет распечатан на отдельном листе.

Каким образом проконтролировать процесс приемки на склад и определить какие товары поставщик недопоставил в соответствии с финансовыми документами?
  • В обработке "Приемка товаров на склад" нажмите на кнопку "Состояние приемки".
  • Будет открыт отчет "Исполнение распоряжений на поступление".

В отчете будут показаны только невыполненные распоряжения. К невыполненным распоряжения относятся также и те распоряжения, по которым была перепоставка товаров. Для каждого распоряжения будет показана следующая информация.

  • К поступлению. Количество товаров, которое должно быть принятно в соответствии с распоряжением. В качестве распоряжения используются документы "Заказ поставщику", "Поступление товаров и услуг", "Возврат товаров от клиента".
  • Принято. Количество принятого товара. Количество принятого товара регистрируется при проведении документа "Приходный ордер на товары" с установленным статусом "Принято".
  • Осталось принять. Количество товаров, которое осталось принять в соответствии с распоряжением.
  • Перепоставка. Количество товаров, которое было принято сверх того количества, которое указано в распоряжении.
  • Принимается. Количество товаров, которое находится в стадии приемки, то есть тот товар, который зарегистрирован в документе "Приходный ордер на товары" с установленным статусом "К поступлению".

С помощью этого отчета можно получить ответы на следующие вопросы.

  • Полностью ли принят товар в соответствии с конкретным распоряжением? Для ответа на этот вопрос необходимо в быстрых настройках отчета в поле "Распоряжение" выбрать из соответствующего списка нужное распоряжение. В отчете будет показано состояние выполнения приемки товаров по конкретному распоряжению. Если товар принят полностью в соответствии с распоряжением, то будет сформирован пустой отчет.
  • Полностью ли осуществлена приемка товара от конкретного поставщика? Для ответа на этот вопрос необходимо в быстрых настройках отчета в поле "Партнер" указать конкретного поставщика. В отчете будет показано состояние выполнения приемки товаров по документам поставки (распоряжениям) конкретного поставщика. Будут показаны те распоряжения, по которым есть расхождения по фактической приемке товаров в соответствии с распоряжениями.

Как открыть список товаров, которые находятся в зоне приемки и которые необходимо разместить в складские ячейки?
  • В форме обработки "Приемка товаров на склад" укажите склад, в ячейках котрого необходимо разместить товар.
  • Перейдите на страницу "Размещение".
  • Укажите ту ячейку, в которую принимался товар (зона приема). Если для склада определена одна ячейка с типом "Приемка" (зона приема), то информация об этой ячейке будет заполнена автоматически после указания склада.
  • В таблице "Номенклатура к размещению" будет показан список товаров, которые необходимо разместить по складским ячейкам.

Как создать задания на размещение всего товара одним исполнителем, но при этом ограничить список товаров в каждом задании весом и объемом товаров?
  • На странице "Размещение" обработки "Приемка товаров на склад" в верхней табличной части "Номенклатура к размещению" нажмите на кнопку "Создать задания".
  • В появившейся диалоговой форме "Настройки" укажите следующие параметры.
  • Установите флаг в поле "Назначить исполнителя" и выберите из справочника "Физические лица" того сотрудника склада, который будет выполнять задания.
  • Установите флаги "Ограничивать по весу" и "Ограничивать по объему" .
  • Введите информацию о весе и объеме товаров, которые должны входить в одно задание.
  • Нажмите кнопку "ОК".

Будет сформирован список заданий для конкретного исполнителя в соответствии с настроенными правилами автоматического размещения по ячейкам. В каждом задании будут присутствовать тот список товаров,общий объем и вес которых не превышает объем и вес, указанный при настройке заданий. Список сформированных заданий будет показан в табличной части "Задания на размещение" в нижней части диалоговой формы "Приемка товаров на склад".

Важно. При формировании заданий на размещение в соответствии с правилами автоматического размещения список ячеек, в которые необходимо разместить товар сортируется по порядку обхода. Только после этого задания разделяются по ограничению веса и объема товаров.

Как создать задания на размещение, если часть товаров может быть размещена в ячейки вручную, а для размещения другой части товаров требуется автопогрузчик?

В этом случае задания формируются последовательно. Предварительно складские ячейки должны быть разделены на два рабочих участка, например, "Размещение вручную" и "Автопогрузчик". Порядок формирования должен быть следующий.

  • На странице "Размещение" обработки "Приемка товаров на склад" в верхней табличной части "Номенклатура к размещению" нажмите на кнопку "Создать задания".
  • В появившейся диалоговой форме "Настройки" укажите следующие параметры.
  • В поле "По одному рабочему участку" укажите рабочий участок, например, "Размещение вручную".
  • Установите флаг в поле "Назначить исполнителя" и выберите из справочника "Физические лица" того сотрудника склада, который будет размещать товар вручную.
  • Установите флаги "Ограничивать по весу" и "Ограничивать по объему" .
  • Введите информацию о весе и объеме товаров, которые должны входить в одно задание.
  • Нажмите кнопку "ОК".

Будет сформирован список заданий для размещения товаров вручную. В список попадут только те товары, которые должны быть размещены в тех складских ячейках, которые относятся к конкретному рабочему участку. То есть это будут складские ячейки, которые имеют доступ к ручному размещению товаров. Список товаров в каждом задании будет ограничен по весу и объему, заданному в настройках. В каждом задании будет указан тот рабочий участок ("Размещение вручную"), который был указан в настройках при формировании задания.

Аналогичным образом сформируйте задания по рабочему участку "Автопогрузчик". Для этих заданий можно задать другое ограничение по весу и объему и назначить другого исполнителя.

Как создать задания по определенному списку товаров?
  • На странице "Размещение" обработки "Приемка товаров" в верхней табличной части "Номенклатура к размещению" выделите список тех товаров, по которым необходимо сформировать задания.
  • Нажмите на кнопку "Создать задания".
  • В появившейся диалоговой форме "Настройки" укажите необходимые параметры, если задания должны быть ограничены по весу, объему.
  • Назначьте исполнителя задания.
  • Нажмите кнопку "ОК".

Будет сформирован список заданий по тем товарам, которые были выделены в списке. Список сформированных заданий будет показан в табличной части "Задания на размещение" в нижней части диалоговой формы "Приемка товаров".

Как создать задание на размещение товаров на основании конкретной поставки товаров (документа "Приходный ордер на товары")?
  • В обработке "Приемка товаров на склад" перейдите на страницу "Поступление".
  • В нижней табличной части "Приходные ордера" снимите флаг "Только непринятые".
  • Установите курсор на тот документ, по которому уже зафиксирован прием товаров в зону приема (у документа установлен статус "Принят".)
  • Нажмите на кнопку "Создать на основании" и выберите пункт "Отбор (размещение) товаров".
  • Будет создано новое задание на размещение товаров. Список товаров будет скопирован из документа "Приходный ордер на товары".

В документе нужно заполнить список ячеек, в которых должен быть размещен товар. Список ячеек может быть заполнен вручную или автоматически с использованием алгоритма автоматического размещения товаров по ячейкам. Для автоматического размещения товаров по складским ячейкам нажмите на кнопку "Разместить" в командной панели табличной части документа.

Как посмотреть список подготовленных заданий, которые еще не переданы исполнителям?
  • В обработке "Приемка товаров на склад" перейдите на страницу "Распоряжения".
  • В табличной части "Задания на размещение"установите флаг "Только не выполненные".

Будет показан список тех заданий на размещение, которые подготовлены, но еще не переданы исполнителям для выполнения.

Можно ли распечатать сразу несколько заданий на размещение товаров?
  • В обработке "Приемка товаров на склад" в списке подготовленных заданий выделите те задания, которые необходимо передать исполнителям для выполнения размещения, используя возможности 1С:Предприятия.
  • Нажмите на кнопку "В работу".
  • В появившемся диалоговом окне укажите как должны быть распечатаны задания: с предварительным просмотром или сразу на принтер.
  • Если все выделенные задания должен выполнить один исполнитель, то укажите исполнителя в форме настройки. Информация об этом исполнителе будет указана во всех выделенных заданиях.
  • Если задания предварительно подготовлены для разных исполнителей, то не указывайте исполнителя в настройках. В выделенных заданиях будет указан тот исполнитель, для которого было подготовлено задание.
  • Нажмите на кнопку "ОК".

Задания будут распечатаны и для них будет установлен статус "В работе". Каждое задание будет распечатано на отдельном листе.

Можно ли посмотреть список тех товаров, которые сейчас размещаются складскими работниками?
  • Список товаров, которые сейчас размещаются складскими работниками, отображается в обработке "Приемка товаров на склад" на странице "Размещение".
  • В таблице "Номенклатура к размещению" информация о количестве товаров, которые сейчас размещаются складскими работниками, показана в графе "Размещается".
