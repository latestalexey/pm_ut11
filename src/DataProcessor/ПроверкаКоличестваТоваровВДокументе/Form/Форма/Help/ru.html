Проверка количества товара в ордере

После заполнения ордера (вручную или с ТСД) по команде Проверить количество можно указать результаты пересчета фактически принятого товара. Строки товара у которого существует расхождение пересчитаного количества и количества указанного в документе выделяются красным цветом. В этом случае, по команде Завершить проверку откроется форма в которой будут показаны товары с зафиксироваными излишками и/или недостачами. После принятия решения о дальнейших действиях можно:

  • Вернуться в форму пересчета, для уточнения результатов фактического пересчета (по команде Вернуться к проверке).
  • По команде Перенести в документ завершить проверку. При этом в ордер попадет фактическое количество товара.

В случае совпадения фактического количества товара с количеством товара указаного в документе, по команде Завершить проверку форма проверки количества товара будет закрыта.
