Форма

форма предназначена для интерактивного выполнения обмена данными с мобильными приложениями в случае, когда обмен данными инициируется не со стороны клиента, а со стороны 1С:Предприятия, т.е. из 1С:Предприятия дается команда на выполнение обмена данными с мобильным приложением.

Для возможности выполнения обмена данными из формы в текущем сеансе пользователя должны быть запущена компонента обмена данными с мобильными приложениями. При этом для работы компоненты используются ранее заданные настройки компоненты. Для того, чтобы просмотреть эти настройки, или задать новые, можно воспользоваться командой Настройки компоненты.

Как указать, с каким клиентом (пользователем и мобильным устройством) необходимо выполнить обмен данными?

При выполнении обмена данными с мобильными приложениями подключение пользователя мобильного приложения характеризуется двумя параметрами- конкретным пользователем системы, имеющим соответствуюшие права, а также конкретным мобильным устройством, которое присутствует в списке зарегистрированных мобильных компьютеров.
Для выполнения обмена необходиммо выбрать в поле Настройки обмена настройки, соответствующие нужному пользователю и мобильноему устройству.

Для чего требуется выбрать мобильную ИБ?

На мобильном устройстве пользователь, для которого выполняется обмен данными, может иметь несколько различных информационных баз. При выборе в поле Имя мобильной базы система отображает список информационных баз, имеющихся на мобильном устройстве. Для обмена нужно выбрать соответствующую ИБ из этого списка.

Какие настройки необходимо задать для выполнения обмена?

Для выполнения обмена данными необходимо указать следующие параметры обмена:

  • Способ связи (нужно выбрать из списка предпочтительный способ связи)
  • Вид клиента (сетевое подключение, если используется мобильное устройство, или другой ПК, либо "Этот компьютер", если клиент запущен на данном компьютере
  • Адрес подключения и порт - сетевой адрес подключенного устройства, либо локальный адрес (127.0.0.1) для текущего компьютера, а также порт TCP|IP, через который выполняется обмен данными.
  • Флаг Передавать данные клиенту - требуется установить, если необходимо передать данные из 1С:Предприятия клиенту. Если требуется только получить данные клиенту, не передавая ему данных, установка флага не требуется.
  • Флаг Выполнить начальную инициализацию ИБ - флаг необходимо установить, если требуется очистить данные, имеющиеся в мобильной ИБ и передать клиенту все данные, удовлетворяющие настройкам обмена.
