С помощью группового изменения объектов пользователь может за одно действие изменить значения реквизитов, дополнительных реквизитов или дополнительных сведений у выбранных объектов определенного типа.

На первой странице помощника выводятся все доступные реквизиты, значения которых можно изменить (если у пользователя нет прав на редактирование реквизита, то такой реквизит в список не попадет). В колонке Значение указывается новое значение реквизита. Если у реквизита требуется задать пустое значение, то необходимо только отметить флажок в колонке Изменять, а колонку Значение оставить пустой.

Нужно иметь в виду, что у определенных объектов могут иметься реквизиты, которые определяют характер данного объекта и которые условно можно назвать "ключевыми" реквизитами. Как правило, "ключевыми" реквизитами являются реквизиты объекта, влияющие на поведение других объектов.

Для того чтобы предупредить необдуманные изменения "ключевых" реквизитов, такие реквизиты по умолчанию недоступны для изменения и отображаются серым цветом. В случае особой необходимости изменения таких реквизитов, можно воспользоваться командой Разрешить редактирование реквизитов.

В некоторых случаях, в списке реквизитов доступных для изменения могут попасть реквизиты, не видимые на форме объекта (например, отключенные функциональными опциями). Изменение таких реквизитов происходит в соответствии с внутренней логикой объекта (целостность, непротиворечивость данных и др.).

С помощью команды История можно быстро выбрать настройки группового изменения, которые применялись для данного типа объектов ранее.

По команде Открыть объект по ссылке или двойным щелчком мыши в списке можно перейти к форме объекта.

По команде Изменить можно сразу же применить новые значения реквизитов к выбранным объектам. Если  дополнительно необходимо настроить список обрабатываемых объектов, то можно воспользоваться кнопкой Далее. На второй странице помощника имеется возможность откорректировать состав изменяемых объектов, а также посмотреть текущие значения изменяемых реквизитов.

Дополнительно на первой странице помощника можно задать особые настройки с помощью команды Настроить параметры изменения.
