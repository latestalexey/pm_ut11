Какая информация представлена в отчете "Сравнение условий по доступным соглашениям"?

В отчете приводится информация по всем доступным для партнера торговым соглашениям. Информация выводится в следующих разрезах:

  • "Валюта" - в какой валюте возможна оплата по соглашению.
  • "Склад" - с какого склада возможна отгрузка. Если по соглашению указано "Без ограничений", то отгрузка товара возможна с любого склада, партнер при оформлении заказа должен выбрать склад самостоятельно.
  • "График оплаты" - можно сравнить условия оплаты заказа по соглашениям.
  • "Налогообложение НДС" - указывается режим налогообложения по тому или иному соглашению.
  • "Цена включает НДС" - указывается включает цена по соглашению НДС или нет.
  • "Контрагент" - конкретный контрагент с которым возможна работа по соглашению. Если по соглашению указано "Без ограничений", то отгрузка товара возможна любому контрагенту партнера.

Помимо этого в отчете можно сравнить цены и суммы заказа в разрезе всех доступных соглашений.
