Какие действия можно выполнить в списке "Мои заказы"?

В списке "Мои заказы" можно выполнить следующие действия:

  • Открыть для просмотра созданный заказ. Подробнее см. справку к заказу.
  • Установить период за который в списке будут отображаться заказы (команда "Установить интервал дат").
  • При большом количестве заказов выполнить в списке поиск необходимых заказов (команда "Найти").
  • Распечатать выделенный в списке заказ (команда "Печать").
  • Ограничить список заказов только невыполнеными заказами (флажок внизу списка "Не показывать закрытые").
  • Посмотреть состояние заказа.
  • Посмотреть список связанных с заказом документов.

Как посмотреть состояние нужного заказа?

Для того чтобы посмотреть состояние нужного заказа, необходимо выделить его в списке и воспользоватся командой "Состояние выполнения". Далее будет открыт отчет "Состояние выполнения заказов" в котором представлено состояние товародвижения каждой позиции заказа. Так же состояние заказа в целом можно увидеть в списке в колонке "Состояние".

Как открыть список связанных с заказом документов?

Что бы посмотреть список связанных с заказом документов, необходимо выделить в списке нужный заказ, и воспользоваться командой "Связанные документы". Далее будет открыт отчет "Структура подчиненности", в который выводится список документов связанных с текущим. Например для заказа в список могут попасть счета на оплату созданные на основании заказа или документы реализации по заказу.
