Подбор товаров в корзину покупателя

Что делать если после выбора соглашения информация о складе незаполнилась автоматически?

Если информация в поле "Склад" незаполнилась автоматически (после выбора соглашения), это значит что по условиям соглашения партнер может выбрать склад для отгрузки самостоятельно, на свое усмотрение. Например, можно выбрать склад наиболее близко расположенный к партнеру.

Важно. Поля "Соглашение" и "Склад" является обязательным для заполнения перед подбором товаров в заказ. Команда "Подобрать товары" будет доступна только в том случае если будут заполнены эти два поля.

Какие действия с товаром можно выполнить в корзине покупателя?

С помощью команды "Удалить" ранее подобранный товар можно удалить из корзины. У подобранных в корзину товаров можно установить/снять флажок "К оформлению". С помощью команды "Оформить заказ" на отмеченные флажком товары будет оформлен заказ. При этом неотмеченные флажком "К оформлению" товары останутся в корзине, а товары на которые был оформлен заказ будут удалены из корзины. С помощью команды "Подобрать товары" можно пополнить корзину товаром. Непосредственно в корзине можно изменить количество заказываемого товара.

Как сравнить условия и цены закупки по разным торговым соглашениям?

После того как товары подобраны в корзину можно сравнить условия покупки и цены товаров по разным торговым соглашениям. Провести анализ можно с помощью отчета "Сравнение условий по доступным соглашениям". Отчет можно открыть в форме "Корзина покупателя", командой "Отчеты" - "Сравнение условий по доступным соглашениям".

Как посмотреть доступность на складе, товара подобранного в корзину?

После того как товары подобраны в корзину можно посмотреть их доступность на складе. Доступность товаров можно оценить с помощью отчета "Остатки по складам". Отчет можно открыть в форме "Корзина покупателя", командой "Отчеты" - "Остатки по складам".

Оформление заказа

Как оформить заказ на товар в корзине покупателя?

Оформить заказ на товар можно в корзине покупателя с помощью помощника "Оформление заказа". Помощник открывается командой "Оформить заказ" в корзине покупателя. Заказ будет оформлен на товар в корзине покупателя с установленными флажками "К оформлению".
