Какие действия можно выполнить в списке "Документы реализации"?

В списке "Документы реализации" можно выполнить следующие действия:

  • Открыть для просмотра документ "Реализация товаров и услуг".
  • Установить период за который в списке будут отображаться документы (команда "Установить интервал дат").
  • При большом количестве документов выполнить в списке поиск необходимых реализация (команда "Найти").
  • Посмотреть список связанных с реализацией документов.

Как распечатать документ "Реализация товаров и услуг"?

Документ "Реализация товаров и услуг" можно из списка реализаций, с помощью команды "Печать" - "Реализация товаров". Этаже команда доступна в форме самого документа реализации.

Как открыть список документов связанных с реализацией?

Что бы посмотреть список связанных с реализацией документов, необходимо выделить в списке нужный документ реализации, и воспользоваться командой "Связанные документы". Далее будет открыт отчет "Структура подчиненности", в который выводится список документов связанных с текущим. Например для документа реализации в список могут попасть счета на оплату по которым производилась оплата реализации или заказ.
