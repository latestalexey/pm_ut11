Как можно открыть отчет "Состояние взаиморасчетов"?

Как изменить контактные данные партнера?

Контактные данные партнера можно изменить в форме "Изменение контактной информации". Форма открывается командой "Изменить контактную информацию" в панели действий "Сервис". В форме можно изменить адрес, электронный адрес и телефон партнера. Редактировать адрес и телефон можно как в полях ввода, так и с помощью отдельных форм.

Как посмотреть, отредактировать или добавить информацию по контактным лицам партнера?

Список контактных лиц партнера находится в форме "Мои контактные лица". Форма открывается соответствующей командой панели навигации. С помощью команды списка "Изменить" можно приступить к редактированию имеющихся записей. По команде "Создать" можно приступить к вводу информации о новом контактном лице партнера. На соответствующих страницах формы указывается Ф.И.О. телефоны и адрес электронной почты контактного лица. На странице "Дополнительная информация" можно указать произвольную текстовую информацию.

Как посмотреть, отредактировать или добавить информацию по контрагентам (юридическим или физическим лицам) партнера?

Список контрагентов партнера находится в форме "Мои контрагенты". Форма открывается соответствующей командой панели навигации. С помощью команды списка "Изменить" можно приступить к редактированию имеющихся записей. По команде "Создать" можно приступить к вводу информации о новом контрагенте партнера. В форме указывается наименование контрагента, устанавливается признак юридического или физического лица. На соответствующих страницах формы указывается контактная информация контрагента. На странице "Дополнительная информация" можно указать произвольную текстовую информацию. Так же можно отредактировать или указать информацию по банковским счетам контрагента.

Где можно указать настройки для работы с системой самообслуживания?

Указать настройки для работы с системой самообслуживания можно в форме "Внешний пользователь". Форма открывается командой "Мои настройки" в панели действий "Сервис". В форме можно изменить имя внешнего пользователя, задать пароль для входа в систему, указать язык интерфейса.

Просмотр соглашений, доступных для осуществления заказа

Как открыть список соглашений с партнером?

Список торговых соглашений находится в форме "Мои соглашения". Форма открывается соответствующей командой в панели навигации.

Как приступить к оформлению заказа?

Партнер может оформить заказ в форме "Корзина покупателя". Форма открывается соответствующей командой панели навигации.

Просмотр заказов и оформленных по ним документов

Где можно увидеть список заказов оформленных ранее в системе самообслуживания?

Список заказов оформленных ранее в системе самообслуживания находится в форме "Мои заказы". Форму можно открыть соответствующей командой в панели навигации.

Где можно увидеть список реализаций оформленных в системе самообслуживания на основании заказов?

Список документов реализации оформленных на основании заказов находится в форме "Документы реализации". Форму можно открыть соответствующей командой в панели навигации.

Где можно увидеть список актов выполненных работ, оформленных по партнеру?

К списку aктов выполненных работ, оформленных по партнеру можно перейти по команде панели навигации "Акты выполненных работ".

Где можно увидеть список счетов на оплату оформленных в системе самообслуживания на основании заказов?

Список счетов оформленных на основании заказов находится в форме "Мои счета на оплату". Форму можно открыть командой в панели навигации "Счета на оплату".

Получение информации о состоянии взаиморасчетов

Информацию о взаиморасчетах можно получить из отчета "Состояние взаиморасчетов". Так же краткая информация о взаиморасчетах представленна на рабочем столе.

Как можно открыть отчет "Состояние взаиморасчетов"?

Отчет "Состояние взаиморасчетов" можно открыть соответствующей командой в панели действий, в группе "Отчеты".
