Какие действия можно выполнить в списке "Акты выполненных работ"?

В списке "Акты выполненных работ" можно выполнить следующие действия:

  • Открыть для просмотра документ "Акт выполненных работ".
  • Установить период за который в списке будут отображаться документы (команда "Установить интервал дат").
  • При большом количестве документов выполнить в списке поиск необходимых актов (команда "Найти").
  • Посмотреть список связанных с актом документов.

Как распечатать документ "Акт выполненных работ"?

Документ "Акт выполненных работ" можно распечатать поизционировавшись на документе и выполнив команду "Печать" - "Акт выполненных работ". Эта команда доступна и в форме самого документа реализации.

Как открыть список документов связанных с "Актом выполненных работ"?

Необходимо выделить в списке нужный документ реализации, и воспользоваться командой "Связанные документы". Далее будет открыт отчет "Структура подчиненности", в который выводится список документов связанных с текущим. Например для документа "Акт выполненных работ" в список могут попасть счета на оплату по которым производилась оплата.
