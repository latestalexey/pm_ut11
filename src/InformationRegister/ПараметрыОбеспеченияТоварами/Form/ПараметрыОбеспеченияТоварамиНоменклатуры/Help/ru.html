Как откорректировать метод управления запасами вручную для конкретной позиции номенклатуры?

Как откорректировать метод управления запасами вручную для конкретной позиции номенклатуры?
  • В списке методов управления запасами для конкретной номенклатурной позиции нажмите на кнопку "Создать".
  • В появившейся диалоговой форме будет заполнена информация о той номенклатурной позиции, из формы которой вызывается список методов управления запасами.
  • Укажите склад, на котором надо откорректировать метод управления запасами.
  • Установите флажок в том методе управления запасами ("Метод управления"), который Вы хотите установить для конкретного товара на данном складе.
  • Нажмите на кнопку "Записать и закрыть".

Важно. Метод управления запасами для конкретной номенклатурной позиции всегда является более приоритетным по сравнению с тем методом, который установлен для всех номенклатурных позиций на складе. В том случае, если Вы в дальнейшем захотите использовать метод управления запасами, который будет рассчитан программой на основе анализа продаж товаров в следующем периоде, то запись о ручной установке метода управления запасами следует удалить из списка.
