Форма записи

Для настройки и сохранения списка печатных форм необходимо выполнить следующие действия.

  • Отметить в списке те печатные формы, которые нужно печатать.
  • В поле "Экземпляров" указать количество печатных копий для каждого документа.
  • Нажать на кнопку "Управление настройками" и выбрать нужный вариант сохранения настроек.

Указанный вариант печати комплекта документов может применяться.

  • При печати любого документа данного типа.
  • При печати документа данного типа, если он оформлен по конкретной организации.
  • При печати документа данного типа , если он оформлен по конкретному клиенту и конкретной организации.

Печать нескольких печатных форм по документу производится по команде Печать-Комплект документов. Документы сразу выводятся на печать без предварительного просмотра.
