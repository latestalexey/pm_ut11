Форма списка

В списке хранятся все настройки комплектов печатных форм, которые были сохранены в программе.

В процессе работы менеджер может для каждого документа настроить свой комплект формирования печатных форм и указать количество печатных копий. Настройка производится в списках документов по команде Печать/Комлект документов с настройкой. В списке возможных печатных форм показываются все те печатные формы, которые могут быть сформированы для данного документа. Менеджер может отметить те формы, которые надо печатать при работе с конкретным клиентом, указать количество копий и сохранить эти настройки для формирования печатных форм при оформлении документов по конкретной организации или по конкретному клиенту.

Добавление списка необходимых печатных форм для документа может производиться и из списка "Настройки печати объектов". Для добавления нового комплекта печатных форм выполните следующие действия.

  • Нажмите на кнопку Добавить.
  • Появится новая форма для настройки списка печатных форм.
  • Выберите документ, для которого необходимо настроить комплект печатных форм.
  • Отметьте в списке нужные печатные формы и укажите количество печатных копий (экземпляров).
  • Если комплект печатных форм должен формироваться по конкретной организации (клиенту), то введите эти данные.
  • Нажмите на кнопку "Записать и закрыть".

Комплект печатных форм для документа будет сохранен и будет использоваться при печати комплекта документов. Для каждой организации/клиента можно настроить свой комплект печатных форм.
