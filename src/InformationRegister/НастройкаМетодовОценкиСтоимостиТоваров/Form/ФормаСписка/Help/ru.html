Выбор метода оценки стоимости товаров

Метод оценки стоимости товаров устанавливается для каждой организации торгового предприятия. Установленный метод оценки стоимости обычно действует в течение года.

Расчет себестоимости по указанному методу оценки стоимости товаров производится регламентным документом "Расчет себестоимости товаров". Регламентный документ обычно оформляется в конце каждого месяца.

Важно. В программе предусмотрена возможность расчета себестоимости по всем организациям с учетом передач товаров между организациями. Метод расчета себестоимостьи в этом случае устанавливается в документе расчета себестоимости

Как добавить новый метод оценки стоимости товаров?
  • В списке "Настройка методов оценки стоимости товаров" нажмите на кнопку "Создать".
  • В появившейся карточке укажите организацию, период действия и метод оценки стоимости товаров.
  • Нажмите на кнопку "Записать и закрыть".

Как настроить правила распределения выручки и себестоимости продаж по подразделениям?

Для анализа финансовых результов в разрезе подразделений можно настроить правила распределения выручки и себестоимости продаж по направлениям деятельности. Для настройки правил распределения в панели навигации формы подразделения используется команда "Настройка распределения продаж".
