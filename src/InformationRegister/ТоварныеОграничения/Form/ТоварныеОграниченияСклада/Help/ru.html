Как отредактировать рассчитанные товарные ограничения из формы склада?

В списке товарных ограничений для конкретного склада показывается список всех номенклатурных позиций, по которым рассчитаны товарные ограничения по данному складу. Для корректировки товарных ограничений выполните следующие действия.

  • Установите курсор на запись по тому товару, для которого необходимо изменить товарные ограничения.
  • Нажмите на кнопку редактирования в командной панели списка ограничений.
  • В появившейся карточке "Товарные ограничения " значения используемых товарных ограничений будут установлены в соответствии с рассчитанными значениями. Исключением является максимальный запас, который не рассчитывается и может быть установлен только вручную.
  • Отредактируйте значения используемых товарных ограничений.
