Отчет "Выполнение регулярных условий соглашений с клиентами"

Отчет предназначен для контроля выполнения тех условий, которые заданы в соглашении с клиентом. Контроль производится по тем соглашениям, в которых заданы регулярные условия продаж. Например, каждый месяц клиент в течение квартала должен закупать товар на определенную сумму.

В отчете показывается.

  • Сумма, на которую клиент должен закупить товар в соответствии с условиями соглашения.
  • Реальная сумма продаж товаров клиенту.
  • Процент выполнения условий клиентом.
