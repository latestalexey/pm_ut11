Отчет "Анализ цен"

Отчет "Анализ цен" предназначен для сравнения цен, указанных в документах продажи (Коммерческое предложение, Заказ клиента, Реализация товаров и услуг) с ценами поставщиков и конкурентов.

В отчете показывается информация о ценах, которые зарегистрированы в документе продажи, с учетом предоставленных скидок (Цена со скидкой) и без учета скидок (Цена документа). Для сравнения показываются те цены поставщиков (конкурентов), которые зарегистрированы по товару документом Регистрация цен поставщика в рамках оформленного соглашения с поставщиком. Если для поставщика (конкурента) зарегистрировано несколько цен по различным условиям поставки (например, цены зависят от сроков и объема поставок), то в отчете будет выведено несколько цен поставщиков (конкурентов) для сравнения и будут указаны те условия, при которых они действуют.

Важно. В отчете показываются только те товары, по которым есть информация о зарегистрированных ценах поставщиков (конкурентов).

Минимальные значения цен в отчете отображаются зеленым цветом, а максимальные значения цен отображаются красным цветом.
