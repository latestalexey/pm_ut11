Отчет "Контроль номеров ГТД товаров"

Отчет предназначен для контроля правильности ведения учета операций с импортными товарами с указанием номера ГТД и страны происхождения. В отчете показывается полная информация о количестве поступивших и отгруженных товаров с детализацией по номерам ГТД и странам происхождения. Используя этот отчет можно проконтролировать прохождение номеров ГТД и стран происхождения по конкретной позиции номенклатуры, по конкретной организации. С помощью отчета можно выявить случаи неправильно указания номеров ГТД при поступлении и отгрузке товаров.
