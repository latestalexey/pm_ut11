Отчет "Анализ эффективности работы торговых представителей"

Отчет позволяет проанализировать, насколько успешно торговые представители справляются с поставленными заданиями. Отчет используется только при схеме работы с выдачей заданий торговым представителям.

Отчет сгруппирован по торговым представителям. Для каждого торгового представителя показывается информация о количестве и сумме отработанных и неотработанных заданий. Показывается процент и сумма выполнения плановых показателей по отношению к фактическим показателям.

  • Кол-во отработанных заданий - количество отработанных заданий, то есть тех заданий, по которым установлен статус "Отработано".
  • Кол-во неотработанных заданий - количество неотработанных заданий, то есть заданий по которым установлен статус "Неотработанно".
  • Среднее кол-во отработанных заданий (в день) - среднее количество заданий, отработанных за день.
  • Процент отработки заданий - процент отработки заданий. Формируется, как отношение количества отработанных заданий к общему количеству выданных заданий.
  • Кол-во просроченных неотработанных заданий - количество заданий, которые неотработаны к плановой дате визита.
  • Кол-во заданий, отработанных позже срока - количество заданий, у которых дата фактического визита больше плановой даты.
  • Среднее кол-во дней просрочки - среднее количество дней по тем заданиям, которые отработаны позже планового срока.
  • Сумма план - плановая сумма задания.
  • Сумма по отработанным заданиям - общая сумма по всем отработанным заданиям.
  • Сумма по неотработанным заданиям - общая сумма по всем неотработанным заданиям.

По каждому заданию и по каждой номенклатурной позиции (если задания детализированы по номенклатуре) сумма отработанного задания сравнивается с плановой суммой задания. В зависимости от значения этого показателя будет показана Сумма недовыполнения плана или Сумма перевыполнения плана.

  • Недовыполнение плана (%) - процентное отношение между суммой недовыполнения плана и плановой суммой.
  • Перевыполнение плана (%) - процентное отношение между суммой перевыполнения плана и плановой суммой.
