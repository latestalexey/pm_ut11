Отчет "Анализ себестоимости товаров по поставщикам"

Отчет "Анализ себестоимости товаров по поставщикам" предназначен для анализа стоимости закупаемых товаров, а также себестоимости и суммы дополнительных расходов, которые распределены на себестоимость поступивших товаров. Анализ может быть выведен с детализацией по конкретным позициям номенклатуры и по конкретным складам. Для отчета предусмотрено два варианта: Анализ поступлений товаров по поставщикам и "Анализ возвратов товаров поставщикам".

Важно. Варианты отчета "Анализ себестоимости товаров по поставщикам" будут доступны пользователю только в том случае, если в программе ведется обособленный учет товаров по поставщикам ( в настройках параметров учета на странице "Финансы " установлен флажок ведения обособленного учета по поставщикам).
