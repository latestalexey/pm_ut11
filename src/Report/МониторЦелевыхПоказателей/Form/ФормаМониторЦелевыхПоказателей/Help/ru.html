Как открыть пользовательские настройки монитора целевых показателей?
  • Нажмите на кнопку "Настроить".
  • Будет открыто диалоговое окно настроек.

Диалоговое окно настроек разделено на две страницы.

  • На странице "Настройки" настраиваются параметры, которые касаются способа отображения показателей в мониторе: группировка, вид представления и т.д.
  • На странице "Варианты анализа" можно ограничить список тех вариантов анализа, которые будут показаны в мониторе и изменить вариант представления каждого варианта.

Можно ли в программе сводный отчет о состоянии дел для предоставления данных руководству?

Для формирования сводного отчета нажмите на кнопку "Сводный отчет" в командной панели монитора целевых показателей.

Будет сформирован отчет, в который входят отображаемые в мониторе показатели в развернутой форме (описание показателя, параметры расчета, диаграммы). Сводный отчет может быть распечатан или отправлен руководителю по электронной почте. Использование сводного отчета позволит произвести экспресс-контроль и анализ целевых показателей вне информационной системы.

Как проконтролировать достижение поставленной цели с помощью статусов?

Для каждого целевого показателя показывается его состояние (статус).

  • Если текущее значение целевого показателя соответствует введенному значению (диапазону), то устанавливается статус Цель достигнута.
  • Если текущее значение целевого показателя находится в пределах тех отклонений, которые указаны для показателя, то устанавливается статус Допустимое значение.
  • Если текущее значение целевого показателя имеет негативное отклонение больше допустимого, то устанавливается статус Недопустимое значение.
  • В том случае, если допустимые или текущие значения целевого показателя не определены, то устанавливается статус Статус неопределен.

Как определить какие тенденции изменения целевого показателя: положительные или негативные?

Тенденции изменения целевого показателя зависят от установленного варианта целевого тренда.

  • Если целевым трендом является максимизация значения, то позитивным будет являться Рост, позитивная динамика, а негативным будет являться Падение, негативная динамика.
  • Если целевым трендом является минимизация значения, то позитивным будет являться Падение, позитивная динамика, а негативным будет являться Рост, негативная динамика.
  • Если целевым трендом является удержание в пределах допустимого значения, то позитивным будет являться Рост, в пределах допустимого диапазона, а негативным Падение, в пределах допустимого диапазона.
  • Если значение показателя остается без изменений, то будет отображаться пиктограмма Без изменений.

Как проконтролировать целевой показатель с помощью краткой формы отображения?

В данном представлении для обозначения состояния дополнительно используется окрашивание в цвет значения показателя. Состояние и тренд отображаются пиктограммой.

  • Красный цвет - недопустимое значение.
  • Оранжевый цвет - допустимое значение или цель достигнута с негативным трендом.
  • Цвет текста формы - допустимое значение или цель достигнута с позитивным трендом или без изменений.

Как контролировать текущее состояние с помощью диаграммы?

С помощью диаграммы производится контроль текущего значения ключевых показателей по статусу, тренду и характеру изменений или структуре, изображенным на диаграмме.

Статус и тренд отображаются соответствующими пиктограммами.

На диаграмме может отображаться следующая информация.

  • Характер изменений значений показателя со временем
  • Структура текущего значения показателя в установленном для показателя разрезе. Например, выручка по клиентам или по менеджерам
  • Текущее состояние показателя и его положение в установленных интервалах (измерительная диаграмма)
  • Сравнение текущего значения показателя со значениями в прошлых периодах

Какие данные можно проконтролировать в таблице?

С помощью таблицы производится контроль текущего значения ключевых показателей по статусу, тренду и характеру изменений или структуре. В таблице отображается развернутая информация о структуре текущего значения. Например, выручка по контрагентам или по менеджерам. Если это предусмотрено настройками, то в таблице можно вывести итоговые значения.

Анализ с использованием монитора целевых показателей

При анализе целевых показателей можно использовать следующие возможности.

  • Проводить анализ взаимосвязанных показателей.
  • Анализировать данные по другому объекту анализа.
  • Изменять периодичность данных.
  • Расшифровывать показатели, используя дополнительные отчеты.
  • Выводить подробное описание варианта анализа.
  • Формировать сводный отчет по анализу всех представленных показателей.

Как произвести анализ взаимосвязанных показателей?
  • Установите курсор на стрелку , отображенную в варианте анализа.
  • Нажмите на правую клавишу мыши для вызова контекстного меню.
  • Выберите пункт Взаимосвязи показателей.

В зависимости от структуры целей в контекстном меню может быть два варианта отображения взаимосвязанных показателей: влияющие показатели и зависимые показатели. При исполнении команды открывается отдельное окно, в которое выводится информация об анализируемом показателе и влияющем (зависимом) показателе. Для каждого показателя выводится диаграмма и справка о ключевых свойствах показателя - цель, целевые значения, фактическое состояние, формула.

Можно ли при анализе показателя быстро изменить объект анализа?

Изменение объектов анализа предусмотрено для диаграмм и таблиц вариантов анализа с типом анализа "Покомпонентное сравнение". Объекты , по которым возможно проведение анализа, должны быть прописаны в шаблонах расчета (СКД).

Пример. В качестве объекта анализа выручки была выбрана организация. В шаблонах расчета варианта анализа предусмотрено также возможность проведения покомпонентного сравнения по партнерам и менеджерам. Пользователь может на ходу изменить объект анализа и проанализировать выручку предприятия в разрезе партнеров или менеджеров.

Для изменения объекта анализа следует выполнить следующие действия.

  • Установите курсор на стрелку, отображенную в варианте анализа.
  • Нажмите на правую клавишу мыши для вызова контекстного меню.
  • Выберите пункт Объекты анализа.
  • Выберите нужный объект анализа из представленного списка.

Система пересчитывает "на лету" вариант анализа с учетом нового объекта анализа. При этом выбранный новый объект анализа сохраняется только до момента закрытия формы (новый вариант анализа не создается). При повторном открытии монитора целевых показателей вариант анализа будет отображен с объектом анализа по умолчанию.

Информация об измененном объекте анализа также передается в форму анализа взаимосвязанных целевых показателей.

Можно ли изменить периодичность расчета целевого показателя, не добавляя нового варианта анализа?

Изменить периодичность анализа можно для тех вариантов анализа, для которых установлен тип "Динамика изменений". Доступны различные шаги периодичности расчетов: день, неделя, декада , месяц и т.д.

Важно. Доступность расчета показателя с различной периодичностью должна быть предусмотрена в шаблоне расчета целевого показателя.

Для изменения периодичности следует выполнить следующие действия.

  • Установите курсор на стрелку , отображенную в варианте анализа.
  • Нажмите на правую клавишу мыши для вызова контекстного меню.
  • Выберите пункт Периодичность.
  • Выберите нужный вариант периодичности из представленного списка.

Система пересчитывает "на лету" вариант анализа с учетом новой периодичности. При этом выбранная периодичность сохраняется только до момента закрытия формы (новый вариант анализа не создается). При повторном открытии монитора целевых показателей вариант анализа будет отображен с периодичностью по умолчанию.

Информация об измененной периодичности также передается в форму анализа взаимосвязанных целевых показателей.

Как расшифровать информацию о показателе, используя дополнительные отчеты?

  • Установите курсор на стрелку , отображенную в варианте анализа.
  • Нажмите на правую клавишу мыши для вызова контекстного меню.
  • Выберите пункт Отчеты.
  • Выберите нужный отчет из представленного списка.

Список отчетов, которые отображаются в списке, определяется на этапе заполнения варианта анализа (отчеты).

Данные в отчете будут представлены с той периодичностью, которая указана в варианте анализа.

Важно. Установленные в варианте анализа отборы и показатели передаются в отчет только в том случае, если имена в отчете для расшифровки и в шаблоне расчета варианта анализа совпадают.

Как посмотреть подробную информацию о показателе, который анализируется?
  • Установите курсор на стрелку , отображенную в варианте анализа.
  • Нажмите на правую клавишу мыши для вызова контекстного меню.
  • Выберите пункт Анализ показателя.
  • Будет показана детальная информация о показателе, варианте его расчета и фактических данных.

Информация может быть распечатана или отправлена по электронной почте руководителю.
