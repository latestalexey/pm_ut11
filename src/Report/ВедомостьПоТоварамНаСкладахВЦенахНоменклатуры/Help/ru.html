Отчет "Ведомость по товарам на складах в ценах номенклатуры"

Отчет предназначен для оценки стоимости товаров на складах по любому из выбранных видов цен, зарегистрированных в справочнике Виды цен. Используется для оценки стоимости товаров в розничных ценах на складах типа Розничный магазин.

Отчет может быть детализирован по документам движения. Для этого используется вариант отчета По документам движения (регистратору).
