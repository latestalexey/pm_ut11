В каких варинтах может быть сформирован данный отчет?

Отчет может быть сформирован в варианте "Контактная информация по партнерам" и "Контактная информация по контактным лицам". Вариант определяется при вызове отчета и не может быть изменен в форме отчета.

Как установить отбор по партнерам, информацию по которым требуется получить в отчете?

Необходимо воспользоваться командой "Отбор по партнерам", расположенной в панели настроек в правой части отчета. Если панель настроек скрыта, то предварительно необходимо вывести отобразить панель, воспользовавшись командой "Отобразить настройки"

Какая информация выводится в варианте "Контактная информация по партнерам"?

Выводится информация о партнерах и их выбранных для вывода данных. Для каждой отобранной для вывода ролей контактного лица формируется отдельная колонка, в которой выводятся контактные лица партнера данной роли, и их контактная информация.

Какие настройки можно применить в варианте отчета "Контактная информация по партнерам"?

  • Определить какую информацию о партнере необходимо вывести.
  • Определить контактные лица каких ролей должы быть выведены.
  • Определить какие виды контактной информации должны быть выведены.

Важно. Если выбрана хотя бы одна роль контактного лица, то необходимо выбрать хотя бы один вид конактной информации и наоборот.

Какая информация выводится в варианте "Контактная информация по контактным лицам"?

Выводится информация о контактных лицах, сгруппированных по партнеру. Устанавливается отбор по ролям контактных лиц согласно настройкам. В отдельных колонках выводятся те виды контактной информации, которые указаны в настройках отчета.

Какие настройки можно применить в варианте "Контактная информация по контактным лицам"?

  • Определить контактные лица каких ролей должы быть выведены.
  • Определить какие виды контактной информации должны быть выведены.

Важно. Необходимо выбрать хотя бы одну роль контактного лица и хотя бы один вид контактной информации.
