Отчет "План-фактный анализ работы торговых представителей"

Отчет предназначен для сравнения плановых количественных показателей торгового представителя с фактическими показателями.

Форма отчета зависит от установленного параметра детализации в настройках параметров учета.

  • При использования детализации до уровня номенклатурных позиций используется два варианта отчета. Отчет может быть сгруппирован по полям Номенклатура/Торговый представитель или Торговый представитель/Номенклатура.
  • Для детализации отчета по заданиям следует установить флаг "Детализировать по заданиям". В отчете можно установить отбор по конкретным партнерам (клиентам), с которыми работает торговый представитель.
