Отчет "Товары в ячейках"

Отчет предназначен для анализа размещения товаров в ячейках. В отчете отображаются следующие данные.

Количество (упаковок)

  • В наличии - общее количество товара, которое лежит в ячейке.
  • К отбору - количество, которое лежит в ячейке, но по которому уже есть задания на отбор (в статусе "Подготовлено" и "В работе").
  • К размещению - количество, которое еще не лежит в ячейке, но по которому уже есть задания на размещение (в статусе "Подготовлено" и "В работе").

Объем ( Вес)

  • Занятый - Объем (вес), занятый размещенным в ячейке товаром. Рассчитывается на основании заданного объема (веса) упаковки товара и количества товаров, находящегося в ячейке.
  • Зарезервированный - Объем (вес), который будет занят, после выполнения заданий на размещение.
  • Освобождаемый - Объем (вес), который освободится после выполнения заданий на отбор по этой ячейке.
  • Свободный - Объем (вес), который сейчас не занят товаром. Информация вычисляется на основании данных о типоразмере ячейке и проценту ее заполнения по объему (весу).

Для отчета предусмотрены следующие варианты формирования.

  • По ячейке. Используя этот вариант отчета можно узнать, что лежит в конкретной ячейке, сколько в нее поместится еще товара или понять, почему программа туда не размещает новый товар. Возможно, после этого коэффициенты наполненности надо будет поправить
  • По номенклатуре. Используя этот вариант отчета можно узнать, в каких ячейках лежит конкретный товар.
  • По области хранения. Используя этот вариант отчета можно узнать, какие складские группы номенклатуры и складские группы упаковок лежат в этой области, чтобы, например, корректировать правила размещения. Можно также выяснить сколько в области хранения свободного места, чтобы планировать поступления товаров и подготавливать области хранения для размещения. Предусмотрен также вариант "По области хранения (детально)" с указанием конкретных товаров.
  • По складской группе номенклатуры . Используя этот вариант отчета можно узнать, насколько разбросаны, например, чайники по складу. При этом может интересовать как обобщенная информация - чайники лежат все в одной области хранения, так и детальная - чайник 2 л разбросан по 20 ячейкам ( не порядок, надо перемещать).
