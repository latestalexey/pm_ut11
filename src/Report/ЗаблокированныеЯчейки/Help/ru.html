Отчет "Отчет по заблокированным ячейкам"

В отчете показывается список заблокированных ячеек. Блокировка ячеек производится отдельным документом "Установка блокировок ячеек". Также ячейки автоматически блокируются в момент оформления пересчета товаров (в документе "Пересчет товаров" установлен статус "В работе").

Предусмотрено три типа блокировки ячеек:

  • Блокировка ячеек для размещения товаров - Размещение.
  • Блокировка ячеек для отбора из них товаров - Отбор.
  • Полная блокировка на отбор и размещение товаров - Полная.

В отчете можно сгруппировать и отобрать ячейки по типу блокировки. Для каждой ячейки показывается тот документ, которым были заблокированы ячейки. Для отмены блокировки конкретной ячейки можно открыть документ блокировки непосредственно из отчета и отменить блокировку конкретной ячейки (установив флажок "Отменено" для конкретной ячейки). В отчете предусмотрена возможность отбора по конкретному складу. Если склад разделен на помещения , то в отчете ячейки будут сгруппированы по помещениям.
