Отчет "Доступность вариантов анализа целевых показателей"

Как проверить для всех ли вариантов анализа установлены ответственные и установлен ли для них доступ к данным вариантам анализа?

Для этих целей используется отчет Отчет по доступу (по ответственным). Используя этот отчет можно получить ответ на следующие вопросы.

  • Как проконтролировать список тех вариантов анализа, по которым не определен ответственный? Установите флажок в поле Ответственный, поле выбора оставьте пустым. Непостредствеено из отчета можно открыть карточку варианта анализа и назначить ответственного.
  • Кто является ответственным за разработку вариантов анализа по конкретному целевому показателю? Установите отбор по конкретному целевому показателю в поле "Цель".
  • Все ли варианты анализа, по которым сотрудник является ответственным доступны ему для анализа? Установите отбор по конкретному ответственному. В отчете будет показа информация о тех вариантах анализа, по которым пользователь является ответственным. Проверьте доступность этих вариантов анализа.
