Какие параметры отбора могут быть установлены пользователем при настройке отборов, ограничивающих действие источника?

Список параметров, которые доступны пользователю для настройки ограничения действия источника определяются при создании источника данных (см. Как указать при настройке источников данных тот список полей, по которым пользователь сможет производить отбор данных в источнике?). В форме настройки пользователь может выбрать конкретные значения (отбор по конкретной группе номенклатуры, отбор по подразделению, складу). Если для пользователя определена возможность установки отбора по произвольным значениям реквизитов, то в списке параметров отбора появится дополнительное поле с названием "Отбор". Установленные пользователем параметры отбора будут сохранены при записи документа "План продаж и внутреннего потребления".
