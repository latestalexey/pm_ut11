Как заполнить информацию о товарах в табличную часть документа планирования вручную?

Для добавления товаров в табличную часть документа используется кнопка "Добавить". В табличную часть документа могут быть добавлены только товары. Для каждого товара добавляется следующая информация.

  • Номенклатура. Информация о товаре выбирается из справочника номенклатуры. Если по номенклатуре ведется учет в разрезе характеристик, то необходимо детализировать план продаж по каждой характеристике номенклатуры. Если для товара предусмотрен учет в разрезе упаковок, то указывается конкретная упаковка, в которой планируется продажа товаров и внутреннего потребления. По умолчанию заполняется базовая упаковка товаров.
  • Количество. Количество товаров, которое планируется продать или отгрузить для внутреннего потребления. Количество товаров указывается по каждому складу. Количество товаров может быть указано на весь период или распределено по датам отгрузки. Если количество товаров указывается на весь период, то в качестве даты отгрузки указывается начальная дата периода планирования. В этом случае распределение по датам отгрузки в соответствии с тем календарем рабочих дней, который указан для склада производится в обработке "Управление запасами".

В какой момент сохраняются правила заполнения плана продаж и внутреннего потребления?

Правила заполнения плана продаж и внутреннего потребления сохраняются в момент записи документа (кнопка "Записать объект"). Документ может быть проведен без предварительной записи (кнопка "Провести и закрыть"). При проведении происходит автоматическая запись документа и информация о настройках правил заполнения также будет сохранена.

Какие статусы могут быть установлены в документе "План продаж и внутреннего потребления"?

В документе могут быть установлены следующие статусы.

  • Создан. Статус устанавливается для вновь созданного документа.
  • На согласовании. Статус устанавливается в том случае, если составленный план продаж и внутреннего потребления требует дополнительного согласовании. На основании документа может быть создан документ "Задание", с помощью которого руководителю подразделения отправляется задача на согласование плана продаж и внутреннего потребления.
  • Утвержден . После согласования с вышестоящим руководством для плана продаж и внутреннего потребления устанавливается статус "Утвержден".

Важно. Планируемое количество товаров, указанное в документе "План продаж и внутреннего потребления", будет учитываться при расчете необходимых запасов товаров в обработке "Управление запасами" только по утвержденным планам, то есть только по тем планам продаж и внутреннего потребления, в которых установлен статус "Утвержден".
