Какие правила предусмотрены при заполнении плана?

При заполнении плана могут быть использованы следующие правила.

  • Обновить состав плана. При установке этого флага, информация в табличной части документа будет очищена и будет заполнена заново в соответствии с сформированными данными.
  • Дополнить состав плана. При установке этого флага, информация в табличной части документа будет дополнена новыми данными, ранее заполненные данные в табличной части документа сохранятся.

Как заполнить список источников, по которым будут сформированы данные в плане продаж и внутреннего потребления?

  • Для добавления нового источника нажмите на кнопку "Добавить новый источник".
  • В появившейся новой строке нажмите на кнопку выбора и выберите из списка источника нужный источник.
  • Аналогичным образом добавьте все остальные источник, данные которых будут использоваться при формировании плана.

Для каждого источника необходимо указать период, который ограничивает выборку данных в соответствии с правилами формирования данных или конкретную дату, на которую будут браться данные из источника. Что именно указывается для источника: период или дата, определяется теми правилами, которые заданы при создании нового источника.

Можно ли для источников данных установить дополнительные правила отбора данных при заполнении данных в плане?

При указании источника при заполнении плана можно для каждого из источников установить дополнительные правила отбора. Список тех полей, которые будут доступны пользователю для установки отбора данных источника определяется при создании источника данных. Для установки отбора данных в источнике данных следует выполнить следующую последовательность действий.

  • Установить курсор на строку с нужным источником в списке.
  • Нажать на пиктограмму в графе "Настройка".
  • В появившемся диалоговом окне установить нужные параметры отбора.
  • Нажать на кнопку "Завершить редактирование".

Установленные настройки отбора будут сохранены после записи документа "План продаж и внутреннего потребления".

Как определить правила совместного действия нескольких источников?

Правила действия совместного применения источников определяются для группы источников. Если источники не объединены в группы, то для всех этих источников используется правило "Сложение". Знак, указанный рядом с источником, определяет с каким знаком происходит сложение источников.

Пример. В список источников введено два источника: объем продаж за прошлый месяц и остатки товаров на дату начала периода планирования. Для источника "Остатки товаров" установлен знак "-". Группировки для источников не установлено. Это означает, что при составлении плана продаж из объема рассчитанных количественных данных по объему продаж будут учтены данные о текущих остатках (планируемый объем продаж будет уменьшен на величину текущих остатков товаров).

Для определения правила совместного действия нескольких произвольно выбранных источников их необходимо объединить в группы. Для объединения источников в группы следует выполнить следующую последовательность действий.

  • В списке источников отметьте те источники, которые надо объединить в группу.
  • Нажмите на кнопку "Сгруппировать источники".
  • Будет создана новая группа. Для группы будет определено правило "Сложение". Можно установить другой порядок взаимодействия источников : Максимум, Среднее или Минимум.

Пример. При составлении плана продаж и внутреннего потребления на апрель месяц необходимо выбрать максимальный объем продаж за месяц в течение первого квартала (январь, февраль, март). В этом случае в список источников добавляется три одинаковых источника данных: объем продаж за месяц. Для каждого из источников указывается разный период отобра данных источника: январь, февраль, март. Эти источники группируются и для них указывается правило: Максимум.

При определении правил совместного действия можно создавать любое количество групп источников и затем для каждой из групп определять правило совместного использования. Например, можно определить максимальный объем продаж за месяц в течение квартала, среднее количество заказываемых товаров в течение квартала и выбрать максимальное значение из этих двух показателей.

Можно ли после расчета данных в соответствии с правилами совместного применения различных источников, увеличить или уменьшить полученные показатели планирования?

Полученные в результате расчета данных показатели планирования можно увеличить или уменьшить на определенный процент.

  • Если Вы хотите увеличить показатели планирования, то в поле "Изменить результат на (%)" установите положительное значение показателя.
  • Если Вы хотите уменьшить показатели планирования, то в поле "Изменить результат на (%)" установите отрицательное значение показателя.

Можно ли при составлении данных округлить данные планирования до целого числа?

Для округления данных планирования до целого числа установите в поле "Округлить до " значение "1". Точность округления может быть выбрана любая.

Можно ли установить дополнительный отбор на те данные, которые были рассчитаны при заполнении плана на основании данных источников?

После расчета данных в соответствии с источниками можно установить дополнительные отборы. Например, при планировании продаж в конкретном розничном магазине, отобрать только те продажи, которые были произведены в конкретном розничном магазине (складе). Список установленных дополнительных отборов определяется списком тех полей, которые заполняются в табличной части документа. Используя графу "Отбор" можно установить отбор по любому из реквизитов, например по дополнительному реквизиту в справочнике "Номенклатура".

Как заполняется информация о количестве и дате отгрузки при автоматическом заполнении плана?

При автоматическом заполнении плана информация о количестве и дате отгрузки заполняется в зависимости от установленного флага "Распределить по рабочим дням".

  • Если флаг "Распределить по рабочим дням" не установлен, то в поле "Количество" устанавливается планируемое количество с учетом всех тех параметров, которые указаны в настройках заполнения плана, а в качестве даты отгрузки устанавливается дата начала периода планирования. Распределение по рабочим дням работы склада в этом случае производится в обработке "Управление запасами".
  • Если флаг "Распределять по рабочим дням" установлен, то количество планируемого товара распределятся пропорционально тому количеству дней, которые определены в календаре, указанном в карточке склада, как рабочие дни склада. В качестве дат отгрузки будут указаны те рабочие дни, которые определены, как рабочие дни склада.
