Можно ли провести перерасчет только нескольких позиций списка?

Для расчета цен не по всем, а только по отмеченным позициям, необходимо выполнить следующие действия:

  • Выделить в списке те позиции, по которым надо рассчитать цены.
  • Нажать на кнопку "Изменить цены".
  • Выбрать пункт меню "Рассчитать вычисляемые (по данным ИБ и формулам)". Появится диалоговое окно выбора параметров расчета цен.
  • В появившемся диалоговом окне установить переключатель "Только выделенные";
  • Нажать на кнопку "Рассчитать".

Аналогичным образом можно рассчитать цены только по тем позициям, по которым цены еще не рассчитаны. В этом случае в диалоговом окне настройки расчета цен необходимо установить переключатель "Только незаполненные".
