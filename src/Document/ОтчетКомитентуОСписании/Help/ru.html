Документ "Отчет комитенту о списании"

Документ "Отчет комитенту о списании" предназначен для отражения факта списания товаров, принятых на комиссию от комитента. Документ может быть оформлен только по тому соглашению, в котором указан вид операции "Прием на комиссию".

Важно. Документ оформляется в тех случаях, если в период , указанный в отчете комитенту, были зафиксированы факты списания товаров, принятых на комиссию. Списание товаров, принятых на комиссию, оформляется документом "Внутреннее потребление товаров" с установленным видом операции "Списание на расходы" или документом "Списание недостач товаров".

Данные о списанных товарах фиксируются за тот период, который указан в отчете комитенту. Количество списанных товаров регистрируется в графе "Списано". Товары в табличную часть документа могут быть подобраны в соответствии с результатами списания товаров, зарегистрированными за период формирования отчета (Заполнить-Заполнить по результатам списания). Если необходимо отчитаться перед комитентом не за все, а только за часть списанных товаров, то используется вариант заполнения "Подобрать по результатам списания ".

Товар списывается по тем ценам, по которым он принимался на комиссию ("Цены комитента"). Цены, по которым товар принимался на комиссию автоматически заполняются в соответствии с теми ценами, которые указаны в соглашении с комитентом, но могут быть изменены пользователем произвольным образом. Для восстановления цен в соответствии с соглашением можно воспользоваться кнопкой автоматического заполнения "Заполнить - Заполнить цены по соглашению".

На странице "Дополнительно" указывается применяемый порядок налогообложения при списании товаров и та дата, когда необходимо оплатить списанные товары комитента ("Дата платежа").

Важно. По взаимной договоренности комиссионер может не оплачивать стоимость списанных товаров комитента, в этом случае для товаров не заполняется информация о цене списания (цена комитента), товар списывается по нулевой стоимости.
