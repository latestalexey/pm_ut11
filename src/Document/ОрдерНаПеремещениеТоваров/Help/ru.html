Документ "Ордер на перемещение товаров"

Документ "Ордер на перемещение товаров" предназначен для оформления перемещения товаров между помещениями одного склада (складской территории).

Важно. Перемещение товаров между помещениями оформляется одним документом, без использования ордерной схемы перемещения.

Перемещение товаров может быть оформлено между помещениями с различными вариантами хранения товаров в складских ячейках в помещениях: в одном из помещений может использоваться справочное хранение товаров в складских ячейках, а в другом помещении - адресное хранение товаров в складских ячейках.

Перемещение товаров между помещениями может быть оформлен поэтапно. Каждый этап перемещения между помещениями регистрируется с помощью статуса, устанавливаемого в документе.

  • В работе. Товар подготавливается к отгрузке из помещения-отправителя.
  • К отгрузке. Товар подготовлен и отгружен из помещения-отправителя.
  • Принят. Товар принят в помещении-получателе.
