Как создать задание на пересчет товаров?

  • В списке документов "Пересчеты товаров" перейдите на страницу "Распоряжения на пересчет товаров (Приказы на инвентаризацию)".
  • Установите курсор на тот приказ на инвентаризацию, в рамках которого должно быть создано задание на пересчет товаров.
  • Нажмите на кнопку "Создать пересчет товаров".
  • Будет создан новый документ "Пересчет товаров".

Можно ли распечатать несколько заданий на пересчет товаров?
  • В списке документов выделите те документы пересчета товаров, по которым необходимо распечатать. Документы должны быть проведены и для них должен быть установлен статус "В работе".
  • Нажмите на кнопку "Печать" на командной панели списка документов.

Будет распечатан список заданий на пересчет товаров. Каждое задание будет распечатано на отдельном листе.
