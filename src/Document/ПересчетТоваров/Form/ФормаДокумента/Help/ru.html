Для каких целей при пересчете товаров необходимо указывать информацию об инвентаризации товаров, в рамках которой производится пересчет товаров?

Информация об инвентаризации, в рамках которой производится пересчет товаров ("Приказ на проведение инвентаризации товаров") заполняется в поле "Основание". Эта информация является обязательной, поскольку в рамках приказа на инвентаризацию фиксируются все пересчеты товаров, произведенные на складе или в различных помещениях одного склада. При этом пересчеты могут быть произведены несколькими исполнителями по разным группам товаров. По результатам всех пересчетов будут впоследствии сформированы результирующие документы по излишкам и недостачам товаров, а также выявлены пересортицы товаров. (см. Как зарегистрировать в программе выявленные в ходе пересчета на складе излишки, недостачи или порчу товаров?)

Как заполнить информацию о товарах , по которым необходимо производить пересчет?

После указания склада (помещения) в документе нажмите на кнопку "Заполнить" в командной панели табличной части документа. Табличная часть документа будет заполнена всеми товарами, по которым имеются остатки на указанном складе (помещении). Информация о товарах будет заполнена с детализацией по характеристикам.

Важно. Если в качестве склада (помещения) указан склад (помещение), для которого ведется адресное хранение остатков товаров в ячейках, то дополнительно для каждого товара будет показан список ячеек, в которых хранится товар. При этом, если товар хранится в нескольких ячейках, то будет сформировано несколько записей для одного товара. Для каждой ячейки будет определена та упаковка, в которой товар в ней хранится.

Учетное количество товаров заполняется в момент первого проведения документа "Пересчет товаров" в статусе "В работе". При этом фиксируются остатки товаров на момент проведения документа. При изменении списка товаров или даты документа информация об учетном количестве автоматически перезаполняется после перепроведения документа.

Важно. При проведении документа "Пересчет товаров" в статусе "В работе" на адресном складе автоматически блокируются те ячейки, которые указаны в документе.

Можно ли отобрать для пересчета только одну группу товаров?
  • В группе "Условия пересчетов товаров" нажмите на кнопку выбора в поле "Отбор".
  • Появится диалоговое окно "Редактирование отбора".
  • Установите нужные параметры отбора, например "Номенклатура" -"В группе" - "Бытовая техника".
  • Нажмие на кнопку "ОК"в диалоговой форме "Редактированние отбора".
  • Параметры отбора будут заполнены в форме документа "Пересчет товаров".
  • Нажмите на кнопку "Заполнить". Будут заполнены только те товары, которые удовлетворяют указанным параметрам отбора. В рассматриваемом примере будут отобраны товары группы "Бытовая техника".

Как распечатать задание на пересчет товаров?
  • После заполнения в документе списка товаров, которые необходимо пересчитать, в поле "Исполнитель" укажите того работника, который будет осуществлять пересчет товаров.
  • Проверьте, что для документа установлен статус "В работе".
  • Если необходимо, чтобы в задании было распечатано учетное количество товаров, то установите флаг "Печатать количество по учету". Учетное количество будет заполнено в момент проведения документа.
  • Нажмите на кнопку "Провести".
  • Нажмите на кнопку "Печать" и выберите пункт "Задание на пересчет".

Будет распечатано задание на пересчет товаров.

Важно. При выдачи заданий на пересчет товаров на складе с адресным хранением товаров в ячейках, после проведения документа "Пересчет товаров" в статусе "В работе", все ячейки, указанные в документе, автоматически блокируются для отбора и приема товаров на время пересчета товаров (пока не будет отмечено выполнение задания по пересчету).

Как ввести информацию о представленных исполнителем результатах по пересчету товаров по заданию на пересчет?

Документ необходимо провести в статусе "Внесение результатов". Информация о количестве товаров, которое было зарегистрировано в процессе пересчета, вводится в поле "Факт".

  • Если в процессе пересчета не было выявлено никаких расхождений, то фактическое количество заполняется равному учетному с помощью команды табличной части "Все действия" - "Фактическое количество по учетному".
  • Если в процессе пересчета был обнаружен товар, который не был зарегистрирован в выданном задании, то информация об этом товаре добавляется в табличную часть документа с помощью кнопки "Добавить" и для товара указывается его фактическое количество. Если пересчет товаров производится на складе с адресным хранением остатков товаров в ячейках, то дополнительно указывается та ячейка, в которой в процессе пересчета был обнаружен "лишний" товар.

Важно. При работе на складе с адресным хранением, блокировка ячеек в момент проведения документа "Пересчет товаров" со статусом "Внесение результатов", регулируется флажком "Блокировка ячеек"

Как зарегистрировать введенные результаты пересчетов товаров?

После заполнения информации о результатах пересчета товаров и проверки этих результатов ответственным сотрудником установите в документе статус "Выполнено" и нажмите на кнопку "Провести и закрыть".

Важно. При работе на складе с адресным хранением проведение документа "Пересчет товаров" со статусом "Выполнено" автоматически снимает блокировку с тех ячеек, которые указаны в документе. Ячейки становятся доступными для отбора и размещения в них товаров.

Что делать если в процессе пересчета обнаружился бракованный товар?

Если в процессе пересчета обнаружился бракованный товар, то в документе можно отразить изменение качества у такого товара. Строка с таким товаром должна быть предварительно разбита на две строки - изменять качество товара необходимо у вновь введенной строчки. Кроме этого, в случаях пересчета на складе с адресным хранением товаров, бракованный товар в результате проведения документа может быть перемещен в отдельную ячейку (например, на складе может быть создана специальная ячейка для бракованных товаров).

Как можно быстро разбить строчку с одним товаром на две?

Для этой операции необходимо воспользоваться командой табличной части пересчета "Все действия" - "Разбить строку". После выполнения команды программа выдаст запрос количества - нужно ввести количество бракованного товара. После этого в табличную часть будет добавлена новая строка с указанным количеством товара. Это же количество товара будет уменьшено в строке к которой применялась команда.

Как изменить качество товара для вновь введенной строчки?

После того как в документ добавлена новая строка с бракованным товаром и указано количество бракованного товара, необходимо выделить добавленную строку и воспользоваться командой табличной части "Изменить качество". После выполнения команды будет открыта форма "Выбор товара другого качества" в которой можно выбрать товар с другим качеством (если ранее для текущей позиции уже создавался товар с другим качеством), либо создать его и выбрать (подробнее см. справку формы "Выбор товара другого качества").

Важно. Изменить качество товара в строке имеющей заполненное количество в поле "По учету" нельзя, поэтому менять качество необходимо у вновь введенной строчки.

Как на складе с адресным хранением в процессе пересчета, можно переместить бракованный товар в отдельную ячейку?

Предварительно на складе необходимо выделить специальную ячейку в которой будет храниться бракованный товар. Далее, в документе пересчета на странице "Дополнительно", нужно установить флажок "Перемещять испорченные товары в ячейку" и указать нужную ячейку. После проведения документа в статусе "Выполнено" бракованный товар будет перемещен в указанную ячейку.

Как в документе можно получить данные из терминала сбора данных?

Для того чтобы загрузить данные из терминала сбора данных в табличную часть "Товары" необходимо воспользоваться командой табличной части "Все действия" - "Загрузить товары из ТСД". Если терминал сбора данных подключен и настроены параметры его подключения, то табличная часть заполниться данными из терминала. В противном случае программа выдаст предупреждение. Если из теминала будет получен штрих код не зарегистрированый в информационной базе, то программа предложит зарегистрировать его и сопоставить с имеющимся в информационной базе товаром.

Как можно добавить данные в табличную часть "Товары" с помощью сканера?

Если сканер подключен и настроены параметры его подключения, то после считывания штрих кода в табличную часть будет добавлен товар соответствующий штрих коду. Если товар уже присутствует в табличной части, то будет увеличено его количество. Если в программе не зарегистрирован считываемый штрих код, то программа предложит зарегистрировать его и сопоставить имеющемуся в информационной базе товару.

Как найти в табличной части товар используя сканер штрих кода?

Для того что бы выполнить поиск товара в табличной части, необходимо воспользоваться командой табличной части "Все действия" - "Найти товары по штрих коду". Далее в открывшейся форме считать штрих код товара. Если товар со считанным штрих кодом присутствует в табличной части, то он будет выделен (на него будет установлен курсор). Если в программе не зарегистрирован считываемый штрих код, то программа предложит зарегистрировать его и сопоставить имеющемуся в информационной базе товару.

Как с помощью весов можно заполнить количество в табличной части документа?

С помощью команды табличной части документа "Получить вес", поле "Количество" (количество упаковок)
текущей строки документа заполнится весом, полученным из подключенных электронных весов. Электронные весы должны быть подключены к компьютеру, а так же настроены параметры их подключения.
