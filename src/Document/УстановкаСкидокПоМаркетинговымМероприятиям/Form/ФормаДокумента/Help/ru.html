Работа с документом "Установка скидок по маркетинговым мероприятиям"

Как ограничить действие скидок (наценок) в рамках маркетингового мероприятия для определенной группы (сегмента) партнеров?

Скидки (наценки) будут действовать только для тех партнеров, которые относятся к тому сегменту партнеров, который указан в документе "Установка скидок по маркетинговым мероприятиям". Сегмент партнеров заполняется автоматически после указания маркетингового мероприятия (если он указан в маркетинговом мероприятии). Пользователь может изменить сегмент партнеров или выбрать другой сегмент партнеров из списка сегментов.

Важно. Ограничение сегмента партнеров имеет смысл только при назначении скидок (наценок) в оптовой торговле.

Как ввести информацию о скидках (наценках), которые будут действовать в рамках маркетингового мероприятия?

Информация о скидках (наценках), которые будут действовать в рамках маркетингового мероприятия, указанного в документе, регистрируется в табличной части документа "Установка скидок по маркетинговым мероприятиям". Для каждой скидки (наценки) указывается срок ее действия в рамках маркетингового мероприятия.

Важно. Если в рамках маркетингового мероприятия в один и тот же период действует несколько скидок (наценок) то правила их взаимодействия определяются теми правилами взаимодействия, которые определены для списка скидок (наценок).

Как ввести информация о новой скидке (наценке) в документ "Установка скидок по маркетинговым мероприятиям"?
  • В табличной части документ "Установка скидок по маркетинговым мероприятиям" нажмите на кнопку "Добавить".
  • В табличной части документа укажите склад, при оформлении продажи с которого будет действовать скидка.
  • Выберите из списка скидок (наценок) нужную скидку (наценку).
  • В полях "Дата начала" и "Дата окончания" укажите период действия скидки (наценки).
  • Аналогичным образом введите информацию обо всех других скидках.

Важно. Введенная скидка будет действовать только при оформлении документов с конкретного склада. Если скидка должна действовать при оформлении документов с нескольких складов (розничных магазинов), то необходимо ввести информацию по каждому складу (розничному магазину).

Как ввести в действие документ "Установка скидок по маркетинговым мероприятиям".

Для ввода в действие документа "Установка скидок по маркетинговым мероприятиям" нажмите на кнопку "Провести и закрыть".

Скидки будут действовать в указанный для них период действия в соответствии с теми ограничениями, которые для них указаны.

Существуют ограничения по действию скидок при розничных продажах (оформленных с помощью документа "Чек ККМ").

  • Будут действовать только скидки (наценки), в которых определены условия "За разовый объем продаж", "За время продажи", "За форму оплаты".
  • Действие розничной скидки ограничивается указанным в документе складом.

При назначении скидок в оптовой торговле ограничений по условиям применения скидок нет. Могут применяться скидки с любыми условиями. Применение оптовой скидки в рамках маркетингового мероприятия ограничивается сегментом партнеров и тем складом, который указан для предоставления скидки.
