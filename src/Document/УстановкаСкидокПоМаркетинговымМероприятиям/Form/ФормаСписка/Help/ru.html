Как создать новый документ "Установка скидок по маркетинговым мероприятиям"?
  • В списке документов "Установка скидок по маркетинговым мероприятиям" нажмите на кнопку "Создать".
  • Будет создан новый документ "Установка скидок по маркетинговым мероприятиям".
  • В новом документе необходимо указать маркетинговое мероприятие и ввести список скидок (наценок), которые будут действовать в рамках маркетингового мероприятия.
