Документ "Установка скидок по маркетинговым мероприятиям"

Документ "Установка скидок по маркетинговым мероприятиям" предназначен для ввода в действие скидок (наценок), зарегистрированных в рамках маркетингового мероприятия (маркетинговой акции). Скидки (наценки), указанные в документе действуют как при продаже товаров оптовым клиентам, так и при продаже товаров в розницу с помощью документа "Чек ККМ".

Информация о маркетинговом мероприятии указывается на закладке "Дополнительно". Указание маркетингового мероприятия не является обязательным, документ "Установка скидок по маркетинговым мероприятиям" может быть оформлен без указания маркетингового мероприятия.

Важно. Для одного склада (розничного магазина) можно указать несколько различных скидок (наценок). В этом случае скидки (наценки) будут действовать в соответствии с теми правилами взаимодействия, которые установлены в списке скидок (наценок).

Важно. При оптовой торговле действие скидки (наценки) может быть ограничено не только конкретным складом, но и сегментом партнеров. При оптовой торговле скидки (наценки) по маркетинговым мероприятиям применяются совместно с теми скидками (наценками), которые определены в соглашении с клиентом. Правила совместного применения скидок(наценок) определяются в списке скидок (наценок).
