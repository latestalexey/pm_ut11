Документ "Операция по платежной карте"

Документ предназначен для отражения операций оплаты клиентом за товары, работы, услуги с помощью платежных (пластиковых) карт международных платежных систем, и отражения операций возврата денежных средств на платежную карту.

Важно. Для оформления операций по оплате платежными картами необходимо предварительно ввести информацию о договоре эквайринга с банком и об эквайринговом терминале, с помощью которого будут оформляться платежи с помощью платежных карт.

Информацию о проведении банком платежей по платежным картам регистрируется в документе "Отчет банка по операциям эквайринга".
