Задания торговым представителям

Назначение

Форма предназначена для заполнения в задании торговому представителю данных о сумме продаж в соответствии с историей продаж клиентов. В том случае, если используется детализация по номенклатурным позициям заполняется список товаров с указанием количественных характеристик.

За какой период производится анализ продаж?

Анализ продаж клиента может производиться за любой период. Период указывается в настройках формы.

Какие варианты расчета показателей предусмотрены?

При расчете объема продаж на основании истории продаж могут быть указаны различные способы расчета среднего объема за анализируемый период:

- в среднем за день - объем вычисляется как средний за день. Данный способ может быть удобен, если от клиента поступает несколько заказов в день, но задание формируется обобщенно, на день.

- в среднем за неделю - объем вычисляется как средний за неделю. Данный способ может быть удобен при планировании заданий сразу на неделю.

- в среднем за месяц - объем вычисляется как средний за неделю. Данный способ может быть удобен при планировании заданий на месяц.

На основании каких данных рассчитывается объем продаж?

Расчет объема продаж производится на основании тех данных , которые зарегистрированы в финансовых документах отгрузки - документах "Реализация товаров и услуг".

Можно ли заполнить информацию о товарах с учетом характеристик?

Да можно. Для этого в форме необходимо установить флаг "Детализировать по характеристикам".

Можно ли в качестве показателя использовать средний объем продаж по группе партнеров?

Да можно. В настройках отбора необходимо указать группу партнеров или произвольный список. В соответствии с указанным вариантом расчета объема продаж ( средний за день, средний за неделю, средний за месяц) будет рассчитан усредненный показатель суммы или количества продаж по группе или списку партнеров.

Как ограничить список номенклатурных позиций?

В настройках обработки можно установить различные варианты отбора номенклатурных позиций: по группе номенклатуры, по виду номенклатуры, по ценовой группе.

Как задать период, за который будет выполняться анализ историй продаж клиента?

Для заполнения информации о периоде нажмите на кнопку выбора интервала. В качестве интервала, за который будет выполняться анализ истории продаж товаров клиенту, можно указать предыдущий месяц, квартал, год и т.д. После выбора интервала будут заполнены конкретные даты. Даты начала и окончания периода анализа продаж могут быть выбраны и вручную. Если даты начала и окончания периода не указаны, то для анализа истории продаж используются данные о всех продажах клиента.

Какие способы расчета объема продаж (планируемого количества товаров) предусмотрены в программе?

При расчете объема продаж на основании истории продаж могут быть указаны различные способы расчета среднего объема за анализируемый период:

  • в среднем за день - объем вычисляется как средний за день. Данный способ может быть удобен, если задание формируется обобщенно, на день для клиента, который оформляет несколько продаж за день.
  • в среднем за неделю - объем вычисляется как средний за неделю. Данный способ может быть удобен при планировании заданий на неделю.
  • в среднем за месяц - объем вычисляется как средний за неделю. Данный способ может быть удобен при планировании заданий на месяц.

В каком случае необходимо устанавливать флаг "Детализировать по характеристикам"?

Устанавливать этот флаг следует в том случае, если необходимо, чтобы товарный состав задания, сформированный на основании истории продаж, был заполнен с учетом конкретных характеристик номенклатуры. В этом случае для товара в табличной части задания будет указана его характеристика и плановое количество товаров будет рассчитано отдельно для каждого товара с конкретной характеристикой.

Если флаг не установлен, то информация о товаре заполнится без характеристики. При расчете планового количества будет рассчитано суммарное количество по всем товарам с характеристиками.

Как заполнить информацию об истории продаж только по товарам конкретной ценовой группы?

При заполнении данных на основе анализа истории продаж может быть задан отбор по номенклатуре, типам и видам номенклатуры. В этом случае в товарный состав задания будут включены только те данные истории продаж, которые удовлетворяют заданному отбору.

При заполнении истории продаж необходимо заполнять информацию только по тем товарам, которые принадлежат ценовой группе "Холодильники". В этом случае устанавливаем флаг в поле "Ценовые группы" и выбираем ценовую группу "Холодильники".

Аналогичным образом можно произвести отбор по определенной группе номенклатуры, виду или типу номенклатуры.

Как задать период, за который будет выполняться анализ историй продаж клиента?

Для заполнения информации о периоде нажмите на кнопку выбора интервала. В качестве интервала, за который будет выполняться анализ истории продаж товаров клиенту, можно указать предыдущий месяц, квартал, год и т.д. После выбора интервала будут заполнены конкретные даты. Даты начала и окончания периода анализа продаж могут быть выбраны и вручную. Если даты начала и окончания периода не указаны, то для анализа истории продаж используются данные о всех продажах клиента.

При расчете плановой суммы задания на основании истории продаж клиента предусмотрены различные способы расчета среднего объема продаж за анализируемый период:

  • В среднем за день - объем продаж вычисляется как средний за день. Данный способ может быть удобен, если от клиента поступает несколько заказов в день, но задание формируется обобщенно, на день.

  • В среднем за месяц - объем продаж вычисляется как средний за месяц. Данный способ может быть удобен при формировании долгосрочных заданий .
