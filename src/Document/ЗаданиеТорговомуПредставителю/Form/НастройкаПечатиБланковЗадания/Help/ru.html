Какие данные можно настроить при печати бланков заданий?
  • Сводное задание.
  • Выводить задачи. Если этот флаг установлен, то в сводном задании будет перечислен не только список клиентов, которых должен посетить торговый предсиавитель, но и список тех вопросов(задач), которые он должен решить с клиентом.
  • Выводить расходы. Будет сформирована дополнительная таблица с указанным количеством строк, в которой торговый представитель будет указывать причину и сумму расходов.
  • Индивидуальное задание.
  • Выводить задачи. При установке флага в задании будет указан список задач, зарегистрированный в задании клиенту.
  • Номенклатура. Дополнительные строки товарного состава. При установке этого флага в бланке задания будет сформирована дополнительная таблица с указанным количеством строк. В эту таблицу торговый представитель будет заполнять список товаров, которые заказал клиент.
  • График оплаты. . Будет сформирована таблица с указанием этапов оплаты, перечисленных в задании.
  • Заметки.Будут сформированы дополнительные строки, в которых торговый представитель будет писать свои заметки после посещения клиента.
