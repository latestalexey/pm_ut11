Отбор коммерческих предложений в списке

Как оставить в списке коммерческих предложений только те предложения, которые оформлены менеджером, работающим с программой?

Для отбора коммерческих предложений текущего пользователя (менеджера) необходимо в поле "Менеджер" выбрать из выпадающего списка значение "Мои документы". . Будут отобраны те коммерческие предложения, в которых менеджер (текущий пользователь) указан в качестве ответственного.

Как оставить в списке только действующие коммерческие соглашения?
  • В поле "Состояние" в нижней части списка выберите значение "Действует".
  • В поле "Срок действия" в нижней части списка выберите значение "Непросроченные".
  • Будут показаны те коммерческие предложения, которые действуют (актуальны) на текущую дату.

Как отобрать коммерческие предложения, по которым завтра истекает срок действия?
  • В поле "Состояние" в нижней части списка выберите значение "Действует".
  • В поле "Срок действия" в нижней части списка выберите значение "Истекающие на дату" и укажите дату, равную послезавтрашней дате.

Будут показаны те коммерческие предложения, у которых завтра истекает срок действия, то есть те коммерческие предложения, у которых в поле "Срок действия" указана дата, которая меньше или равна завтрашней дате.

Как отобрать коммерческие предложения, у которых завтра истекает срок согласования с клиентом?
  • В поле "Состояние" в нижней части списка выберите значение "Ожидается согласование".
  • В поле "Срок выполнгения" в нижней части списка выберите значение "Истекающие на дату" и укажите дату, равную послезавтрашней дате.

Будут показаны те соглашения, у которых установлен статус "Не согласовано" и дата согласования меньше или равна указанной (завтрашней) дате.

В качестве даты согласования используется дата, которая указана в поле "Срок действия". Соглашение должно быть согласовано до момента начала его действия.

Почему коммерческие предложения в списке окрашены разными цветами?

В списке коммерческих предложений используется следующая градация цветов.

  • Кирпичным цветом помечаются все просроченные коммерческие предложения (поля в списке "Текущее состояние" и "Срок действия"), то есть коммерческие предложения, у которых истек срок действия или срок согласования.
  • Серым цветом помечаются закрытые коммерческие предложения.

Примечание. Изменить цвет фона (текста) или условия, по которым помечаются соглашения в списке, а также добавить свои условия, по которым будет раскрашен список, можно в режиме 1С:Предприятия, используя стандартные возможности платформы по настройке условного оформления списка.

Как создать новое коммерческое предложение из списка коммерческих предложений?
  • В списке коммерческих предложений в командной панели нажмите на кнопку "Создать".
  • Будет создано новое коммерческое предложение.
  • Валюта коммерческого соглашения будет установлена равной валюте управленческого учета.
  • Дата коммерческого предложения будет установлена равной текущей дате.
  • В качестве менеджера, оформляющего коммерческое предложение, будет указан текущий пользователь, работающий с программой.

Как изменить статус (отменить) несколько коммерческих предложений?
  • В списке коммерческих предложений отметьте те предложения, по которым необходимо изменить статус. Для отметки коммерческих предложений используйте стандартные возможности "1С:Предприятия".
  • Нажмите на кнопку "Установить статус".
  • В появившемся списке выберите один из статусов: Не согласовано, Действует, Отменено.
  • У отмеченных коммерческих предложений будет установлен выбранный статус.
