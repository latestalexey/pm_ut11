Как создать новое коммерческое предложение на основании уже существующего?
  • В форме коммерческого предложения нажмите на кнопку "Создать на основании".
  • В появившемся списке выберите "Коммерческое предложение клиенту".
  • Будет создано новое коммерческое предложение клиенту.

В новое коммерческое предложение будут скопированы все данные из ранее оформленного предложения. Ранее оформленное коммерческое предложение будет указано в качестве основания во вновь созданном коммерческом предложении.

Важно. Цепочку оформленных коммерческих предложений можно посмотреть в форме "Структура подчиненности". Форма структуры подчиненности вызывается с помощью одноименной кнопки на панели навигации формы "Коммерческое предложение клиента".

Работа с коммерческим предложением

Работа с коммерческим предложением состоит из следующих этапов

  • Заполнение данных в коммерческом предложении.
  • Регистрация коммерческого предложения.
  • Печать коммерческого предложения, отправка по электронной почте.

В том случае, если условия, указанные в коммерческом предложении отличаются от условий соглашения, то добавляется этап согласования коммерческого соглашения.

Заполнение основных данных в коммерческом предложении

Можно ли в коммерческом предложении ввести информацию о том сотруднике (контактном лице) с которым ведутся переговоры по коммерческому соглашению?

Информация о контактном лице партнера регистрируется на закладке "Дополнительно". Контактное лицо выбирается из списка тех контактных лиц, которые определены для партнера, выбранного в документе.

С этим контактным лицом ведутся непосредственные взаимодействия при работе с коммерческим предложением (электронные письма, встречи, телефонные звонки и т.д.).

Список взаимодействий можно посмотреть из карточки контактного лица. Для открытия карточки нажмите на кнопку "О" рядом с полем ввода контактного лица.

Как выбрать клиента, для которого надо сформировать коммерческое предложение?

Информация о клиенте заполняется в поле "Клиент". Выбор клиента осуществляется из списка партнеров. Для выбора доступны только те партнеры, для которых установлен признак "Клиент". В списке выбора можно установить дополнительные отборы. Например отобрать только тех клиентов, за работу с которыми ответственен менеджер, работающий с программой (флаг "Только мои"). Для выбора партнера необходимо установить курсор на нужного клиента в списке и нажать на кнопку "Выбрать".

После выбора клиента в документе может быть автоматически заполнена информация о соглашении. Алгоритм заполнения соглашения следующий.

  • Если для клиента (партнера) определено одно индивидуальное соглашение, то оно подставляется автоматически.
  • Если для клиента (партнера) определено несколько индивидуальных соглашений, то ничего не подставляется и пользователь должен выбрать соглашение вручную.
  • Если для партнера не определены индивидуальные соглашения, то определяются сегменты, к которым принадлежит партнер. Затем определяется список типовых соглашений, определенных для тех сегментов, в которые включен партнер.
  • Если для партнера доступно только одно типовое соглашение - то оно заполняется. Если для партнера доступно более одного типового соглашения - то информация о типовом соглашении не заполняется и пользователь должен выбрать типовое соглашение вручную.

Как выбрать соглашение, на основании которого будут регистрироваться условия продаж с клиентом вручную?

Если соглашение после выбора клиента (партнера) не заполнилось автоматически, то информацию о нем нужно заполнить вручную. Для быстрого выбора соглашения с клиентом в поле ввода "Соглашение" можно набрать первые буквы наименования или номер соглашения с клиентом. Также можно воспользоваться кнопкой выбора из списка соглашений.

В списке выбора показываются как индивидуальные, так и типовые соглашения, которые действуют для указанного в документе клиента. Для выбора доступны только действующие соглашения с неистекшим сроком действия. Для выбора соглашения необходимо установить курсор на нужное соглашение в списке выбора и нажать кнопку "Выбрать". После выбора соглашения в коммерческом предложении автоматически заполняются условия продаж, зарегистрированные в соглашении.

Важно. Изменять условия продажи в коммерческом предложении могут те пользователи, которые обладают правом отклонения от условий продаж. Если пользователь не обладает таким правом, то после измений условий продаж по сравнению с соглашением, необходимо запустить процедуру согласования условий продаж. Контроль отклонений от условий продаж производится при проведении коммерческого предложения в статусе "Действует".

В какой валюте оформляется коммерческое предложение?

Коммерческое предложение оформляется в той валюте, которая определена в соглашении, указанном в коммерческом предложении. В коммерческом предложении можно изменить валюту, однако это потребует дополнительного согласования условий коммерческого соглашения.

Как указать дату, до которой действует коммерческое предложение?

Дата, до которой действительно коммерческое предложение, указывается в поле "Срок действия". После выбора соглашения срок действия коммерческого предложения автоматически устанавливается равным дате окончания того соглашения, которое указано в документе. Пользователь может вручную изменить срок действия. По истечении срока действия коммерческое предложение перестает действовать. Для таких коммерческих предложений менеджер может установить статус "Отменено" и коммерческое предложение будет неактуальным.

Как указать условия оплаты в коммерческом соглашении?
  • Условия оплаты регистрируются в графике оплаты на закладке "Дополнительно".
  • График оплаты заполняется в соответствии с указанным в документе соглашением. Пользователь может выбрать другой график оплаты или заполнить новый график оплаты, только в том случае, если график оплаты не был определен в соглашении. Если график оплаты определен в соглашении, то изменение графика оплаты потребует дополнительного согласования.

Как указать в коммерческом предложении планируемый срок отгрузки товаров или выполнения предоставляемых услуг?

Планируемый срок отгрузки товаров или выполнения предоставляемых услуг в коммерческом предложении указывается в поле "Срок поставки" на странице "Дополнительно". Планируемый срок поставки указывается в днях. Планируемый срок поставки автоматически заполняется в соответствии с соглашением, указанным в коммерческом предложении. Если срок поставки в соглашении не определен, то пользователь его указывает в коммерческом предложении. На основании указанного срока поставки и даты, указанной в заказе клиента, будет рассчитана конкретная дата выполнения отгрузки товаров или выполнения услуг по заказу, оформленному в рамках коммерческого предложения.

Ввод информации о товарах и услугах

Список товаров и услуг, которые могут быть предложены клиенту, вводится в табличную часть документа "Коммерческое предложение". В табличную часть заполняется полный список товаров и услуг, которые могут быть предложены клиенту. В процессе согласования коммерческого предложения у тех позиций, которые согласованы с клиентом, устанавливается признак активности номенклатурной позиции.

Как добавить новую номенклатурную позицию в коммерческое предложение?

Для добавления новой позиции нажмите на кнопку "Добавить" в табличной части документа. Новая номенклатурная позиция выбирается из справочника "Номенклатура".

Для новой номенклатурной позиции автоматически устанавливается признак активности (флажок в поле "Активность"). Пользователь может снять этот флаг вручную по одной позиции или воспользоваться соответствующими обработками по групповому снятию (установке) активности для нескольких позиций номенклатуры.

При расчете итоговой суммы документа учитываются только активные позиции. В документе справочно показывается информация о том, на какую сумму были предложены товары клиенту, и на какую сумму он согласился оформить покупку.

Как установить признак активности для нескольких позиций коммерческого предложения?

  • В табличной части документа "Коммерческое предложение" отметьте те позиции, по которым необходимо установить признак активности, используя стандартные возможности "1С:Предприятие".
  • Нажмите на кнопку "Выбор строк" в командной панели табличной части документа.
  • Из появившегося списка выберите пункт "Установить отметки выбора выделенных строк".
  • У выделенных строк будет установлен флажок в поле "Активность".

Как снять активность у произвольно выбранных строк коммерческого предложения?
  • В табличной части документа "Коммерческое предложение" отметьте те позиции, по которым необходимо снять признак активности, используя стандартные возможности "1С:Предприятие".
  • Нажмите на кнопку "Выбор строк" в командной панели табличной части документа.
  • Из появившегося списка выберите пункт "Снять отметки выбора у выделенных строк".
  • У выделенных строк будет убран флажок в поле "Активность".

Указание цен для позиций в коммерческом предложении

Значения цен, по которым предлагаются товары и услуги клиенту, заполняются в соответствии с теми условиями, которые зарегистрированы в соглашении с клиентом, которое указано в коммерческом предложении. При добавлении новой позиции цены автоматически заполняются в соответствии с соглашением.

Важно. Если в коммерческом соглашении регистрируются цены, отличные от указанных в соглашении (или в соглашении цены не определены), то необходимо выполнить согласование условий коммерческого предложения.

Как заполняются цены на позиции в соответствии с теми условиями, которые зарегистрированы в соглашении?

При добавлении новой позиции цены на нее автоматически устанавливаются в соответствии с теми ценами, которые зарегистрированы в соглашении. При перевыборе соглашения цены автоматически пересчитываются в соответствии с теми ценами, которые определены в новом соглашении.

Предусмотрена также возможность восстановления значения цен в соответствии с соглашением. Для восстановления цен в соответствии с соглашением необходимо нажать на кнопку "Цены" и выбрать пункт меню "Заполнить цены по соглашению".

Как указываются цены в коммерческом предложении: с учетом или без учета НДС?

Цены в коммерческом предложении могут быть указаны с учетом или без учета НДС. Это определяется тем какая политика указания цен определена в соглашении с клиентом, для которого оформляется соглашение. Из соглашения копируется значение флага "Цены включают НДС". Изменение флага требует согласование коммерческого соглашения.

Если цены указываются с учетом НДС, то устанавливается флаг "Цена включает НДС". Если цены указываются без учета НДС, то флаг "Цены включают НДС" снят.

Примечание. По умолчанию флаг "Цена включает НДС" в форме коммерческого предложения не показывается.

Можно ли в одном коммерческом предложении предложить клиенту один и тот же товар по разным видам цен?

В рамках одного коммерческого предложения можно предложить клиенту товар по разным видам цен. Например, товары из ценовой группы "Обувь" по оптовым ценам, а товары из группы "Холодильники" - по дилерским ценам. Для этого в табличной части коммерческого предложения необходимо указать для конкретных товаров нужные виды цен. Для выбора доступны только те виды цен, в которых признак "Цена включает НДС" совпадает с одноименным признаком в документе.

Пример. При покупке 10 штук товаров в коммерческом предложении указывается оптовый вид цены, а при покупке 1000 штук товаров указывается дилерский вид цены. При получении коммерческого предложения покупатель может отметить тот вариант, который его больше устраивает, установив для выбранной позиции признак активности.

Данная возможность ограничивается той ценовой политикой, которая указана в соглашении, по которому оформляется коммерческое предложение. Если в соглашении определены конкретные виды цен для групп товаров, то именно они и будут применяться в коммерческом предложении. Изменение ценовой политики требует согласование условий, определенных в коммерческом предложении. Согласование коммерческого предложения также необходимо в том случае, если в коммерческом предложении фиксируются произвольные цены (в соглашении ценовая политика не задана).

Как заполнить цены в коммерческом предложении по разным видам цен?

Важно. Такую возможность следует использовать только в том случае, если виды цен и их значения не определены в соглашении, по которому оформляется коммерческое предложение клиенту.
  • В табличной части документа "Коммерческое предложение" отметьте те товары, для которых необходимо заполнить значения цен по определенному виду цен. Для отметки позиций используйте стандартные возможности 1С:Предприятия.
  • Нажмите на кнопку "Цены".
  • Выберите пункт меню "Заполнить цены в выделенных строках по виду цен".
  • В появившемся списке выберите тот вид цен, по которому необходимо заполнить значения цен для выделенных позиций. В списке показываются только те виды цен, у которых флаг "Цена включает НДС" совпадает с одноименным флагом соглашения, по которому оформляется коммерческое предложение. Цены на выделенные позиции будут заполнены в соответствии с теми значениями, которые зарегистрированы по данному виду цен.
  • Аналогичным образом заполните позиции в табличной части по другому виду цен.

Примечание. Если все позиции надо заполнить по одному виду цен, то нажмите на кнопку "Цены" и выберите пункт меню "Заполнить цены во всех строках по виду цен".

Можно ли указать в коммерческом предложении цены на товары, отличные от тех цен, которые зарегистрированы в прайс-листе предприятия?

Такая возможность существует только в том случае, если в соглашении, указанном в коммерческом предложении не определены виды цен или конкретные цены на товары.

Информация о ценах на конкретные позиции вводится в поле "Цена" в табличной части документа для каждой номенклатурной позиции. Цены вводятся в валюте, которая указана в коммерческом предложении. Цены вводятся с учетом или без учета НДС, в зависимости от того флага, который установлен в соглашении с клиентом, которое указано в коммерческом соглашении.

Работа со скидками

Как назначить ручную скидку (наценку) в коммерческом предложении?
  • В командной панели табличной части документа выберите команду "Скидки (наценки)".
  • В появившемся списке выберите команду "Назначить ручную скидку (наценку)".

Как отменить ручную скидку (наценку) в документе?

Для отмены ручной скидки (наценки) используйте команду "Скидки (наценки) - Отменить ручные скидки (наценки)".

Как установить автоматическую скидку по карте лояльности (дисконтной карте)?
  • В форме документа выберите команду "Считать карту лояльности" в комадной панели или нажмите комбинацию быстрых клавиш Ctrl+Alt+F7.
  • В появившейся форме укажите код карты. Код карты можно указать вручную или считать сканером штрихкода или ридером магнитных карт (в зависимости от вида карты).
  • Нажмите на кнопку "Готово".
  • Информация о считанной карте будет записана в поле "Карта лояльности" на странице "Дополнительно".

Скидки по карте лояльности будут рассчитаны совместно со всеми остальными автоматическими скидками в соответствии с условиями их совместного применения. Расчет автоматических скидок производится отдельной командой.

Важно. Если карта лояльности еще не зарегистрирована, то программа предложит ее зарегистрировать. При регистрации будет использоваться помощник регистрации карт лояльности.

Как посмотреть общий список тех автоматических скидок (наценок), которые могут быть назначены по документу?

Для просмотра общего списка автоматических скидок (наценок) , которые могут быть назначены по документу выберите команду "Скидки(наценки) - Назначить автоматические скидки (наценки)". В списке будут показаны все автоматические скидки (наценки), которые могут быть назначены по документу, в том числе и управляемые скидки (наценки), которые могут быть назначены менеджером вручную.

Можно ли рассчитать автоматические скидки без предварительного просмотра назначаемых автоматических скидок?

Для этих целей используется команда - "Скидки (наценки) -Рассчитать скидки (наценки)".

Важно. При расчете автоматических скидок анализируется условия замены карты лояльности. Если эти условия выполнены то программа предложит заменить карту лояьности на другую карту (например, серебряную карту заменить на золотую). Замену карты можно произвести непосредственно при оформлении документа. Новая карта регистрируется с помощью помощника регистрации карт лояльности.

Как открыть отчет по всем автоматически примененным скидкам (наценкам) в документе?
  • В панели навигации формы выберите команду "Примененные скидки".
  • Будет показан общий список всех автоматических скидок, которые были применены в документе.

Как ввести дополнительные комментарии к коммерческому предложению?

Менеджер при оформлении коммерческого предложения может ввести любую произвольную информацию в качестве дополнительного комментария к коммерческому предложению. Эта информация вводится в тестовом поле "Информация для печати" на странице "Дополнительно". Данная информация будет использоваться при печати коммерческого предложения. На странице "Комментарий" можно ввести дополнительную внутреннюю информацию, которая не будет доступна клиенту ( не используется в печатной форме коммерческого предложения).

Регистрация коммерческого предложения

Для того, чтобы зарегистрировать коммерческое предложение, документ необходимо записать и провести. Перед записью коммерческого предложения осуществляется проверка заполненности обязательных реквизитов. Также проверяется соответствие параметров, указанных в коммерческом предложении, тем параметрам, которые определены в соглашении. Например, если партнер в коммерческом предложении не принадлежит к сегменту партнеров, указанном в типовом соглашении, то коммерческое предложение не регистрируется, программа предложит провести согласование коммерческого предложения. Также проверяется вхождение цен коммерческого предложения в диапазон возможных цен продажи (максимальная и минимальная цена продажи).

Как ввести в действие коммерческое предложение?

Для ввода в действие коммерческого предложения необходимо установить статус "Действует" и нажать на кнопку "Провести". Соглашение начинает действовать с даты, которая указана в поле "Срок действия".

Как откорректировать действующее коммерческое предложение?

Корректировка действующего коммерческого предложения допустима только в том случае, если в рамках этого коммерческого предложения не были оформлены документы. Корректировка коммерческого предложения доступна только ответственным сотрудникам (например, руководителям подразделений).

Для корректировки коммерческого предложения выполните следующие действия.

  • Нажмите на кнопку "Изменить" в панели действий формы коммерческого предложения.
  • Измените нужные параметры.
  • Нажмите на кнопку "Провести и закрыть".

Как отменить коммерческое предложение?

Для этого в форме типового соглашения необходимо установить статус "Отменено" и нажать на кнопку "Провести".

Важно. Предусмотрена возможность отмены (установки определенного статуса) нескольких произвольно выбранных коммерческих предложений.

Сервис

Как посмотреть этапы согласования коммерческого предложения с клиентом?

Просмотр этапов согласования состоит из просмотра зарегистрированных версий коммерческого предложения. В процессе согласования новые версии коммерческого предложения фиксируются путем ввода нового коммерческого предложения на основании предыдущего.

Для просмотра списка версий в форме коммерческого предложения в панели навигации формы выберите "Структура подчиненности". Будет показан общий список коммерческих предложений. Из структуры подчиненности можно открыть любое коммерческое предложение.

Как распечатать коммерческое предложение?

Коммерческое предложение может быть распечатано из формы или из списка документов (кнопка "Печать"). В печатную форму выводится вся необходимая информация, в том числе информация о дате и сроке действия коммерческого предложения, перечень номенклатурных позиций, информация для печати. Печатная форма может быть сформирована в форматах mxl (1C:Предприятие), doc (Microsoft Word) или odt (OpenOffice.org Writer).

Как открыть отчет для анализа цен конкурентов?

В панели навигации формы коммерческого предложения выберите пункт "Анализ цен ". Будет сформирован отчет "Анализ цен". В отчете будут автоматически отображены те позиции, которые указаны в коммерческом предложении. В отчете будет произведено сравнение цен, указанных в коммерческом предложении с зарегистрированными ценами конкурентов и поставщиков.

Как запустить процесс согласования коммерческого предложения?
  • В форме коммерческого предложения нажмите на кнопку "Создать на основании".
  • В появившемся списке выберите пункт "Согласование коммерческого предложения клиенту".

В появившейся форме бизнес-процесса "Согласование продажи" укажите срок, до которого должно быть согласовано коммерческое предложение и введите текстовое описание обоснования отклонений от типовых условий продаж (важный клиент, большие объемы поставок и т.д.). Для запуска бизнес-процесса нажмите на кнопку "Старт". Программа определит какие условия продажи (ценовые, финансовые или логистические) необходимо согласовывать и направит задачи тем исполнителям, которые отвечают за эти процессы.

Как посмотреть в какой стадии находится процесс согласования условий коммерческого соглашения и результаты согласования?
  • Для просмотра процесса согласования используйте команду Согласование в панели навигации формы коммерческого предложения. В форме Согласование будет показан список направленных исполнителям задач и состояние их выполнения. Предусмотрена возможность просмотра просроченных задач, а также отбор задач по их статусам ( в процессе согласования, согласовано, несогласовано).
  • Для просмотра результатов согласования используйте отчет Результаты согласования. Отчет открывается с помощью одноименной команды в панели навигации формы коммерческого предложения.

Можно ли оформить документ в электронном виде?

Если с контрагентом существует договоренность об электронном взаимодействии, то документ учетной системы может быть оформлен и отправлен получателю в электронном виде.

Как создать заказ на основании коммерческого предложения?

При создании заказа клиента на основании коммерческого предложения в заказе заполняется следующая информация.

  • Партнер, с которым было согласовано коммерческое соглашение.
  • Соглашение, по которому было оформлено коммерческое предложение.
  • Валюта, в которой было заключено коммерческое предложение.
  • Сделка, которая указана в коммерческом предложении.

Табличная часть заказа клиента заполняется теми номенклатурными позициями, которые отмечены в качестве активных позиций в коммерческом предложении. Условия продаж заполняются на основании тех условий продаж, которые были зарегистрированы в коммерческом предложении. Если какие-либо условия продаж не были определены в коммерческом предложении, то они дополнительно заполняются на основании тех условий продаж, которые зарегистрированы в соглашении. На основании данных об условиях продаж и даты заказа, рассчитываются даты и суммы оплат по этапам оплаты по заказу, устанавливаются цены или виды цен для номенклатурных позиций.

При создании заказа также осуществляется попытка поиска контрагента (юридического или физического лица) партнера. Если для партнера определен только один контрагент, то он подставляется в заказ клиента.

Как можно получить данные из терминала сбора данных в коммерческом предложении?

Для того чтобы загрузить данные из терминала сбора данных в табличную часть "Товары" коммерческого предложения необходимо воспользоваться командой табличной части "Все действия" - "Загрузить товары из ТСД". Если терминал сбора данных подключен и настроены параметры его подключения, то табличная часть заполниться данными из терминала. В противном случае программа выдаст предупреждение. Если из теминала будет получен штрих код не зарегистрированый в информационной базе, то программа предложит зарегистрировать его и сопоставить с имеющимся в информационной базе товаром.

Как можно добавить данные в табличную часть "Товары" с помощью сканера?

Если сканер подключен и настроены параметры его подключения, то после считывания штрих кода в табличную часть будет добавлен товар соответствующий штрих коду. Если товар уже присутствует в табличной части, то будет увеличено его количество. Если в программе не зарегистрирован считываемый штрих код, то программа предложит зарегистрировать его и сопоставить имеющемуся в информационной базе товару.

Как найти в табличной части товар используя сканер штрих кода?

Для того что бы выполнить поиск товара в табличной части, необходимо воспользоваться командой табличной части "Все действия" - "Найти товары по штрих коду". Далее в открывшейся форме считать штрих код товара. Если товар со считанным штрих кодом присутствует в табличной части, то он будет выделен (на него будет установлен курсор). Если в программе не зарегистрирован считываемый штрих код, то программа предложит зарегистрировать его и сопоставить имеющемуся в информационной базе товару.
