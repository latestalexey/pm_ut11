Документ "Отчет комитенту о продажах"

Документ "Отчет комитенту о продажах" регистрирует факт продажи товаров комитента. Документ может быть оформлен только по тому соглашению, в котором указан вид операции "Прием на комиссию".

Данные о продажах товаров фиксируются за тот период, который указан в отчете комитенту. Количество проданных товаров регистрируется в графе "Продано". Табличная часть документа может быть заполнена теми товарами, которые были проданы за период, указанный в отчете комитенту (Заполнить - Заполнить по результатам продажи). Если предполагается отчитаться перед комитентом не за все проданные товары, то следует воспользоваться вариантом заполнения Заполнить - Подобрать по результатам продажи.

Важно. При заполнении отчета комитенту учитывается информация о тех комиссионных товарах, которые были возвращены покупателем после оформления отчета комитенту. Информация об этих товарах заполняется в отчете комитенту с минусом.

В документе фиксируется информация о ценах, по которым товар принимался на реализацию ("Цены комитента") и ценах, по которым товар был продан комиссионером ("Цена продажи").

  • Цена комитента заполняется в соответствии с теми ценами, которые зарегистрированы в соглашении с комитентом. При необходимости цена может быть изменена пользователем произвольным образом. Для воостановления цен в соответствии с соглашением используется кнопка "Заполнить-Заполнить цены по соглашению".
  • Цена продажи заполняется той ценой продажи, по которой был продан товар комитента с учетом всех предоставленных скидок и наценок в момент продажи комиссионного товара. Изменять цену продажи не рекомендуется.

На основании данных, введенных в документ "Отчет комитенту о продажах", рассчитывается комиссионное вознаграждение. Способ расчета комиссионного вознаграждения может быть указан в соглашении с комитентом или непосредственно в документе "Отчет комитенту о продажах". Информация о комиссионном вознаграждении указывается на странице "Комиссионное вознаграждение". В документе указывается сумма комиссионного вознаграждения, сумма НДС комиссионного вознаграждения и дата выплаты комиссионного вознаграждения ("Дата платежа"). На сумму комиссионного вознаграждения может быть оформлен счет-фактура ("Ввести счет-фактуру").

Важно. Сумма комиссионного вознаграждения, которое начислено комитенту, может быть учтена при расчетах с комитентом за проданные товары. В этом случае на странице "Комиссионное вознаграждение" устанавливается флаг "Учитывать комиссионное вознаграждение".
