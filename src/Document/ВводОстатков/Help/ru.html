Документ "Ввод остатков"

Документ "Ввод остатков" предназначен для ввода начальных остатков товаров, остатков денежных средств и взаиморасчетов с партнерами торгового предприятия (поставщиками, клиентами). Для каждой организации и каждого типа операции необходимо оформить отдельный документ ввода остатков.

Перед началом ввода начальных остатков на странице "Настройка" необходимо выбрать тип операции, по которой необходимо ввести начальные остатки. В зависимости от выбранного типа операции будет сформирована страница с нужными показателями ввода начальных остатков.

Ввод начальных остатков денежных средств

В программе отдельно фиксируются остатки наличных и безналичных денежных средств. Для каждой организации оформляются два документа : по вводу остатков наличных денежных средств в кассах организации и по вводу остатков наличных денежных средств на расчетных счетах организации.

Ввод начальных остатков товаров

В программе остатки собственных товаров, остатки товаров, принятых на комиссию и остатки товаров у комиссионеров регистрируются отдельными документами с соответствующими типами операции.

  • Остатки товаров регистрируются по каждому складу и каждой организации.
  • Если склад разделен на помещения, то остатки товаров регистрируются по каждому помещению.
  • При вводе остатков, принятых на комиссию, по каждому комитенту оформляется отдельный документ ввода остатков.
  • Остатки товаров у комиссионеров регистрируются отдельными документами по каждому комиссионеру.
  • Для импортных товаров остатки должны вводиться с учетом номеров ГТД.

Остатки на адресном складе (складе, в котором ячейки используются для хранения остатков товаров) вводятся только после инвентаризации и пересчета товаров в каждой ячейке. В документе ввода остатков должен быть указан приказ на проведение инвентаризации. В рамках приказа на проведение инвентаризации оформляются документы Пересчет товаров, в которых указывается количество товаров в каждой конкретной ячейке.

Ввод остатков взаиморасчетов

В программе отдельно вводятся остатки взаиморасчетов по поставщикам и клиентам. При вводе остатков взаиморасчетов можно фиксировать, как остатки полученных (перечисленных) авансов, так и остатки задолженности клиентов перед организацией торгового предприятия или остатки задолженности организации перед поставщиками.

При регистрации задолженности клиентов (поставщиков) можно указать номер и дату документа, по которому имеются остатки задолженности:заказ клиента (заказ поставщику) или реализация товаров и услуг(поступление товаров и услуг). На основании данных о номере и дате будет создан пустой образ документа, который будет использоваться в качестве ссылки и уточняющей информации в документе ввода остатков.

Важно. При регистрации авансов по заказам клиентов рекомендуется предварительно зарегистрировать заказ клиента с указанием списка товаров, которые необходимо отгрузить клиенту и установить для заказа статус Согласовано. После этого необходимо в документе ввода остатков зарегистрировать сумму аванса и в качестве ссылки выбрать ранее зарегистрированный заказ клиента. Аналогичным образом можно зарегистрировать авансы, перечисленные по заказам поставщиков. Во всех остальных случаях достаточно указать только номер и дату документа, по которому возникла задолженность, созданные автоматически образы документов изменять не рекомендуется.

При вводе остатков по взаиморасчетам с комиссионерами и комитентами задолженность по сумме проданных товаров и задолженность по сумме комиссионного вознаграждения оформляется отдельно. В качестве ссылки при оформлении задолженности можно указать номер и дату того документа "Отчет комиссионера" или "Отчет комитенту", по которым имеется задолженность.

При регистрации задолженности подотчетных лиц отдельно регистрируется задолженность по выданным подотчетникам наличным денежным средствам и по перечисленным ему на расчетный счет безналичным средствам. При этом в качестве дополнительной информации можно зарегистрировать номер и дату документа денежных средств, которым были выданы денежные средства подотчетнику, Если подотчетник израсходовал больше денежных средств, чем ему было выдано, то можно зафиксировать перерасход денежных средств и в качестве дополнительной информации зарегистрировать номер и дату авансового отчета, по которому надо зарегистрировать перерасход.

Ввод остатков задолженности банка по эквайринговым операциям

В программе можно зарегистрировать остаток задолженности банка по перечислению денежных средств по эквайринговым операциям (оплата или возврат денежных средств по платежным картам). Для ввода такой задолженности в документе следует установить тип операции Сторно отчетов по эквайрингу. В список надо ввести информацию о тех платежных картах, по которым банк еще не прислал отчет по операциям эквайринга.
