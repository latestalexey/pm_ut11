Документ "Корректировка обособленного учета запасов"

Документ предназначен для корректировки данных при ведении обособленного учета по поставщикам, сделкам, менеджерам, подразделениям, группам финансового учета. Ведение обособленного учета позволяет детализировать данные о поступлениях и расходах в рамках объектов обособленного учета (поставщиков, сделок и т.д.). Обособленный учет ведется только в том случае, если в настройках параметров учета (страница Финансы) установлен флажок "Учитывать себестоимость товаров по видам запасов" и выбран хотя бы один из объектов для ведения обособленного учета.
