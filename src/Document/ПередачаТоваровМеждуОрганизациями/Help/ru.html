Документ "Передача товаров между организациями"

Документ "Передача товаров между организациями" предназначен для оформления передач товаров между организациями.

В документе указывается информация о тех организациях, между которыми необходимо оформить передачу товаров и склад, с которого надо списать товар одной организации и принять товар другой организации. После указания организации на закладке "Дополнительно" заполняется информация о применяемом способе налогообложения ("Налогообложение"). Способ налогообложения заполняется в соответствии с той системой налогообложения , которая применяется для организации-отправителя.

Предусмотрены следующие варианты оформления передачи товаров между организациями, которые определяются видом операции ("Операция"), установленном в документе.

  • Передача до реализации. Товар одной организации будет передан другой организации на условиях купли-продажи. Необходимость в оформлении такой операции может возникнуть в случае перерегистрации, когда остатки товаров одной организации следует перенести на другую организацию.
  • Передача на комиссию до реализации. Товар одной организации будет передан другой организации на условиях комиссии. Необходимость в оформлении такой операции может возникнуть в случае перерегистрации, когда в одной организации имеются остатки комиссионных товаров, которые надо передать на комиссию другой организации.
  • Передача после реализации. Организация продала товар другой организации, необходимо оформить операцию перепродажи товаров между организациями по факту продажи товаров другой организации.
  • Передача на комиссию после реализации. Организация продала комиссионный товар другой организации, необходимо оформить операцию передачи на комиссию товаров между организациями по факту продажи товаров другой организации.

Список товаров, которые передаются между организациями, может быть заполнен вручную или автоматически с помощью кнопки "Заполнить". Предусмотрены различные варианты заполнения табличной части документа в зависимости от вида оформляемой операции передачи товаров между организациями. При автоматическом заполнении используются данные об остатках, рассчитанные на дату документа.

  • Передача до реализации. В табличную часть можно подобрать товары, которые имеются на указанном в документе складе у организации-отправителя ("Подобрать по остаткам товаров у отправителя"). Вариант заполнения "Подобрать по отрицательным остаткам у получателя" используется в том случае, если на предприятии не контролируются остатки товаров организаций и товары можно продать от имени любой организации (не настроена схема интеркампани).
  • Передача на комиссию до реализации. В табличную часть можно подобрать товары, которые имеются на указанном в документе складе у организации-отправителя ("Подобрать по остаткам товаров у отправителя").
  • Передача после реализации. Заполнение табличной части возможно только в том случае, если на предприятии настроена схема интеркампани между организациями ("Продажа") и отражены продажи товаров другой организации. Табличная часть заполняется теми товарами организации- отправителя, которые были проданы от имени организации -получателя ("Заполнить по остаткам к передаче").
  • Передача на комиссию после реализации. Заполнение табличной части возможно только в том случае, если на предприятии настроена схема интеркампани между организациями ("Передача на комиссию") и отражены продажи товаров другой организации. Табличная часть заполняется теми товарами организации- отправителя, которые были проданы от имени организации -получателя ("Заполнить по остаткам к передаче").

Важно. При настроенной схеме интеркампани между организациями документ "Передача товаров между организациями" может быть создан автоматически на основании данных о продажах товаров другой организации. Список организаций, по которым были оформлены продажи других организаций отображается на странице "К оформлению".

В документе фиксируется информация о ценах, по которым товар будет передан между организациями. Цены заполняются в соответствии с зарегистрированными ценами по тому виду цен, который указан в документе (страница "Дополнительно"). Используются только те виды цен, у которых установлен флаг "Использовать при передаче между организациями". Если на предприятии определен только один такой вид цен, то он будет автоматически заполнен в документе. Если для расчета между организациями определено несколько видов цен, то необходимо выбрать нужный вид цен в документе. Для автоматического заполнения цен на товары в соответствии с зарегистрированными ценами указанного вида используется кнопка "Заполнить - Заполнить цены по виду цен". Если товары между организациями заполняются по разным видам цен, то для заполнения используется вариант заполнения "Заполнить цены выделенных строк по виду цен".

Важно. В списке видов цен, показываются только те цены, у которых флаг учета НДС совпадает с флагом, указанном в документе на закладке "Дополнительно" - "Цены включают НДС".

По документу может быть оформлен и распечатан счет-фактура о продаже товаров организации-отправителя. Счета-фактура оформляется при нажатии на кнопку "Ввести счет-фактуру". Предусмотрена печать документов: документа отгрузки товаров от имени организации -отправителя ( Торг-12) и документа поставки товаров в организацию-получатель (М-15).

Важно. Счет-фактура оформляется только в случае оформления операции перепродажи товаров между организациями. При оформлении операции передачи на комиссию счет-фактура оформляется после фиксации факта продажи товаров по документу "Отчет по комиссии между организациями". Долг между организациями и необходимость оформления расчетов между организациями также возникает только при оформлении операции перепродажи товаров.

Расчет между организациями за переданные товары может оформляться как наличными, так и перечислением денежных средств с расчетного счета одной организации на расчетный счет другой организации. Для оформления расчетов используется документ "Распоряжение на оплату" , который может быть введен на основании документа "Передача товаров между организациями".

  • Если расчет производится наличными, то на закладке "Дополнительно" указывается вид оплаты "Наличная" и дата платежа. Для оформления оплаты оформляется документ "Распоряжение на оплату" с установленным видом операции "Выдача ДС в другую организацию".
  • Если расчет производится путем перечисления денежных средств на расчетный счет другой организации, то на закладке "Дополнительно" указывается вид оплаты "Безналичная" и дата платежа. Для оформления оплаты оформляется документ "Распоряжение на оплату" с установленным видом операции "Перечисление ДС в другую организацию".
