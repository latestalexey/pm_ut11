Документ "Отчет банка по операциям эквайринга"

Документ предназначен для регистрации детальных отчетов по операциям эквайринга, полученных из обслуживающих банков. Отчеты представляют собой расшифровку платежей, поступивших на расчетный счет или списанных с расчетного счета, при оформлении платежей с использованием платежных карт.

Для оформления операций по оплате платежными картами используется документ "Эквайринговая операция".

Важно. Предусмотрена возможность автоматической загрузки отчетов банка по эквайринговым операциям из внешнего файла. Для этих целей используется обработка "Загрузка отчета банка по эквайрингу".
