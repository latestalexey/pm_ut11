Какие данные отображаются в списке отчетов банков по эквайрингу?

В списке документов отчетов банков по эквайрингу отображаются следующие данные:

  • Номер, Дата - номер и дата отчета по эквайринговым операциям, полученным из банка.
  • Эквайрер - название банка, от которого получен отчет по эквайринговым операциям.
  • Сумма документа - общая сумма операций по платежным картам.
  • Проведено банком- признак проведения банком операции по зачислению денежных средств по платежным картам на расчетный счет организации. Устанавливается при включении отчета по эквайрингу в выписку банка.

Как отобрать отчеты о платежах по платежным картам по конкретному договору эквайринга?
  • В списке "Отчеты банков по эквайрингу" в поле быстрого отбора укажите договор эквайринга. Для быстрого выбора можно указать название договора.
  • В списке будут показаны, только те отчеты, которые были оформлены по выбранному договору эквайринга.

Как отобрать отчеты, которые получены из конкретного банка ( по всем заключенным с ним договорам эквайринга)?
  • Установите курсор в графу "Эквайрер".
  • Нажмите на кнопку "Найти".
  • Будет открыто диалоговое окно поиска.
  • В поле "Где искать" будет установлено "Эквайрер".
  • В поле "Что искать" выберите банк, по которому Вы хотите посмотреть отчеты.
  • Будут показаны отчеты конкретного банка.

Можно провести быстрый поиск по названию банка. Для этого установите курсор в поле "Эквайрер" и начните набирать название банка. Диалоговое окно поиска при этом будет открыто автоматически.

Аналогичным образом можно произвести отбор по любому из полей, указанных в списке. Например, можно найти отчет по эквайринговым операциям по его номеру.

Как зарегистрировать новый отчет банка по эквайринговым операциям?
  • В списке "Отчеты банка по эквайрингу" укажите договор эквайринга, по которому регистрируется отчет.
  • Нажмите на кнопку "Создать".
  • Будет создан новый документ "Отчет банка по эквайрингу".
  • В новом документе будет заполнена информация о договоре эквайринга в соответствии с теми данными, которые были указаны в списке. По указанному договору эквайринга автоматически заполняется информация валюте документа. Валюта документа определяется валютой банковского счета, указанного в договоре эквайринга.
