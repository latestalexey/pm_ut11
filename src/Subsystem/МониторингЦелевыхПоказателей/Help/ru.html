Мониторинг целевых показателей

Контроль и анализ деятельности предприятия

Программа позволяет своевременно выявлять "проблемные" участки при оперативном управлении и помогает руководителю принимать правильные управленческие решения. С помощью подсистемы "Мониторинг целевых показателей" руководитель может получить информацию по ключевым, интересующим его процессам. Список ключевых показателей настраивается в соответствии с особенностями ведения бизнеса в конкретном торговом предприятии.

Подсистема позволяет не только контролировать текущее состояние показателей. В соответствии с установленными пороговыми значениями показателя программа автоматически информирует руководителя в ситуациях, когда состояние показателя еще допустимое, но приближается к критичному значению. Также автоматически отмечаются те ситуации, когда показатель резко вышел за границы допустимого. Информация о текущем состоянии показателей может быть выведена на рабочий стол.

Данные для анализа доступны пользователям в разделе "Мониторинг целевых показателей". Доступ к этому разделу имеют те пользователи, в профиль которых включена роль "Подсистема мониторинг целевых показателей".

Настройка системы мониторинга

Настройка подсистемы "Мониторинг целевых показателей"

Настройка подсистемы "Мониторинг целевых показателей" позволяет определить структуру целей предприятия и соответствующих им целевых показателей, в соответствии с которыми будет контролироваться деятельность торгового предприятия с точки зрения степени достижения поставленных целей. Для каждого целевого показателя могут быть заданы различные варианты анализа.

Структура целей представляет из себя дерево, на верхнем уровне которого находится те основополагающие цели ("мечты"), которые хотелось бы достичь руководителю предприятия. Каждая из этих основополагающих целей состоит из множества подцелей, успешное достижение которых и обеспечивает достижение основной цели.

Каждой цели соответствует один целевой показатель. Для каждого целевого показателя в соответствии с условиями достижения цели настраиваются соответствующие варианты анализа.

Пример. Для достижения основной цели "Рост прибыли" необходимо успешное выполнение двух подцелей: "Рост валовой прибыли" и "Снижение косвенных расходов". Для того чтобы обеспечить рост валовой прибыли необходимо обеспечить рост выручки и снижение себестоимости товаров. Таким образом достижением цели "Рост прибыли" будет является максимизация значений целевого показателя "Прибыль", а для подцелей максимизация значений показателя "Выручка" и минимизация значений показателя "Себестоимость товаров" соответственно.

Для анализа целевых показателей используются различные варианты анализа, построенные с использованием схем компоновки данных. Пользователь может использовать готовые шаблоны схем компоновки данных или разрабатывать свои схемы компоновки данных. Для дополнительной детализации данных могут быть использованы все варианты отчетов конфигурации – как поставляемые, так и сохраненные пользователями.

Программа поставляется с заполненной структурой целевых показателей и списком вариантов для их анализа. Пользователь может уточнить значения параметров анализа, добавить новые цели и целевые показатели, разработать свою структуру целей и целевых показателей. Настройку структуры целей, целевых показателей и вариантов их анализа производит администратор системы в соответствии с ранее подготовленной руководством предприятия моделью управления предприятием.

Настроенная структура целевых показателей и вариантов их анализа используется для показа информации о деятельности предприятия в мониторе целевых показателей.

Настройка структуры целей и показателей

Как открыть структуру целей?
  • В списке разделов выберите раздел "Мониторинг целевых показателей".
  • В панели навигации выберите пункт "Варианты анализа целевых показателей".
  • Структура целей отображается в левой части списка.

Важно. Для редактирования информации о целях можно использовать отдельный справочник "Структура целей". По умолчанию он не включен в панель навигации данного раздела.

Настройка вариантов анализа целевых показателей

Как открыть список вариантов анализа целевых показателей?
  • В списке разделов выберите раздел "Мониторинг целевых показателей".
  • В панели навигации выберите пункт "Варианты анализа целевых показателей".

Программа поставляется с заполненным списком вариантов анализа каждого целевого показателя. Пользователь может изменить вариант анализа или добавить новый вариант.

Настройка доступности вариантов анализа для пользователей/подразделений

Как открыть форму для настройки доступности вариантов анализа показателей для конкретных пользователей/подразделений?

  • В списке разделов выберите раздел "Мониторинг целевых показателей".
  • В группе Настройки выберите Настройка доступности вариантов анализа.

Настройка периодичности автоматического расчета источников данных вариантов анализа.

Программа позволяет настроить автоматический расчет данных для представления результатов расчета в мониторе целевых показателей. Для этих целей используется соответствующее регламентное задание "Расчет источников данных вариантов анализа". Можно настроить расписание в соответствии с которым будет выполняться автоматический расчет.

Пользовательская настройка монитора целевых показателей

При анализе данных с помощью монитора целевых показателей пользователь может произвести и сохранить персональные настройки: отобрать только нужные показатели и варианты их анализа из списка доступных, изменить форму представления или группировку показателей.

Контроль целевых показателей предприятия

Для контроля целевых показателей предприятия предназначен "Монитор целевых показателей". Используя монитор целевых показателей можно как проанализировать текущие значения показателей, так и проанализировать процесс достижения поставленной цели.

При контроле целевых показателей можно использовать как краткую форму представления показателей, так и более расширенную с отображением соответствующих таблиц и диаграмм. Предусмотрена возможность группировки показателей по степени их важности или по категориям целей.

Контроль с использованием монитора целевых показателей

Как открыть монитор целевых показателей?
  • В списке разделов выберите раздел "Мониторинг целевых показателей".
  • В панели навигации выберите пункт "Монитор целевых показателей".

В мониторе будут автоматически показаны те варианты анализа, которые доступны данному пользователю в соответствии с установленными настройками.

Важно. Монитор целевых показателей доступен только тем пользователям, для которых установлена роль (право) "Подсистема мониторинг целевых показателей".

Как производится оперативный контроль ключевых показателей предприятия на предмет отклонения от установленных целевых значений/диапазонов?

Данный вариант контроля используется в том случае, если заданы конкретные значения (диапазоны) целевых показателей и допустимые позитивные и негативные отклонения. При этом контролируются следующие данные.

  • Статус целевого показателя.
  • Целевой тренд.

Данные могут быть представлены в различных вариантах отображения, в зависимости от настроек: кратко, диаграмма, таблица.

Как производиться контроль целевых показателей при неустановленных граничных значениях (диапазонах)?

Если для целевых показателей граничные значения не установлены, то в мониторе можно проконтролировать только текущие значения показателей и тенденции их изменения (рост, падение, без изменений) в соответствии с установленным целевым трендом.

При этом статусы и характеры трендов (негативный или позитивный) системой не рассчитываются. Состояние при неустановленных целях выводится в виде пиктограммы вопроса.

Для экспресс-анализа используется краткая форма монитора целевых показателей, которая может быть размещена на рабочем столе.
