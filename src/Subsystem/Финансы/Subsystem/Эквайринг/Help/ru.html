Эквайринг

Программа обеспечивает возможность учета поступлений и возвратов денежных средств, поступивших с помощью банковских пластиковых карт различных платежных систем.

Работа с подсистемой "Эквайринг" состоит из следующих этапов.

  • Настройка эквайринга. Задачами настройки и обеспечением работоспособности подсистемы эквайринга занимается администратор системы.
  • Регистрация оплаты и возврата оплаты по платежным картам (по операциям эквайринга). Эти операции выполняет кассир.
  • Взаимодействия с банком по операциям эквайринга. Эти операции выполняет казначей.

Работа с подсистемой эквайринга возможна только в том случае, если в разделе "Администрирование/Финансы" установлен флаг " Оплаты платежными картами".

Настройка эквайринга

Перед оформлением эквайринговых платежей необходимо зарегистрировать в программе договора эквайринга, в которых будут определены условия обслуживания банком операций с платежными картами и указать список тех эквайринговых терминалов, с помощью которых будут оформляться эквайринговые операции.

Работа со списком договоров эквайринга

Как открыть список договоров эквайринга?
  • В панели разделов выберите раздел "Финансы".
  • В панели навигации раздела выберите пункт "Договоры эквайринга".

Регистрация оплаты и возврата оплаты по эквайрингу

В программе предусмотрена возможность отражения операций оплаты клиентом за товары, работы, услуги с помощью платежных (пластиковых) карт международных платежных систем, и отражение операций возврата денежных средств на платежную карту.

Работа со списком эквайринговых операций

Как открыть список эквайринговых операций (оплата с использованием платежных карт)?
  • В списке разделов выберите раздел "Финансы".
  • В панели навигации раздела выберите пункт "Эквайринговые операции".

Регистрация отчетов банков по эквайрингу

В программе предусмотрена возможность регистрации детальных отчетов по операциям эквайринга, полученных из обслуживающих банков. Отчеты представляют собой расшифровку платежей, поступивших на расчетный счет или списанных с расчетного счета, при оформлении платежей с использованием платежных карт.

Работа со списком отчетов банка по эквайрингу

Как посмотреть отчеты банка по эквайринговым операциям?
  • В списке разделов выберите раздел "Финансы".
  • В панели навигации раздела выберите пункт "Отчеты банков по эквайрингу".

Регистрация нового отчета банка по эквайрингу

Можно ли загрузить данные об отчетах по эквайрингу автоматически?
  • Выберите раздел "Финансы".
  • В группе "Сервис" выберите "Загрузка отчета банка по эквайрингу".

Предусмотрен следующий сценарий работы.

  • Выберите XML файл с данными, содержащий отчет банка по операциям эквайринга.
  • Выберите статью расходов, на которую будет списана сумма эквайринговой комиссии, и при необходимости аналитику расходов.
  • В случае, когда в организации используются подразделения для учета доходов и расходов, выберите подразделение.
  • Установите или снимите флажок Проводить документы при загрузке.
  • Загрузите отчет банка по эквайрингу.

Контроль расчетов с банком по эквайрингу

Программа обеспечивает контроль расчетов с банком по операциям эквайринга. Эта возможность предназначена для сверки зарегистрированных оплат и возвратов денежных средств по платежным картам и суммы денежных средств, зарегистрированных банком по операциям эквайринга.

Как открыть отчет для сверки оформленных в программе платежей по платежным картам с информацией, зарегистрированной в банке по данным платежам?
  • В панели разделов выберите раздел "Финансы".
  • В панели действий раздела выберите "Отчет банков по эквайрингу".
