Управление торговыми представителями

Продажи через торговых представителей

В программе предусмотрена возможность оформления продаж клиентам через торговых представителей. Работа с торговыми представителями возможна только в том случае если включена функциональная опция "Ведется работа через торговых представителей". При работе с торговыми представителями возможно использовать следующие варианты работы. Используемый вариант работы настраивается в разделе "Администрирование" на странице "Торговые представители".

  • Планирование работы торговых представителей производится в центральном офисе. Торговым представителям выдаются задания и регистрируется факт выполнения задания. На основании заданий могут быть автоматически создан заказ клиента. Для обеспечения такой схемы работы в настройках параметров учета должен быть установлен вариант использования заданий - Используются для управления торговыми представителями.
  • Торговые представители сами планируют свою работу с клиентами и могут самостоятельно формировать задания и оформлять заказы клиентов. Для обеспечения такой схемы работы в настройках параметров учета должен быть установлен вариант использования заданий - Используются торговыми представителями для планирования.
  • Торговые представители работают с клиентами без предварительных заданий. Программа предоставляет торговым представителям информацию о клиентах и условиях работы с ними (соглашения с клиентом). Торговый представитель оформляет заказы клиентов, ищет новых клиентов и передает эту информацию в центральный офис. Для обеспечения такой схемы работы в настройках параметров учета должен быть установлен вариант использования заданий - Не используются.

При работе с торговыми представителями могут использоваться различные мобильные устройства.

Регистрация условий обслуживания клиентов

Выполняется регистрация условий обслуживания партнеров торговым представителем. Могут быть определены следующие условия обслуживания клиентов.

  • График посещения клиентов торговыми представителями.
  • Соглашения с клиентом, в рамках которого производится обслуживание торговым представителем.
  • Закрепление торгового представителя.
  • Закрепление супервайзера (куратора, который отвечает за работу торговых представителей с клиентом).

Формирование заданий торговым представителям

В конфигурации предусмотрено планирование работы торговых представителей. Для планирования работы торговых представителей используются документы "Задание торговому представителю". Сформированные задания передаются торговым представителем для работы с клиентом. Если торговый представитель в своей работе использует мобильные устройства, то информация о выданных ему заданиях автоматически загружается в мобильное устройство. Если торговый представитель не использует мобильное устройство, то ему передаются распечатанные бланки заданий.

При оформлении заданий производится анализ необходимости посещения торговыми представителями клиентов на основе данных об условиях посещения клиентов торговыми представителями. При оформлении заданий можно использовать информацию о тех товарах, которые ранее закупались клиентом. Задания оформляются в соответствии с теми условиями продаж, которые зарегистрированы в соглашении с клиентом.

Формированием заданий обычно занимается супервайзер - менеджер, ответственный за работу с торговыми представителями.

Важно. Планирование заданий можно использовать только в том случае, если в функциональной опции "Задания торговым представителям" установлено значение "Используются для управления торговыми представителями".

Анализ потребностей в посещении клиентов торговыми представителями

Как открыть форму для анализа потребностей клиентов в посещении торговыми представителями и формирования заданий?

Для анализа потребностей и формирования заданий используется форма "Подготовка заданий". Для открытия формы выполните следующие действия:

  • Откройте раздел "Продажи".
  • Выберите пункт "Подготовка заданий".

В форме "Подготовка заданий" будут показаны те клиенты (партнеры), в карточке которых установлен флаг "Обслуживаются торговыми представителями"

Распределение нагрузки торговых представителей

С помощью какого отчета можно произвести анализ распределения нагрузки торговых представителей?

Для этих целей используется отчет "Распределение торговых представителей". Для вызова отчета выберите в списке отчетов по продажам выберите вариант отчета "Распределение торговых представителей".

Обмен данными с мобильными приложениями

В программе предусмотрена возможность обмена данными с мобильным приложением, используемым торговым представителем в своей работе. Выгрузка и загрузка данных производится автоматически в соответствии с указанными в программе настройками мобильного приложения для каждого торгового представителя. Порядок работы торгового представителя с мобильным приложением, установленным на мобильном устройстве, описан в отдельном файле Руководство пользователя мобильного приложения "Торговый представитель", который включен в поставку конфигурации.

Настройка правил обмена с мобильными устройствами

В программе предусмотрена возможность работы торговых представителей с использованием мобильных устройств (персональные компьютеры, КПК и т.д.). Предусмотрена возможность автоматического обмена данными между управляющей программой и мобильными устройствами.

Для мобильных устройств, которые используются при работе торговых представителей, должны быть определены те правила обмена, которые будут использоваться для осуществления обмена данными между управляющей системой и мобильными устройствами.

Процесс настройки правил обмена состоит из следующих этапов.

  • Определение списка версий мобильных приложений, используемых при работе мобильных устройств.
  • Определение списка мобильных устройств, которые могут быть использованы торговыми представителями.
  • Настройка правил обмена для каждого мобильного устройства.

Определение списка мобильных устройств

Список мобильных компьютеров, которые используют торговые представители, хранится в справочнике "Мобильные компьютеры" (раздел "Администрирование").

Настройка правил обмена

Для каждого торгового представителя (пользователя) необходимо создать те настройки обмена, которые будут использоваться при обмене данными между мобильным компьютером, использующим мобильное предложение, и информационной базой центрального офиса.

Где регистрируется информация о том, по каким правилам производится обмен данными между мобильным устройством и управляющей системой?

Эта информация регистрируется в списке настроек торговых представителей.

  • В списке разделов выберите раздел "Продажи".
  • В панели навигации раздела выберите команду "Настройки торговых представителей".

Ввод данных с бланков заданий

После визита к клиенту торговый представитель, который работает без мобильного устройства, заполняет вручную данные в выданном ему бланке заданий. В бланке заданий торговый представитель вводит информацию о результатах выполнения поставленных перед ним задач, информацию о заказанных клиентом товарах, заметки и прочую информацию о клиентах. В бланк задания также вносится информация о расходах торгового представителя.

Информация о выполнении задания вводится в ранее оформленное и выданное торговому представителю задание. Регистрируется информация о расходах торгового представителя. На основании данных о заказанных позициях в информационной базе регистрируется заказ клиента.

Ввод данных с бланка заданий обычно осуществляет пользователь, ответственный за работу с торговыми представителями (супервайзер).

Как открыть список заданий для внесения данных о результатах выполнения задания торговым представителем?
  • Откройте раздел "Продажи".
  • Выберите пункт меню "Задания торговым представителям".

Как актуализировать данные или ввести новые данные о клиентах в соответствии с той информацией , которую предоставил торговый представитель?

Для ввода информации о новых клиентах или актуализации данных клиента необходимо открыть справочник "Клиенты". Справочник открывается из раздела "Продажи".

В справочнике "Клиенты" регистрируется информация о новом клиенте. При регистрации нового клиента следует использовать помощник ввода нового клиента. Используя помощник ввода можно ввести всю доступную информацию о клиенте , в том числе информацию о его адресах, телефонах и контактном лице. Помощник ввода нового клиента вызывается по кнопке "Создать" в справочнике "Клиенты".

Анализ работы торговых представителей

Для анализа работы торговых представителей используются различные отчеты, которые показывают как эффективность работы торговых представителей с клиентами, так и сравнивают запланированные показатели работы с фактическими (план-фактный анализ). Отчеты по работе с торговыми представителями вызываются из раздела "Продажи".
