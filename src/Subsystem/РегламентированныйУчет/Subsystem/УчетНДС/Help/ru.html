Учет НДС

Регламентированный учет НДС

Программа позволяет формировать книгу покупок и книгу продаж. В программе не реализуются никакие другие аспекты учета НДС, в том числе формирование бухгалтерских проводок, заполнение декларации и пр.
Книги покупок и продаж формируются по профильным операциям конфигурации (например, реализация товаров и услуг, поступление товаров и услуг). По непрофильным операциям (учет по которым ведется в конфигурации не в полном объеме, например учет ОС) для полноты формирования книг покупок и продаж так же ведется учет НДС.

Перед началом учета необходимо выполнить два этапа настройки системы:

  • Проконтролировать установку значений по умолчанию связанных с НДС, для подстановки в первичные документы (например, установку ставок НДС в карточках товаров, установку типа налогообложения в соглашениях с клиентами/поставщиками и т.д.);
  • В форме "Настройка учета НДС" выполнить настройку подсистемы учета НДС для каждой организации (форма открывается соответствующей командой в разделе "Регламентированный учет").

Перед тем как приступить к формированию книги покупок и книги продаж необходимо выполнить следующие регламентные операции (операции выполняются в форме "Регламентные операции по НДС", форма открывается соответствующей командой в разделе "Регламентированный учет", в панели действий "Сервис"):

  • сформировать счета-фактуры на аванс (на авансы выданные поставщикам – вручную, на авансы полученные от покупателей - автоматически);
  • отразить в учете НДС операций по поступлению и выбытию товаров и услуг;
  • распределить НДС, если ведется раздельный учет НДС;
  • подтвердить ставку 0%, если ведется раздельный учет НДС;
  • скорректировать НДС, если требуется отразить в учете непрофильную операцию.

Настройка учета НДС

Перед началом учета НДС необходимо выполнить два этапа настройки системы:

Проконтролировать установку значений по умолчанию (связанных с НДС) для подстановки в первичные документы:

  • Реквизит "Ставка НДС" в справочнике "Номенклатура" (для всех элементов).
  • Реквизит "Система налогообложения" в регистре сведений "Системы налогообложения организаций".
  • Реквизит "Система налогообложения" в документах "Соглашение с покупателями/поставщиками".
  • Флажок "Розничная торговля облагается ЕНВД" для складов организаций в регистре сведений "Применения ЕНВД".

В форме "Настройка учета НДС" выполнить настройку подсистемы.

Как открыть форму настройки учета НДС?

Открытие формы настройки учета НДС производится по команде "Настройка учета НДС" в разделе "Регламентированный учет".

Регламентные операции по учету НДС

Регламентные операции по НДС формируются на одноименной форме в разделе "Регламентированный учет". Для того, чтобы сформировать книгу покупок и книгу продаж необходимо выполнить следующие регламентные операции:

  • сформировать счета-фактуры на аванс (на авансы выданные поставщикам – вручную, на авансы полученные от покупателей - автоматически);
  • отразить в учете НДС операций по поступлению и выбытию товаров и услуг;
  • распределить НДС, если ведется раздельный учет НДС;
  • подтвердить ставку 0%, если ведется раздельный учет НДС;
  • скорректировать НДС, если требуется отразить в учете непрофильную операцию.

Перед формированием регламентных операций с помощью отчета "Контроль НДС" можно выполнить проверку правильности отражения учета НДС в первичных документах.

Для отражения учета НДС в системе используются следующие документы:

  • Для отражения профильных операций служит документ "Формирование записей книги покупок и книги продаж".
  • Для раздельного учета НДС служат документы "Распределение НДС" и "Подтверждение нулевой ставки НДС".
  • Для отражения непрофильных операций служат документы корректировок НДС "Запись книги покупок" и "Запись книги продаж".

Можно ли в программе выполнить контроль первичных документов для учета НДС, перед формированием отчетности?

Для целей контроля первичных документов перед формированием отчетности по НДС служит отчет "Контроль НДС". Отчет предназначен для проверки того, что выполнены все подготовительные действия для начала выполнения регламентных операций по НДС. Отчет можно открыть в панели отчетов в разделе "Регламентированный учет".

Формирование отчетности по учету НДС

Как открыть отчеты "Книга покупок", "Книга продаж" и т.д.?

Программа позволяет формировать следующие отчеты связанные с учетом НДС:

  • "Книга покупок";
  • "Книга продаж";
  • "Журналы счет фактур полученные";
  • "Журналы счет фактур выданные".

Все отчеты находятся в панели отчетов "Отчеты по регламентированному учету". Панель открываются соответствующей командой в разделе "Регламентированный учет".
