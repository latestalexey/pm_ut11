Маркетинговые мероприятия

Управление маркетинговыми мероприятиями

В программе можно регистрировать те маркетинговые мероприятия, которые проводятся в торговом предприятии : проведение выставок, рекламные акции, маркетинговые акции по продаже товаров и т.д. В рамках проведения маркетинговых акций можно устанавливать различные скидки (наценки). Скидки (наценки) устанавливаются документом "Установка скидок (наценок) по маркетинговым мероприятиям".

Важно. Использование маркетинговых мероприятий в программе включается соответствующей функциональной опцией (Администрирование/Маркетинг/Маркетинговые мероприятия). Выключение этой опции отключает не только использование маркетинговых мероприятий , но и возможность установки скидок с помощью документа "Установка скидок по маркетинговым мероприятиям".

Регистрация маркетинговых мероприятий

В программе можно регистрировать различные маркетинговые мероприятия: проведение выставок, проведение рекламных компаний, проведение маркетинговых акций и т.д. Маркетинговые мероприятия можно сгруппировать по каналам рекламного воздействия. Например, можно создать канал рекламного воздействие "Участие в выставках" и по нему регистрировать конкретные мероприятия по проведению конкретных выставок "Бухучет и аудит", "Складская логистика" и т.д.

По каждому мероприятию можно сформировать план проведения мероприятия, определить список участников маркетингового мероприятия, хранить всю историю взаимодействий (телефонные звонки, электронные письма, встречи и т.д.).

Важно. Использование маркетинговых мероприятий в программе включается соответствующей функциональной опцией (Администрирование/Маркетинг/Маркетинговые мероприятия). Выключение этой опции отключает не только использование маркетинговых мероприятий , но и возможность установки скидок с помощью документа "Установка скидок по маркетинговым мероприятиям".

Как открыть список каналов рекламного воздействия?
  • В панели разделов выберите раздел "Маркетинг".
  • В панели навигации раздела выберите "Каналы рекламных воздействий".

Как открыть список маркетинговых мероприятий?
  • В списке разделов выберите раздел "Маркетинг".
  • В панели навигации раздела выберите пункт "Маркетинговые мероприятия".

Установка скидок по маркетинговым мероприятиям

В рамках маркетингового мероприятия могут быть установлены различные скидки (наценки). Список скидок (наценок), действующих при проведении маркетинговой акции регистрируется в документе "Установка скидок по маркетинговым мероприятиям".

Создание документа "Установка скидок по маркетинговым мероприятиям"

Документ "Установка скидок по маркетинговым мероприятиям" может быть создан из списка документов или на основании маркетингового мероприятия.

Как открыть список документов "Установка скидок по маркетинговым мероприятиям"?
  • В списке разделов выберите раздел "Маркетинг".
  • В панели навигации раздела выберите пункт "Установка скидок по маркетинговым мероприятиям".

Анализ откликов по маркетинговым мероприятиям

Для анализа откликов по маркетинговым мероприятиям используется отчет "Анализ первичного интереса". Отчет позволяет проанализировать как эффективность использования различных каналов воздействия (реклама, выставки и т.д.) , так и проанализировать, какой из источников воздействия является наиболее эффективным (конкретная выставка, совет знакомого и т.д.).

Как открыть отчет для анализа сравнительной эффективности каналов рекламного воздействия (выставок, рекламных компаний и т.д)?
  • В списке разделов выберите раздел "Продажи".
  • В панели действий в группе "Отчеты" выберите отчет "Анализ первичного интереса".
