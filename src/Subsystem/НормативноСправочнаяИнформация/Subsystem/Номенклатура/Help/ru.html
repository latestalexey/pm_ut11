Настройка ведения номенклатуры

Перед началом заполнения информации о номенклатуре необходимо заполнить основные классификаторы , которые используются при вводе информации о номенклатуре.

  • Классификатор единиц измерения.
  • Классификатор видов номенклатуры.
  • Классификатор видов и значений свойств номенклатуры.

Формирование и поддержка видов номенклатуры

Информация о всех видах номенклатуры, используемых на предприятии, заполняется в справочнике "Виды номенклатуры". Справочник "Виды номенклатуры" - это классификатор, предназначенный для объединения номенклатуры по общим признакам учета. В конфигурации предусмотрены варианты ведения дополнительного учета номенклатуры в зависимости от ее вида - например, учет по дополнительным характеристикам и учет упаковок номенклатуры. Используя этот классификатор, можно задать произвольные дополнительные сведения для номенклатуры и характеристик, относящихся к определенному виду номенклатуры.

Как открыть список видов номенклатуры?
  • В списке разделов выберите раздел "Нормативно справочная информация".
  • В панели навигации раздела выберите пункт "Виды номенклатуры".

Формирование и поддержка упаковок номенклатуры

Информация о всех видах упаковок, которые могут использоваться на предприятии, вводится в справочник "Единицы измерения". Для каждой единицы измерения, введенной в справочник, заполняется информация о коде единицы измерения, кратком и полном наименовании единицы измерения, а также международное сокращение единицы измерения.

Все реквизиты следует заполнять в соответствии с Общероссийским классификатором единиц измерения (ОКЕИ).

Как открыть справочник единиц измерения?

  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Единицы измерения".

Задание наборов упаковок

Как открыть список наборов упаковок?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Наборы упаковок".
  • Будет открыт список наборов упаковок.

В списке наборов упаковок присутствует предопределенный элемент "Индивидуальный набор для номенклатуры". Этот элемент следует использовать в том случае, если список упаковок для номенклатуры задается индивидуально, непосредственно в карточке номенклатуры.

Задание типоразмеров упаковок

Как ввести информацию о типовых размерах упаковок, применяемых в торговом предприятии?
  • В справочнике "Типоразмеры упаковок" нажмите на кнопку "Создать".
  • В новой карточке введите наименование типоразмера упаковки. Наименование будет использоваться для заполнения информации о размерах упаковки.
  • Укажите линейные размеры: высоту, ширину, глубину упаковки. Размеры указываются в тех единицах измерения, которые определены в настройках параметров учета.
  • Объем будет рассчитываться автоматически в соответствии с указанными в настройках параметров учета единицей измерения объема и коэффициента пересчета.

Важно. Если на складе будут использоваться упаковки для которых не важны размеры или эти параметры упаковок не используются, то введите в качестве типового размера упаковки типоразмер с неограниченными размерами.

Настройка шаблона ценников и этикеток

Как открыть список шаблонов ценников и этикеток?
  • В списке разделов выберите раздел "Розничные продажи".
  • В панели навигации раздела выберите "Шаблоны этикеток и ценников".

Задание шаблонов ценников и этикеток для видов номенклатуры

Для каждого вида номенклатуры может быть задан свой шаблон для печати ценника и этикетки.

Как открыть список видов номенклатуры?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Виды номенклатуры".
  • Будет открыт список видов номенклатуры.

Задание складских групп упаковок

Как открыть список складских групп упаковок?
  • В спитске разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите "Складские группы упаковок".

Есть ли возможность создать шаблон для наименований номенклатурных позиций?

В программе существует возможность задать правила по которым у номенклатурной позиции будет автоматически формироваться "Рабочее наименование" и "Наименование для печати". Если для номенклатуры ведется учет в разрезе характеристик, то по такому же принципу могут формироваться "Рабочее наименование" и "Наименование для печати" у характеристики номенклатуры. Правила задаются с помощью шаблонов. Шаблоны указанных наименований формируются для каждого вида номенклатуры (в справочнике "Виды номенклатуры", в группе полей "Шаблоны наименований").

Ведение списка номенклатуры

Система позволяет хранить информацию о номенклатуре предприятия, ее видах (товар, услуга), характеристиках, упаковках.

Как открыть список позиций номенклатуры?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите "Номенклатура".
