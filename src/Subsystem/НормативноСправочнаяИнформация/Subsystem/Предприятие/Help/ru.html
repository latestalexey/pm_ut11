Предприятие

Ведение информации о структуре предприятия

Система позволяет заполнять и поддерживать в актуальном состоянии справочную информацию о структуре предприятия.

Ведение информации об организации

Торговое предприятие может состоять из нескольких организаций, от имени которых будут оформляться документы. В программе ведется учет в разрезе организаций торгового предприятия. Для каждой организации определяется список касс, банковских счетов, система налогообложения. Для каждой организации можно настроить свою систему распределения финансовых результатов по направлениям деятельности.

Как открыть список организаций?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите "Организации".

Настройка списка обособленных подразделений

В программе можно вести расчеты с обособленными подразделениями организации. Обособленное подразделение организации это любое территориально обособленное от нее подразделение, по месту нахождения которого оборудованы стационарные рабочие места. В программе предусмотрено ведение учета для обособленных подразделений двух видов: выделенных на отдельный баланс (филиалов, представительств) и тех подразделений, которые не выделены на отдельный баланс (торговые точки, магазины и т.д.).

Важно. Регистрация обособленных подразделений, выделенных на отдельный баланс, возможна только в том случае, если включена функциональная опция Обособленные подразделения на отдельном балансе.

Как открыть список складов торгового предприятия?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Склады (складские территории)".

Как открыть список подразделений, входящих в состав торгового предприятия?
  • В панели разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Структура предприятия".

Ведение информации о проектах

Информация о проектах используется для ввода данных о долгосрочных проектах. Программа позволяет хранить информацию об этапах проекта (план-проекта), регистрировать участников проекта, контролировать выполнение этапов проекта, выдавать необходимые распоряжения участникам проекта по выполнению того или иного этапа. В рамках проекта можно хранить список всех взаимодействий с участниками проекта.

Как открыть справочник "Проекты"?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Проекты".
