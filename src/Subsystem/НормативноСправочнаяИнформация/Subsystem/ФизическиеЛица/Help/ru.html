Настройка и ведение списка физических лиц

Настройка ведения списка физических лиц

Перед началом заполнения списка физических лиц необходимо выполнить необходимые настройки по ведению списка физических лиц.

  • Заполнить классификаторы таким образом, чтобы при заполнении данных физического лица пользователь мог выбрать значение из готового списка.
  • Настроить виды контактной информации для ввода контактной информации по физическим лицам.
  • Указать список дополнительных свойств (дополнительных реквизитов и сведений).
  • Указать список тех видов связей, которые могут быть между физическими лицами.

Заполнение классификатора документов, удостоверяющих личность

Для каждого физического лица можно хранить информацию по стандартным типам документов (паспорт, свидетельство о рождении и т.д.). В поставку программы входит классификатор документов, удостоверяющих личность, в котором заполнены предопределенные виды документов. При необходимости пользователь может отредактировать этот классификатор и ввести свои виды документов.

Как открыть классификатор документов, удостоверяющих личность?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Документы, удостоверяющие личность".

Как добавить новый вид контактной информации в справочник "Физические лица"?
  • Откройте список видов контактной информации ("Администрирование"- "Виды контактной информации").
  • Установите курсор на группу "Контактная информация справочника "Физические лица"".
  • Нажмите на кнопку "Создать". Появится карточка нового вида контактной информации.
  • Укажите название нового вида контактной информации, например "Рабочий телефон".
  • Укажите тип информации, например "Телефон".
  • Введите дополнительную информацию, если это необходимо. Например, можно указать, что информация будет редактироваться только в диалоговом окне.
  • Нажмите на кнопку "Записать и закрыть".
  • Информация о новом виде контактной информации автоматически появится в форме справочника "Физические лица".

Заполнение видов связей между физическими лицами

В программе предусмотрена возможность хранения связей между физическими лицами. Для хранения видов связей , которые могут быть применены в торговом предприятии предусмотрен справочник "Виды связей физических лиц".

Как открыть справочник для определения видов связей между физическими лицами?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Виды связей физических лиц".

Ведение списка физических лиц

Программа позволяет вести список физических лиц и хранить подробную информацию о них. В качестве дополнительной информации по физическому лицу может быть добавлена информация о взаимосвязях физических лиц между собой, а также информация о том, кем является данное физическое лицо: контрагентом (индивидуальным предпринимателем), контактным лицом одного или нескольких партнеров и т.д.

Как открыть список физических лиц?
  • В списке разделов выберите раздел "Нормативно-справочная информация".
  • В панели навигации раздела выберите пункт "Физические лица".
