Как оформить новый документ расчета себестоимости?
  • В списке регламентных документов нажмите на кнопку "Создать".
  • Будет создан документ "Расчет себестоимости товаров".
  • В документе указывается текущая дата, в качестве ответственного указывается текущий пользователь, который оформляет данный документ.
  • Нажмите на кнопку выбора организации и выберите ту организацию, по которой должна быть рассчитана себестоимость.
  • Введите в поле "Комментарий" необходимые пояснения по расчету себестоимости.
  • Нажмите на кнопку "Провести и закрыть".

Важно. Расчет себестоимости производится за текущий месяц. Если в течение текущего месяца были оформлены передачи товаров между организациями, то в документе будет предложено выбрать несколько организаций. В список добавляются те организации, между которыми были оформлены документы передачи товаров. Расчет себестоимости будет производиться одновременно по компании в целом (без учета передач товаров между организациями) и по каждой организации отдельно ( с учетом передач товаров между организациями).

Как отобрать в списке только те регламентные документы, которые относятся к конкретной организации?
  • Выберите в поле "Организация" в списке конкретную организацию, регламентные документы которой необходимо посмотреть.
  • В списке будут автоматически отобраны только документы выбранной организации.
  • При оформлении нового документа в документ будет автоматически подставляться информация о выбранной организации.

Работа с документом "Распределение расходов на себестоимость товаров"

Как добавить новый документ "Распределение расходов на себестоимость товаров"?
  • В списке регламентных документов укажите организацию по которой необходимо провести распределение расходов на себестоимость товаров.
  • Нажмите на кнопку "Добавить".
  • Выберите из появившегося списка документ "Распределение расходов на себестоимость товаров".
  • В новом документе заполните список расходов, которые необходимо распределить на себестоимость товаров.
  • Предусмотрено как автоматическое заполнение списка расходов, так и ручное заполнение списка расходов.
  • Проверьте правильность заполнения списка расходов.
  • Проведите документ.

Важно. С помощью одного документа можно оформить распределение расходов на себестоимость товаров по всем организациям. Для этого следует в документе установить переключатель в положение По всем организациям.

Как добавить новый документ для распределения доходов и расходов по направлениям деятельности?
  • В списке регламентных документов укажите организацию, для которой необходимо произвести распределение доходов и расходов по направлениям деятельности.
  • Нажмите на кнопку "Создать" .
  • Выберите из появившегося списка документ "Распределение доходов и расходов". Будет создан новый документ "Распределение доходов и расходов".
  • В документе будет заполнена организация, в соответствии с той организацией, которая была указана в списке регламентных документов.
  • В качестве ответственного в документе будет указан текущий пользователь.
  • В табличной части документа заполните список доходов и расходов, который необходимо распределить по направлениям деятельности.
  • Проведите документ.

Документ при проведении распределяет прочие доходы и расходы на направления деятельности в соответствии с заданным в документе способом распределения и аналитикой.

Важно. Распределение расходов и доходов по направлениям деятельности можно производить по всем организациям одновременно. Если необходимо выполнить такую операцию, то установите в документе переключатель в положение По всем организациям.
