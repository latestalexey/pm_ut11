Журнал документов "Складские акты"

Журнал предназначен для хранения списка складских ордеров: Приходные ордера на товары, Расходные ордера на товары, Ордер на перемещение товаров между помещениями. Складские ордера могут оформляться только на основании документов-распоряжений. Для оформления складских ордеров при приеме и отгрузке товаров предусмотрены специальные обработки (рабочие места кладовщиков): Приемка товаров и Отгрузка товаров. Из данного журнала можно оформить только документ Ордер на перемещение товаров между помещениями.
