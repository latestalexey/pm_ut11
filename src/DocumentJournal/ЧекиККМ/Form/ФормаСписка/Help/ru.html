Как указать кассу ККМ, по которой будут оформляться розничные продажи?

В поле "Касса ККМ" выберите ту кассу, на которой должны пробиваться чеки ККМ. Для выбора доступны только те кассы, которые подключены в режиме фискального регистратора (для кассы установлен тип "Фискальный регистратор" и подключена модель используемого фискального регистратора).

Как открыть кассовую смену?

В рабочем месте кассира нажмите на кнопку "Открыть смену". Будет открыта кассовая смена и кассир сможет оформлять новые чеки на продажу товаров.

В рабочем месте кассира внизу списка чеков после открытия смены будет выведена строка с состоянием смены (смена открыта) и имя кассира (текущего пользователя) , который в данный момент работает с программой.

Важно. Кассовая смена может быть открыта и закрыта в любое время. Продолжительность кассовой смены не должна превышать 24 часов. В течение одного дня на одной кассе может работать несколько кассиров. При передаче кассовой смены кассир закрывает смену, сдает выручку, выходит из программы. Новый кассир входит в программу под своим именем и открывает новую кассовую смену.

Как отобрать список отложенных чеков в течение всего периода работы кассы ККМ?
  • В списке чеков выберите кассу ККМ, по которой нужно посмотреть список отложенных чеков.
  • В поле "Фильтр" установите значение "Все отложенные".

Как оставить в списке только те чеки, которые оформлены в текущей кассовой смене?
  • В списке чеков выберите кассу ККМ, по которой нужно посмотреть список текущих чеков.
  • В поле "Фильтр" установите значение "Только текущая смена".

Как оставить в списке только те чеки, которые были отложены в текущей смене?
  • В списке чеков выберите кассу ККМ, по которой нужно посмотреть список отложенных чеков.
  • В поле "Фильтр" установите значение "Отложенные по текущей смене".

Как отобрать список чеков, оформленный в конкретной кассе ККМ?

Для этого в списке надо выбрать конкретную кассу ККМ.

Как ввести новый чек ККМ?
  • В рабочем месте кассира нажмите на кнопку "Создать".
  • Будет создан новый документ "Чек ККМ" .
  • В новом документе будет автоматически заполнена информация о кассе ККМ, по которой открыта кассовая смена.
  • Информация о складе заполнится в соответствии с тем складом, который указан для кассы ККМ.

Примечание. Если чек необходимо оформить по другой кассе, то небходимо открыть кассовую смену для другой кассы и пробить чек по другой кассе.

Как оформить чек на возврат товара?

  • В рабочем месте кассира (списке документов "Чек ККМ") установите курсор на поле "Номер чека".
  • Нажмите на кнопку "Найти".
  • В поле "Что искать" укажите номер того чека, по которому покупатель возвращает товар.
  • Нажмите на кнопку "Найти".
  • В списке останется тот чек, по которому покупатель хочет вернуть товар.
  • Нажмите на кнопку "Чек ККм на возврат:создать на основании".
  • Будет сформирован чек на возврат, на основании указанного чека.

Примечание. Для того , чтобы убрать отбор в списке, нажмите кнопку "Отменить поиск в списке" (рядом с кнопкой "Найти").

Как внести денежные средства в денежный ящик кассы ККМ?
  • Нажмите на кнопку "Внесение денег" в командной панели списка.
  • Появится список распоряжений (документов "Расходный кассовый ордер"), которым была оформлена выдача денежных средств в кассу ККМ.
  • Установите курсор на нужный документ и нажмите на кнопку "Принять в кассу".

Денежные средства будут внесены в денежный ящик ККМ. Информация о том, какое количество денежных средств находится в денежном ящике ККМ отображается в нижней части списка.

Как выполнить процедуру выемки денежных средств из денежного ящика кассы ККМ?
  • В рабочем месте кассира (списке документов "Чек ККМ" ) нажмите на кнопку "Выемка денег".
  • Укажите сумму денежных средств, которые изымаются из кассы ККМ.
  • Нажмите на кнопку "ОК".

Важно. После выемки денежных средств из кассы ККМ необходимо передать денежные средства в кассу предприятия.

Важно. В некоторых моделях ФР предусмотрена возможность автоматической регистрации инкассации денежных средств при закрытии кассовой смены. В таких кассах следует установить флаг "Автоматическая инкассация денежных средств при снятии Z-отчета на фискальном регистраторе". При установке флага при закрытии кассовой смены автоматически формируется документ "Выемка денежных средств из кассы ККМ".

Закрытие кассовой смены

Как посмотреть информацию о всех оформленных чеках в течение кассовой смены?

Информация обо всех оформленных чеках отображается в рабочем месте кассира (список чеков ККМ). Чеки, по которым был оформлен возврат товаров, помечаются флагом в поле "Возврат".

Можно ли распечатать на фискальном регистраторе отчет без гашения (x-отчет)?

Для печати на фискальном регистраторе отчета без гашения выполните следующие действия.

  • В панели действий "Сервис" раздела "Продажи" выберите пункт "Управление фискальным регистратором".
  • Нажмите на кнопку "Печать отчета без гашения".

Примечание. С помощью команды "Печать отчета с гашением" можно сформировать z-отчет на фискальном регистраторе без закрытия кассовой смены.

Как закрыть кассовую смену?

Для закрытия кассовой смены нажмите на кнопку "Закрыть смену " в командной панели списка "Чеки ККМ" .

Будет сформирован и распечатан Z-отчет за кассовую смену по оформленным чекам и автоматически сформирован документ "Отчет о розничных продажах".
