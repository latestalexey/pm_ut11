Справочник "Мобильные компьютеры"

Справочник "Мобильные компьютеры" предназначен для хранения списка мобильных компьютеров, которые используются при работе торговых представителей.

При выдаче торговому представителю мобильного компьютера в список вводится новая запись о мобильном компьютере с указанием наименования компьютера и его серийного номера.

Для работы с мобильными компьютерами используется продукт "1С:Предприятие 8 Расширение для карманных компьютеров". Для получения лицензии на работу с этим продуктом необходимо пройти процедуру лицензирования на веб-сайте 1С. Переход на веб-сайт может быть произведен непосредственно из карточки мобильного компьютера по кнопке "Активация лицензии на веб-сайте 1С".

Данные по лицензированию (регистрационный номер, регистрационный код, код активации) могут быть сохранены в карточке мобильного компьютера для их дальнейшего использования.

Данные о мобильном компьютере используются при настройке правил обмена данными между компьютером торгового представителя с управляющей системой.
