Условия обслуживания клиентов торговыми представителями

Форма предназначена для группового назначения новых или изменения существующих условий обслуживания партнеров торговыми представителями.
Для изменения имющихся условий необходимо выделить в списке те условия, которые требуется изменить, после чего выбрать команду Изменить выделенные условия. Для добавления новых условий необходимо выбрать команду Добавить условия.

Как отобрать в списке условия обслуживания, по которым закреплен конкретный торговый представитель?

Для отбора по торговому представителю необходимо задать быстрый отбор в поле Торговый представитель в верхней части формы

Как отобрать в списке условия обслуживания, по которым указан конкретный куратор?

Для отбора по куратору необходимо задать быстрый отбор в поле Куратор в верхней части формы

Как отобрать в списке условия обслуживания, принадлежащие партнерам, отнесенным к конкретному бизнес-региону?

Для отбора по бизнес-региону необходимо задать быстрый отбор в поле Бизнес-регион в верхней части формы

Как изменить несколько условий обслуживания?

Для изменения нескольких условий одновременно необходимо выделить в списке те строки, которые требуется изменить, после чего выбрать команду Изменить

Как добавить однотипные условия обслуживания для нескольких партнров сразу?

Для того, чтобы назначить нескольким партнерам одинаковые условия обслуживания, необходимо выбрать команду "Добавить условия".
