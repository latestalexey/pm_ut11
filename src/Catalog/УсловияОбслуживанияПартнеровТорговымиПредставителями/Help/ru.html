Справочник "Условия обслуживания партнеров торговыми представителями"

В справочнике "Условия обслуживания партнеров торговыми представителями" указываются те условия в соответствии с которыми клиент обслуживается торговым представителем.

Условия обслуживания торговым представителем могут быть указаны для каждого партнера торгового предприятия. Для партнера указывается торговый представитель, который выбирается из списка пользователей программы, указывается расписание времени посещения торговым представителем партнера. Также можно указать куратора (супервайзера) , который будет контролировать работу торгового представителя с клиентом.

Информация об условиях обслуживания партнеров торговыми представителями используется при оформлении заданий торговым представителям, при планировании работы торговых представителей.

Важно. Работа с торговыми представителями регулируется установкой флага "Обслуживается торговым представителем" в карточке партнера. Если этот флаг не установлен, то для данного партнера работа с торговым представителем не предусмотрена. Условия обслуживания необязательны к заполнению при работе с торговым представителем при оформлении карточки нового партнера. Флаг "Обслуживается торговым представителем" может быть установлен в любой момент времени, а условия обслуживания могут быть заданы в любой момент времени, например, при оформлении документа "Задание торговому представителю".
