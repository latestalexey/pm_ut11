Справочник "Маркетинговые мероприятия"

Справочник "Маркетинговые мероприятия" предназначен для регистрации различных маркетинговых мероприятий: проведения выставок, проведения рекламных компаний, проведения маркетинговых акций и т.д.

Маркетинговые мероприятия можно сгруппировать по каналам рекламного воздействия. Например, можно создать канал рекламного воздействие "Участие в выставках" и по нему регистрировать конкретные мероприятия по проведению конкретных выставок "Бухучет и аудит", "Складская логистика" и т.д.

В рамках маркетингового мероприятия можно устанавливать дополнительные скидки. Дополнительные скидки по маркетинговым мероприятиям регистрируются документом "Установка скидок по маркетинговым мероприятиям". В рамках маркетингового мероприятия можно указать список участников и создать план проект выполнения заданий по маркетинговому мероприятию.
