Маркетинговые мероприятия

Как добавить новое маркетинговое мероприятие?

Как ввести информацию о скидках (наценках), которые действуют в рамках маркетингового мероприятия?

Как добавить новое маркетинговое мероприятие?
  • В списке маркетинговых мероприятий нажмите на кнопку "Создать".
  • Будет создано новое маркетинговое мероприятие.
  • Укажите наименование мероприятия и введите данные о плановых сроках проведения мероприятия.

Как ввести информацию о скидках (наценках), которые действуют в рамках маркетингового мероприятия?
  • В списке "Маркетинговые мероприятия" установите курсор на то маркетинговое мероприятие, по которому надо зарегистрировать скидки (наценки).
  • Нажмите на кнопку "Создать на основании".
  • Выберите из списка "Установка скидок по маркетинговым мероприятиям".
  • Будут создан новый документ "Установка скидок по маркетинговым мероприятиям".
  • В документе будет заполнена информация о маркетинговом мероприятии. Если в маркетинговом мероприятии был определен сегмент партнеров, для которого оно действует, то эта информация также перенесется в документ "Установка скидок по маркетинговым мероприятиям".
  • В табличную часть документа необходимо ввести информацию о скидках ( наценках), которые будут действовать в рамках маркетингового мероприятия.

Примечание. Новый документ "Установка скидок по маркетинговым акциям" может быть введен из списка документов и в нем может быть указано маркетинговое мероприятие вручную.
