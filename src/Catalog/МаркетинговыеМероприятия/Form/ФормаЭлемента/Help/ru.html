Как заполняется информация о плановом и фактическом сроке действия мероприятия?

При вводе нового мероприятия заполняется информация в полях "Плановая дата начала" и "Плановая дата окончания". После завершения мероприятия информация о фактических датах проведения мероприятия уточняется и вводится в поля "Дата начала" и "Дата окончания".

Как задать целевые сегменты номенклатуры и клиентов для проведения маркетингового мероприятия?

Эта информация задается в полях "Сегмент номенклатуры" и "Семент партнеров".

Важно. Для каждого мероприятия может быть создан отдельный целевой сегмент по товарам (сегменту номенклатуры) или для группы клиентов (сегмент партнеров).

Как закрыть мероприятие после выполнения всех работ по мероприятию?

После выполнения всех работ по маркетинговому мероприятию у него можно вручную установить признак "Завершено" и провести документ.

Работа по мероприятию

Работа со списком партнеров и контактных лиц

Как указать список лиц, которые участвуют в маркетинговом мероприятии?

С каждым маркетинговым мероприятием можно связать ряд партнеров и их контактных лиц, которые так или иначе участвуют в нем. Это могут быть поставщики, конкуренты, лица, участвующие в организации работ по мероприятию и т.д. Список таких партнеров и контактных лиц заполняется менеджером в форме маркетингового мероприятия. Для каждого партнера можно указать ту роль, которую он играет в данном мероприятии. Роли партнеров выбираются из соответствующего классификатора ролей. Роли контактных лиц, участвующих в мероприятии из классификатора не выбираются - их можно указать в комментариях к строке участника.

Как добавить нового партнера в список участников маркетингового мероприятия?

В список можно добавлять любых партнеров, в том числе тех, которые входят в сегмент, запрещающий продажи (заключение сделок). После добавления для партнера нужно указать роль, выбрав ее из классификатора ролей.

  • На странице "Партнеры и контактные лица" нажмите на кнопку "Добавить".
  • В новой строке укажите выберите партнера из списка и выберите роль партнера.
  • Аналогичным образом заполните информацию о других участниках маркетингового мероприятия.

Как добавить контактное лицо в список участников маркетингового мероприятия?

Помимо партнера можно указать контактное лицо. Если уже выбран партнер, множество возможных контактных лиц ограничивается контактными лицами этого партнера. При добавлении контактного лица автоматически подставляется партнер и, если роль партнера не определена, пользователь указывает роль партнера.

Управление проектом маркетингового мероприятия

Для маркетингового мероприятия можно создать план проекта, определить последовательность задач проекта.

Как открыть план проект по маркетинговому мероприятию?
  • В панели навигации формы маркетингового мероприятия выберите пункт "План проекта".
  • Откроется список тех заданий, которые должны быть выполнены в рамках данного маркетингового мероприятия.
  • Для каждого задания указывается исполнитель, куратор, плановая и фактическая дата выполнения задания, стадия выполнения задания.

Управление взаимодействиями по мероприятию

Как посмотреть список взаимодействий по маркетинговому мероприятию?
  • В панели навигации формы маркетингового мероприятия выберите пункт "Взаимодействия".
  • Будет показан список всех взаимодействий по данному маркетинговому мероприятию (телефонные звонки, встречи, электронные письма и т.д.). В списке будут показаны те взаимодействия, в которых маркетинговое мероприятие указано в качестве предмета взаимодействия.

Как спланировать взаимодействия по маркетинговому мероприятию?

  • В списке "Партнеры и контактные лица" установите курсор на того участника, по которому надо запланировать взаимодействие.
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Запланировать взаимодействие".
  • Будет создана запись о новом запланированном взаимодействии с партнером.

Как создать электронное письмо на основании маркетингового мероприятия?
  • В списке "Партнеры и контактные лица" установите курсор на того участника, которому нужно позвонить.
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Написать письмо".
  • Будет создано новое электронное письмо.

При создании электронного письма на основании маркетингового мероприятия автоматически заполняется реквизит "предмет". В качестве получателя письма указывается контактное лицо партнера и его электронный адрес. Если информация о контактном лице не заполнена, то заполняется информация о партнере.

Как создать телефонный звонок на основании маркетингового мероприятия?
  • В списке "Партнеры и контактные лица" установите курсор на того участника, которому нужно позвонить.
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Позвонить".
  • Будет создана запись о новом телефонном звонке клиенту .

В качестве абонента будет указано контактное лицо партнера. В поле "Как связаться" будут указана информация о телефонах контактного лица. Если контактное лицо не указано, то в качестве абонента будет указана информация о партнере.

Как запланировать встречу на основании маркетингового мероприятия?
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Запланировать встречу".
  • Будет создана запись о новой встрече.
  • В качестве участноков встречи будет заполнен весь список участников, указанный в маркетинговом мероприятии. Для всех участников будет указана контактная информация.

Можно ли по маркетинговому ввести задачу (поручение, напоминание)?

По маркетинговому мероприятию можно вести перечень задач с указанием подробных действий по задачам и контролем их выполнения .

Можно ли в рамках маркетингового мероприятия хранить внешнюю информацию в виде файлов (план выставки, договор о рекламе и т.д.). ?

К маркетинговому мероприятию можно прикрепить любое количество дополнительных файлов.

Создание электронного письма на основании маркетингового мероприятия

При создании электронного письма на основании маркетингового мероприятия автоматически заполняется реквизит "предмет". В список получателей письма добавляются записи, соответствующие контактной информации партнеров и контактных лиц маркетингового мероприятия с типом "e-mail".
