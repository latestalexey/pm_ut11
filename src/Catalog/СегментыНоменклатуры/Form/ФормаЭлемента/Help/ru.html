Работа с сегментом номенклатуры

Какая информация должна быть заполнена в карточке сегмента номенклатуры?

Наименование - текстовое наименование сегмента. Наименование сегмента показывается в списках для выбора и используется для быстрой идентификации сегмента номенклатуры.
Описание - подробное описание назначения сегмента номенклатуры, правил его формирования.
Способ формирования - способ формирования сегмента.

Предусмотрено три способа формирования сегмента.

  • "Формировать вручную" - состав сегмента пользователь формирует вручную, есть возможность добавлять новые позиции номенклатуры в сегмент, удалять имеющиеся позиции номенклатуры из сегмента. При создании сегмента вручную можно задавать правила, по которым будут отбираться позиции номенклатуры в сегмент.
  • "Периодически обновлять" - состав сегмента вычисляется по правилам, указанным в настройках. Состав сегмента может обновляться пользователем вручную или с помощью регламентного задания. Регламентное задание может выполняться в соответствии с заданным периодом обновления. Результаты вычисления (состав сегмента) сохраняются в информационной базе и используются до момента очередной итерации вычисления (обновления).
  • "Формировать динамически" - состав сегмента вычисляется каждый раз по правилам, которые для него заданы, в информационной базе результаты вычислений не хранятся.

После указания всех основных параметров нажмите на кнопку "Записать". Настройка правил формирования сегмента производится только после того, как определены основные параметры сегмента (в частности, способ формирования сегмента партнера).

Важно. После записи информации о сегменте изменить способ формирования сегмента может только пользователь, которому разрешено изменять реквизиты объекта. Динамические сегменты (способ формирования - Формировать динамически) используются только для анализа данных номенклатуры, они не могут быть использованы в качестве условий продаж в соглашении, установке условий по скидкам, ценам и т.д.

Определение правил формирования сегмента номенклатуры

Как открыть форму для настройки правил формирования сегментов партнеров?
  • В карточке сегмента номенклатуры нажмите на кнопку "Редактировать".
  • Будет открыта форма для настройки правил формирования сегмента номенклатуры.
  • Для настройки правил формирования сегмента будет загружена основная схема, с помощью которой может быть произведена настройка правил формирования сегмента с использованием тех параметров, которые определены для номенклатуры.

Как задать правила по которым будет сформирован состав сегмента номенклатуры?

При настройке правил формирования сегмента используются система компоновки данных. Пользователю предоставляется упрощенная схема настройки. С помощью упрощенной схемы настройки пользователь может редактировать следующие настройки правил формирования сегмента.

  • Производить отбор данных в соответствии с параметрами, которые определены схемой компоновки сегмента ("Отбор").
  • Задавать значение параметров, если это предусмотрено схемой компоновки сегмента (Параметры).

При формировании нового сегмента загружается основная схема компоновки данных для формирования состава сегмента номенклатуры. В поставку программы входят дополнительные упрощенные схемы настройки, которые поставляются в виде шаблонов.

Можно ли в качестве параметров отбора выбрать такие параметры, как конкретная группа номенклатуры или объем продаж определенного вида номенклатуры?

Для этого в программе разработаны шаблоны запросов (схем компоновки данных) , которые могут быть использованы пользователем.

  • В форме настройки правил формирования сегмента в поле "Схема компоновки данных"
  • выберите из выпадающего списка нужный шаблон.
  • Будет загружен шаблон схемы компоновки данных, с помощью которого можно произвести отбор по дополнительным параметрам.

В программе предусмотрены следующие варианты шаблонов для настройки правил формирования сегмента номенклатуры.

  • Основная схема. Загружается автоматически при создании нового сегмента номенклатуры. В качестве параметров отбора могут быть использованы любые реквизиты номенклатуры, в том числе характеристики номенклатуры, а также дополнительные сведения и дополнительные реквизиты. .
  • Продажи. В качестве параметров отбора кроме реквизитов номенклатуры может быть использована информация о о продажах партнеров. В качестве дополнительных параметров отбора добавляются такие параметры, как "Валовая прибыль", "Себестоимость Оборот", "Сумма выручки Оборот" , "Сумма дополнительных расходов Оборот", "Количество Оборот". Используя этот вариант настройки можно например, отобрать те позиции номенклатуры, у которых сумма выручки за прошлый период меньше определенного значения. Также в этой схеме предусмотрена возможность отбора по типу запасов (собственный товар, комиссионный товар), по конкретному складу или по продажам конкретному партнеру.

Как отредактировать схему компоновки данных?

Важно. Схема компоновки данных может быть отредактирована только в режиме "толстого клиента".
  • Для редактирования схемы компоновки данных нажмите на кнопку "Редактировать" в форме сегмента.
  • В диалоговом окне упрощенных настроек нажмите на кнопку "Редактировать схему компоновки данных".

Схема компоновки данных изначально содержит два запроса.

  • "ФормированиеСегмента" - запрос, определяющий правила вхождения номенклатуры в сегмент, запрос должен возвращать только два поля: "Номенклатура" и "Характеристика".
  • "ВыводСегмента" - запрос, в котором задаются поля, которые будут отображаться в отчете о составе сегмента.

Важно. Имена запросов менять нельзя. Запросы должны быть связаны между собой, связь удалять нельзя.

Пользователь может отредактировать текст и параметры этих запросов, однако он должен обеспечить присутствие в их результатах полей, которые обеспечивают связь этих запросов между собой и корректное размещение данных в регистре, хранящем множество позиций номенклатуры, входящих в сегмент.

Схема, разработанная пользователем, в списке схем компоновки данных будет сохранена в качестве произвольной схемы.

Как сформировать сегмент номенклатуры?
  • Для формирования сегмента номенклатуры нажмите на кнопку "Сформировать" в карточке сегмента.
  • Будет сформирован список номенклатурных позиций (состав сегмента) в соответствии с теми правилами, которые указаны в настройках формирования сегмента.

Важно. Динамические сегменты формируются без участия пользователя. Формирование сегмента запускается в момент просмотра состава сегмента.

Важно. Сегменты, для которых установлен способ формирования "Периодически обновлять" могут обновляться по расписанию с помощью регламентных заданий. Для каждого сегмента используется свое регламентное задание, расписание которого задается в форме сегмента. Список регламентных заданий для формирования сегментов можно посмотреть в разделе "Администрирование"/"Поддержка и обслуживание" ("Регламентные и фоновые задания").

Как определить пересечения состава данного сегмента с другими сегментами номенклатуры
  • По команде панели навигации Пересечения сегмента перейти к отчету "Пересечения сегмента номенклатуры". В данный отчет выводится информация о сегментах, в состав которых входит хотя бы одна номенклатура, включенная в данных сегмент. Для каждого из таких сегментов выводится номенклатура, которая входит и в состав данного сегмента.
