Определение способов распределения доходов и расходов на направления деятельности

В программе предусмотрена возможность гибкой настройки распределения доходов и расходов по направлениям деятельности.

Для каждой статьи доходов и расходов можно задать свой способ распределения по направлениям деятельности.

Пример. Сумма расходов по проведению маркетинговой акции должна пропорционально распределяться по направлениям деятельности "Продажа дистрибюторам" и "Оптовая торговля".

Для задания правил распределения по одному или нескольким направлениям деятельности используется список "Способы распределения доходов и расходов по напралениям деятельности".

По каким правилам могут быть распределены доходы и расходы по направлениям деятельности?

Выбор правила распределения осуществляется из списка значений:

  • Пропорционально расходам. Данное правило распределения может использоваться при распределении прочих расходов по видам деятельности. Базой распределения служит сумма себестоимости продаж, распределенная по видам деятельности, осуществляемым организацией.
  • Пропорционально доходам. Данное правило распределения может использоваться при распределении прочих расходов по видам деятельности. Базой распределения служит сумма выручки от продаж организации, распределенная по видам деятельности.
  • Пропорционально валовой прибыли. Данное правило распределения может использоваться при распределении прочих расходов по видам деятельности. Базой распределения служит сумма выручки от продаж за минусом себестоимости продаж, распределенная по видам деятельности.
  • Пропорционально коэффициентам. Данное правило может использоваться при распределении прочих расходов и распределении выручки и себестоимости продаж по видам деятельности. Если выбрано правило "Пропорционально коэффициентам", то становиться доступной колонка "Коэффициент" в табличной части. Коэффициенты распределения по видам деятельности задаются пользователем вручную.

Правилом распределения - задается фактически база распределения расходов. Существует два способа установки правила распределения расходов, в зависимости от характера расхода.

1. Для распределения расхода выбирается та база распределения, от изменения которой напрямую зависит величина расхода. Пример: заработная плата менеджеров по продажам (определяется как процент от выручки реализованных товаров) должна распределяться пропорционально доходам.

2. Если величина расходов напрямую не зависит от базы распределения, но известно, что расход нужно равномерно распределить по видам деятельности. В этом случае принимается решение - по какому правилу следует выполнить распределение (пропорционально доходам, расходам, валовой прибыли или коэффициентам, задаваемым вручную). Способ распределения выбирается наиболее соответствующий выбранному правилу.

Важно. Изменить правила распределения можно только при добавлении нового способа распределения.

Как добавить новый способ распределения доходов и расходов по направлениям деятельности?

В списке "Способы распределения по направлениям деятельности" нажмите на кнопку "Создать". Будет создана новая карточка, в которой необходимо заполнить информацию о правилах распределения и список направлений деятельности.

Как заполнить список направлений деятельности, на которые будут распределяться доходы и расходы?

Информация о направлениях деятельности заполняется в табличной части "Направления деятельности". В список добавляются те направления деятельности, на которые планируется выполнять распределение прочих расходов, а также выручку и себестоимость продаж.

Если направления деятельности для способа распределения не заданы, то распределение будет осуществляться без отбора – на все направления деятельности торгового предприятия.
