Справочник "Способы распределения по направлениям деятельности"

Справочник "Способы распределения по направлениям деятельности" предназначен для указания способов распределения доходов и расходов по направлениям деятельности.

Информация о применяемом способе распределения заполняется в статьях доходов и статьях расходов, которые распределяются по направлениям деятельности. Распределение доходов и расходов по направлениям деятельности производится регламентным документом "Распределение доходов и расходов".
