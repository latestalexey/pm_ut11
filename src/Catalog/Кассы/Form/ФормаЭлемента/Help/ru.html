Как указать организацию, к которой принадлежит касса?

  • Если информация о кассе вводится из списка, в котором произведен отбор по конкретной организации, то информация об организации заполняется автоматически.
  • Если организация не заполнилась, то ее надо обязательно указать. Каждая касса обязательно должна относиться к конкретной организации.
  • Выбор организации осуществляется в списке выбора справочника "Организации". Идентификация организации может осуществляться по краткому наименованию организации или ИНН.

Важно. Поле "Организация" является обязательным к заполнению, контролируется при записи нового элемента справочника. После записи информации о кассе изменить организацию нельзя.

Как указать валюту, в которой хранятся денежные средства в кассе?

Выбор валюты осуществляется из списка выбора справочника "Валюты". Для быстрого ввода информации о валюте можно задать код валюты или ее наименование.

После записи элемента справочника "Кассы", валюту денежных средств изменить нельзя.

Как установить связь кассы с определенным подразделением предприятия и как эта настройка влияет на учет?

В карточке кассы укажите конкретное подразделение, выбрав его из справочника Структура предприятия. Информация о данном подразделении будет использоваться при выгрузке кассовых документов, оформленных по данной кассе в конфигурацию Бухгалтерия предприятия. В конфигурации "Бухгалтерия предприятия" будет создано соответствующее подразделение: Организация+подразделение, указанные в карточке кассы. Информация об этом подразделении будет автоматически заполнена во всех кассовых документах, которые будут выгружаться в конфигурацию "Бухгалтерия предприятия".

Как формируется наименование кассы?

Наименование кассы может быть заполнено вручную в соответствующем поле. Для кассы предсмотрено автоматическое формирование наименования. При записи информации о кассе, если информация о наименовании не заполнена пользователем, то оно формируется автоматически по формуле: <Наименование организации> (<Наименование валюты денежных средств>). Предусмотрена возможность подстановки наименования в соответствии с указанным шаблоном по кнопке "Выбор из списка".

Как записать введенные данные о кассе?

Для записи введенных данных о кассе нажмите на кнопку "Записать и закрыть". После записи информации о кассе не допускается редактировать данные об организации, к которой относится касса и о валюте кассы.

Как указать, что расход денежных средств по кассе может не контролироваться?

Если конроля расхода денежных средств по конкретной кассе не требуется, то необходимо отключить такую возможность для кассы. Для этого необходимо установить флажок Разрешить выдачу денежных средств без "заявок на оплату". После этого расход денежных средств по кассе можно будет осуществлять без обязательного ввода документа "Заявка на расходование денежных средств".

Как указать список тех касс, перемещение денежных средств между которыми не должно контролироваться?
  • Установите флажок "Разрешить передачу денежных средств без распоряжений на перемещение".
  • Нажмите на гиперссылку "Указать кассы".
  • Укажите список касс, в которые можно будет перемещать денежные средства из указанной кассы без указания утвержденного распоряжения на перемещение денежных средств.
