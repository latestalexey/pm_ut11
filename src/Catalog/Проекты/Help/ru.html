Справочник "Проекты"

Справочник "Проекты" предназначен для ведения списка долгосрочных проектов, которые используются в торговом предприятии. В справочнике хранится список проектов предприятия с указанием темы проекта, плановых и фактических сроков работы по проекту, ответственного по проекту, привлеченных к проекту лиц. Например, можно с помощью этого справочника вести проект по обучению сотрудников или проект работы с аутсорсинговой компанией (аудит, консалтинг). В рамках проекта можно вести список взаимодействий с участниками проекта, оформлять и контролировать задания сотрудникам на различных этапах проекта, хранить файлы документации по проекту.
