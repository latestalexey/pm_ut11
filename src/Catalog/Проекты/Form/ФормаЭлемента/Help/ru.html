Как указать список партнеров, участвующих в проекте?

Список партнеров, участвующих в проекте заполняется в табличной части карточки проекта. Для добавления нового участника проекта используется кнопка "Добавить". Для каждого участника проекта указываются следующие данные.

  • Наименование партнера. Заполняется из справочника "Партнеры".
  • Роль партнера. Указывается та роль, которую выполняет партнер, как участник проекта. Информация о возможных ролях партнера предварительно вводится в справочник "Роли партнеров в сделках и проектах". По умолчанию в этом списке присутствует две записи: клиент и конкурент. При необходимости в этот справочник можно добавить необходимые роли.
  • Контактное лицо. Котактное лицо, через которого можно взаимодействовать с партнером. Заполняется из списка тех контактных лиц, которые определены для партнера.
  • Комментарий. Для каждого партнера можно ввести произвольный текстовый комментарий.

Как ввести информацию об этапах проекта?
  • В панели навигации формы проекта выберите пункт "План проекта".
  • Откроется список тех заданий (этапов проекта), которые должны быть выполнены в рамках данного проекта.
  • Для каждого задания указывается исполнитель, куратор, плановая и фактическая дата выполнения задания, стадия выполнения задания.

Как посмотреть список взаимодействий по проекту?
  • В панели навигации формы проекта выберите пункт "Взаимодействия".
  • Будет показан список всех взаимодействий по данному проекту (телефонные звонки, встречи, электронные письма и т.д.). В списке будут показаны те взаимодействия, в которых проект указан в качестве предмета взаимодействия.

Как спланировать взаимодействия по проекту?
  • В списке "Партнеры и контактные лица" установите курсор на того участника, по которому надо запланировать взаимодействие.
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Запланировать взаимодействие".
  • Будет создана запись о новом запланированном взаимодействии с партнером.

Как создать электронное письмо участнику проекта?
  • В списке "Партнеры и контактные лица" установите курсор на того участника, которому нужно позвонить.
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Написать письмо".
  • Будет создано новое электронное письмо.

При создании электронного письма в рамках проекта автоматически заполняется реквизит "предмет". В качестве получателя письма указывается контактное лицо партнера и его электронный адрес. Если информация о контактном лице не заполнена, то заполняется информация о партнере.

Как создать телефонный звонок участнику проекта?
  • В списке "Партнеры и контактные лица" установите курсор на того участника, которому нужно позвонить.
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Позвонить".
  • Будет создана запись о новом телефонном звонке клиенту .

В качестве абонента будет указано контактное лицо партнера. В поле "Как связаться" будут указана информация о телефонах контактного лица. Если контактное лицо не указано, то в качестве абонента будет указана информация о партнере.

Как запланировать встречу для участников проекта?
  • Нажмите на кнопку "Взаимодействия".
  • В появившемся списке выберите пункт "Запланировать встречу".
  • Будет создана запись о новой встрече.
  • В качестве участников встречи будет заполнен весь список участников, указанный в списке участников проекта. Для всех участников будет указана контактная информация.

Как указать, что работы по проекту завершены?

После завершения работ по проекту выполните следующие действия.

  • В поле "Дата окончания" укажите дату фактического завершения работ по проекту.
  • Установите флаг в поле "Завершен".
  • Нажмите на кнопку "Записать и закрыть".

Создание электронного письма на основании проекта

При создании электронного письма на основании проекта автоматически заполняется реквизит "предмет". В список получателей письма добавляются записи, соответствующие контактной информации партнеров и контактных лиц проекта с типом "e-mail".

Работа с задачами по проекту

Открытие списка задач проекта

Открыть список задач проекта можно из формы проекта. Проектом с этой точки зрения может выступать Сделка или Маркетинговое мероприятие. В списке отображаются только задачи текущего проекта.
