Как создать новый шаблон ценника или этикетки?

Новый шаблон ценника или этикетки может быть создан заново или на основании того шаблона ценники или этикетки, которые предусмотрены в программе в качестве шаблонов по умолчанию.

Для заполнения шаблона ценника или этикетки в соответствии с тем шаблоном, который предусмотрен в качестве шаблона по умолчанию, выполните следующие действия.

  • В новом шаблоне ценника (этикетки) нажмите на кнопку "Шаблон по умолчанию" .
  • Из выпадающего списка выберите нужный пункт "Ценник по умолчанию" ("Этикетка по умолчанию").
  • Будет загружен шаблон ценника (этикетки), который хранится в программе.
  • Укажите наименование шаблона.
  • В поле "Количество по вертикали" укажите то количество ценников (этикеток), которое должно располагаться на листе печати по вертикали.
  • В поле "Количество по горизонтали" укажите то количество ценников (этикеток), которое должно располагаться на листе печати по горизонтали.
  • Нажмите на кнопку "Записать и закрыть".

Шаблон ценника можно отредактировать, добавив в него любое поле из списка доступных полей. При создании или редактировании шаблона используются те стандартные правила работы, которые применяются при работе с табличными документами в 1С:Предприятии.

Важно. Для каждого макета можно настроить свои параметры печати: принтер, размер бумаги, поля отступа и т.д.
