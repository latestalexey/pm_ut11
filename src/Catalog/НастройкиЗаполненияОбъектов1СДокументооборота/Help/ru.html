Справочник содержит настройки, которые позволяют автоматически заполнить реквизиты документа 1С:Документооборота из данных объекта, для которого создается документ.

Например для документа "Счет от поставщика" можно указать:

  • для данного объекта создается входящий документ,
  • поле Организация входящего документа заполняется из поля Организация счета,
  • поле Отправитель входящего документа заполняется из поля Корреспондент счета,
  • поле Вид входящего документа заполняется значением "Счет на оплату" 
  • поле Гриф доступа входящего документа заполняется значением "Общий"
  • поле Регистрационный номер входящего документ заполняется из поля Номер счета,
  • поля Сумма и Валюта входящего документа заполняются из одноименных полей счета,
  • и т.п.


В поле Объект конфигурации указывается, для какого объекта конфигурации-потребителя выполняется настройка. В поле Объект 1С:Документооборота указывается документ какого типа (входящий, исходящий или внутренний) соответствует объекту потребителю. В табличной части выводятся все реквизиты документа 1С:Документооборота указанного типа.

Для каждого реквизита можно задать правило его заполнения. Для этого необходимо выделить строку с нужным реквизитом и нажать кнопку Изменить. Будет открыта форма ввода правила заполнения, в которой можно указать способ заполнения реквизита.

Для очистки правила, заданного для некоторого реквизита, необходимо выделить строку с нужным реквизитом и выбрать в контекстном меню команду Очистить или нажать клавишу Del.
