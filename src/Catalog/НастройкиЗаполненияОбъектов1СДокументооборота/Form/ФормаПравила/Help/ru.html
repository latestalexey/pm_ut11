Правило заполнения реквизита объекта 1С:Документооборота можно задать тремя способами:

  • Из реквизита объекта конфигурации. Данный способ означает, что реквизит объекта 1С:Документооборота будет заполнен из некоторого реквизита объекта-потребителя. При выборе данного способа требуется указать, из какого реквизита объекта-потребителя требуется получить значение и подставить в реквизит объекта 1С:Документооборота.
  • Из указанного значения. Данный способ означает, что реквизит объекта 1С:Документооборота будет заполнен из значения, указанного пользователем. При выборе данного способа требуется указать то значение, которе будет использовано для заполнения реквизита 1С:Документооборота. Можно указать значение только того типа, который имеет реквизит 1С:Документооборота.
  • Из выражения на встроенном языке. Данный способ означает, что реквизит объекта 1С:Документооборота будет заполнен из результата выражения на встроенном языке 1С. При выборе данного способа требуется указать выражение, результат которого будет использован для заполнения реквизита 1С:Документооборота. В выражение могут входить любые операции, в том числе вызов экспортных процедур и функций. Результат выражения должен быть присвоен переменной с именем "Результат". Переменная с именем "Источник" содержит ссылку на объект-потребитель, через эту переменную можно обращаться к реквизитам объекта-потребителя либо использовать как параметр при вызове функций.
