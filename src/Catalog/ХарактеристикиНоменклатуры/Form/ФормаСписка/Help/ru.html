Как ввести список характеристик для позиции номенклатуры?

Для позиции номенклатуры можно ввести список тех индивидуальных характеристик, которые назначаются для конкретной позиции номенклатуры. Для ввода дополнительных характеристик используется справочник Характеристики, который вызывается из панели навигации формы позиции номенклатуры.

Важно. Список характеристик для конкретной позиции номенклатуры вводится только в том случае, если для вида номенклатуры, к которому принадлежит данная позиция, предусмотрено ведение индивидуального списка характеристик. Если для вида номенклатуры используется общий список характеристик, то ввод новых характеристик в этом случае производится в справочнике Виды номенклатуры.
