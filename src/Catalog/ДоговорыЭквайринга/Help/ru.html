Справочник "Договоры эквайринга"

Справочник "Договоры эквайринга" предназначен для определения тех условий, в соответствии с которым будут обслуживаться банком операции по оформлению оплаты клиентов с помощью платежных карт. В договорах эквайринга также регистрируется список тех эквайринговых терминалов, с помощью которых будут обслуживаться операции с платежными картами.

Справочник "Договоры эквайринга" используется при оформлении операций по оплате клиентом платежными картами (эквайринговые операции). Предусмотрена возможность автоматической загрузки отчетов банка по операциям эквайринга, в соответствии с заключенным с ним договором.

Информацию о всех операциях с платежными картами, произведенными по договору эквайринга можно посмотреть в отчете "Контроль операций эквайринга".
