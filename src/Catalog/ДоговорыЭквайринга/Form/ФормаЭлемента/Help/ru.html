Работа с договором эквайринга

С кем должен быть заключен договор эквайринга для обслуживания расчетов банковскими картами?

Организация может заключить договоры на обслуживание с одним или нескольким банками-эквайрерами. Договор эквайринга является правовой основой для осуществления схемы расчетов банковскими картами между банком и организацией, которая принимает платежи от покупателей при помощи карт.

Банк–эквайрер самостоятельно устанавливает порядок проведения авторизации операции по оплате товаров и услуг в торговых точках с использованием банковских карт, который фиксируется в договоре эквайринга.

Справочник "Договоры эквайринга" используется для идентификации платежа по эквайрингу. Договор эквайринга включает в себя следующую информацию:

  • Эквайрер – банк, с которым заключен договор эквайринга.
  • Организация - организация, от имени которой заключается договор с банком;
  • Контрагент - контрагент банка-эквайрера, от имени которого заключается договор с организацией;
  • Банковский счет организации для идентификации счета, на который перечисляются денежные средства по эквайрингу;
  • Список видов платежных карт. Перечень видов платежных карт, обслуживаемых по договору эквайринга.

Как указать банк-эквайрер?

Банк-эквайрер указывается в поле "Эквайрер". В форме нового договора эквайринга необходимо выбрать банк-эквайрер, с которым заключен договор о приеме в оплату товаров и услуг банковских карт. По этому договору банк устанавливает оборудование для работы с картами, обеспечивает обучение персонала работе с этими системами, а также их материально-техническое обслуживание.

Предварительно информация о банке-эквайрере регистрируется в справочнике "Партнеры". Для банка-эквайрера устанавливается вид бизнес-отношений - "Прочие отношения".

Эквайрер выбирается из списка справочника "Партнеры" с фильтром по партнерам, для которых установлен вид бизнес-отношений "Прочие отношения".

Для быстрого выбора банка-эквайрера в поле ввода "Эквайрер" можно набрать краткое наименование банка. При выборе банка-эквайрера автоматически определяется контрагент (юридическое лицо). Обычно для банка существует одно строго определенное юридическое лицо, поэтому информация в этом поле заполняется автоматически.

Можно начать заполнение с поля "Контрагент". В поле ввода введите ИНН юридического лица банка. Будет автоматически заполнена информация об юридическом лице банка ("Контрагент") и информация о самом банке ("Эквайрер").

Как указать банковский счет, на который будут зачисляться денежные средства по операциям с использованием платежных карт (эквайринговых операций)?

Информация о расчетном счете , который будет использоваться для зачисления денежных средств по эквайринговым операциям заполняется в поле "Банковский счет". Для выбора доступны все зарегисрированные банковские счета по всем организациям. Для быстрого выбора можно в поле ввода указать номер банковского счета.

Как указать список тех платежных карт, оплата по которым будет приниматься в рамках оформляемого договора-эквайринга?

Список платежных карт указывается в табличной части "Виды обслуживаемых платежных карт". В этом списке указываются все те виды платежных карт, которые будут обслуживаться банком в соответствии с заключенным с ним договором эквайринга.

Для ввода нового вида платежной карты выполните следующие действия.

  • Нажмите на кнопку "Добавить" в командной панели списка.
  • В появившейся строке введите текстовое наименование платежной карты, например "VISA".
  • Аналогичным образом введите информацию о всех остальных видах платежных карт.

Информация о видах платежных карт используется при выборе платежной карты при регистрации оплат с помощью платежной карты.

Работа со списком эквайринговых терминалов

Как посмотреть какие эквайринговые терминалы могут использоваться для оформления платежей по договору эквайринга?

В карточке договора эквайринга в панели навигации выберите пункт "Эквайринговые терминалы".

Будет показан список эквайринговых терминалов. В списке отображаются эквайринговые терминалы, которые в рамках договора с банком-эквайрером регистрируются в организации, осуществляющей прием оплаты платежными картами.

В форме списка эквайринговых терминалов выводится следующая информация:

  • Наименование - наименование эквайрингового терминала
  • Код - код эквайрингового терминала
  • Касса - касса, к которой подключен эквайринговый терминал
  • Договор эквайринга - договор, в соответствии с условиями которого работает эквайринговый терминал.
  • Организация - организация, с которой заключен договор эквайринга.
