Как указать для какой розничной торговой точки будет использоваться данная касса: автоматизированной или неавтоматизированной торговой точки?

Эта информация определяется тем типом кассы, который выбран в карточке кассы ККМ.

  • Если касса используется в неавтоматизированной торговой точке, то в поле "Тип кассы" выберите - "Автономная ККМ".
  • Если касса используется в автоматизированной торговой точке (торговом зале, магазине), то в поле "Тип кассы" выберите "Фискальный регистратор" или "ККМ Offline".

Важно. Режим "Фискальный регистратор" устанавливается для тех касс, которые для оформления продаж используют данную программу. Режим ККМ Offline используется в том случае, если кассы работают в автономном режиме с использованием внешних программ (например, АТОЛ: Рабочее место кассира).

Как указать организацию, от имени которой будут оформляться розничные продажи в розничной торговой точке (розничном магазине)?

Информация об организации заполняется в одноименном поле в карточке "Кассы ККМ". Розничные продажи могут осуществляться от имени любой организации. Для каждой кассы ККМ может быть определена своя организация, от имени которой будут осуществляться продажи товаров в розницу.

Как указать склад, с которого будут регистрироваться продажи по данной ККМ?

Информация о складе выбирается в поле "Склад". В качестве склада может быть указан любой склад, для которого указан тип склада "Розничный магазин".

Важно. Одна касса ККМ может обслуживать продажи на одном складе. В качестве склада используется складская территория (розничный магазин, удаленная торговая точка) без разделения по конкретным секциям (отделам). В то же время для одной складской территории (розничного магазина, удаленной торговой точки) может быть задано несколько различных касс.

Как подключить торговое оборудование к конкретной кассе ККМ?

Торговое оборудование подключается только к той кассе, которая имеет тип - Фискальный регистратор или ККМ Offline. В карточке кассы ККМ в поле "Внешнее оборудование" из выпадающего списка выберите модель, которая используется для данной кассы.

Примечание. Предварительно информация об используемых моделях торгового оборудования и о параметрах его подключения к рабочим местам пользователей регистрируется с помощью обработки "Подключение и настройка торгового оборудования".

В какой валюте могут осуществляться розничные продажи через кассу ККМ?

Обычно розничные продажи через кассу ККМ оформляются в рублях (валюте регламентированного учета, которая указана в настройках параметров учета). При вводе информации о новой кассе эта валюта заполняется в качестве валюты по умолчанию.

Программа допускает оформление розничных продаж через кассу ККМ в любой валюте, зарегистрированной в справочнике "Валюты", если у торгового предприятия есть разрешение на оформление наличной продажи через кассу ККМ в валюте, отличной от валюты регламентированного учета.

Для каких целей используется и как формируется наименование кассы ККМ?

Наименование кассы ККМ используется для быстрого заполнения и поиска информации о кассе ККМ при оформлении документов. В поле "Наименование" пользователь может ввести любое удобное и понятное ему название кассы ККМ.

Предусмотрена также возможность автоматического наименования кассы ККМ в соответствии с заполненными параметрами кассы. Вариант формирования наименования можно выбрать из выпадающего списка: Тип кассы (Наименование склада).
