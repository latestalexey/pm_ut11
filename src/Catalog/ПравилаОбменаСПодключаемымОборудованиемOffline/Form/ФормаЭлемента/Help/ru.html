Для каких целей служит правило обмена касс?

По правилам обмена происходит загрузка товара на кассы работающие в режиме Offline. Для каждой кассы должно существовать минимум одно правило. Одно и тоже правило может быть назначено нескольким кассам.

Что необходимо заполнить в карточке правила обмена для кассы?

В карточке правила обмена необходимо задать наименование правила, в поле "Тип подключаемого оборудования" выбрать значение "ККМ Offline", обязательно указать склад. Для товаров выгружаемых в кассу программа автоматически присваивает код, а в правиле можно ограничить список товаров, указав "Максимальный код". В правиле можно установить отбор номенклатуры, например, можно загружать только те позиции номенклатуры, которые имеются в наличии на указанном складе (флажок "Только имеющиеся на складе").

Как просмотреть список товаров участвующих в обмене с кассой?

По гиперссылке "Показать список товаров для редактирования" можно открыть список товаров участвующих в обмене с ККМ (в соответствии с установленным фильтром). В списке можно отредактировать автоматически назначенный код товару (под которым товар будет загружен в кассу). По команде "Администрирование" – "Удалить свободные коды в конце списка" из конца списка будут удалены строки с пустой номенклатурой. При заполнении списка производится проверка наличия у товара штрих-кода и значения цены. Значение цены проверяется по тому типу цен, который указан в карточке выбранного склада. Если для отобранных товаров не установлены цены или штрих-код, то информация об этом отображается в соответствующих колонках списка. Если в правиле задано ограничение списка по максимальному коду, то товары с кодом превышающим максимальный участвовать в обмене не будут и будут выделены в списке красным цветом. По команде "Обновить" список товаров участвующих в обмене с ККМ будет записан.

Для каких целей служит правило обмена для весов?

По правилам обмена происходит загрузка товара в весы. Для каждых весов должно существовать минимум одно правило. Одно и то же правило может быть назначено нескольким весам.

Что необходимо заполнить в карточке правил для весов?

В карточке правила обмена необходимо задать наименование правила, в поле "Тип подключаемого оборудования" выбрать значение "Весы с печатью этикеток", обязательно указать склад. Для товаров выгружаемых в весы программа автоматически присваивает код (PLU), а в правиле можно ограничить список товаров, указав "Максимальный код". Так же необходимо указать "Префикс весового товара". Префикс устанавливается в качестве второй цифры штрих-кода (слева) весового товара. Первая цифра штрих-кода весового товара всегда двойка. Таким образом, по первым двум цифрам штрих-кода касса может различать весовой и штучный товар. О значении, которое должно быть установлено в поле "Префикс весового товара" более подробно см. документацию к кассе или программному обеспечению, установленному на кассу. Так же в правиле обмена необходимо указать единицу измерения веса (выбрать единицу измерения с заполненным полем "Тип единицы измерения" значением "Весовая"). В загрузке весов будут принимать участие товары только с такой же единицей измерения.

Как просмотреть список товаров участвующих в обмене с весами?

По гиперссылке "Показать список товаров для редактирования" можно открыть список товаров участвующих в обмене с весами (в соответствии с установленным фильтром). В списке можно отредактировать автоматически назначенный код товару (под которым товар будет загружен в весы). По команде "Администрирование" – "Удалить свободные коды в конце списка" из конца списка будут удалены строки с пустой номенклатурой. При заполнении списка производится проверка наличия у товара цены. Значение цены проверяется по тому типу цен, который указан в карточке выбранного склада. Штрих-код для весового товара будет сформирован в процессе выгрузки автоматически и предварительно формировать его не требуется. Если в правиле задано ограничение списка по максимальному коду, то товары с кодом превышающим максимальный участвовать в обмене не будут и будут выделены в списке красным цветом. По команде "Обновить" список товаров участвующих в обмене с весами будет записан.
