Как настроить учет НДС для конкретной организации?

Настройка выполняется в форме "Настройка учета НДС" (форма открывается соответствующей командой панели навигации, в разделе "Регламентированный учет"). В форме содержится список организаций (настроек учета НДС для организаций) учет по которым ведется в информационной базе. Открыв форму настройки для конкретной организации, можно приступить к настройке учета НДС.

Как создать запись о новой организации?
  • В списке организаций нажмите на кнопку "Создать".
  • Будет создана новая запись об организации, в которой необходимо заполнить все необходимые данные.

Важно. В списке организаций может присутствовать предопределенная организация "Управленческая организация". Данная организация используется для раздельного отражения операций по регламентированному и управленческому учету. От имени этой организации оформляются те операции, которые должны быть отражены только в управленческом учете. Управленческая организация появляется в списке, если в разделе "Администрирование/Финансы" включена функциональная опция "Раздельные операции закупки и продажи для упр. и регл. учета."

Какие данные необходимо ввести по новой организации?

При вводе новой организации следует указать следующие данные.

  • Наименование - краткое наименование организации, по которому будет удобно находить организацию в списках.
  • Сокращенное, Полное - сокращенное и полное официальное наименование, которое будет использоваться при печати документов.
  • Юр/Физ.лицо - указывается кем является организация с юридической точки зрения: юридическим или физическим лицом, резидентом или нерезидентом.
  • Префикс - используется для разделения нумерации документов по разным организациям В начале номера документа добавляется указанный префикс.
  • ИНН, КПП, Код по ОКАТО, Код по ОКПО, ОГРН - информация о кодах организации в соответствии с ее учредительными документами.

На закладке "Адреса и телефоны" вводится информация о контактной информации организации (адреса, телефоны, факс). Для указания адресов можно использовать адресный классификатор.

Важно. По умолчанию у каждой организации есть семь предопределенных видов контактной информации: Email, Почтовый адрес, Другая информация, Телефон, Факс, Фактический адрес, Юридический адрес. При необходимости список видов контактной информации можно расширить.
