Справочник "Организации"

Справочник "Организации" предназначен для ведения списка организаций, входящих в состав предприятия и хранения постоянных сведений о них. Для каждой организации заполняется необходимая информация в соответствии с ее учредительными документами. Организация может быть юридическим, физическим лицом или нерезидентом.

Для организации определяется список ответственных лиц: руководитель организации, главный бухгалтер. Информация об ответственных лицах является периодической. При смене руководителя или главного бухгалтера можно зарегистрировать информацию о новых ответственных лицах организации.

На закладке "Адреса и телефоны" вводится контактная информация организации. Для организации обязательно указание юридического и физического адреса. Эта информация, так же как и информация об ответственных лицах используется при заполнении печатных форм документов. Пользователь может добавить любые дополнительные виды контактной информации в справочник "Организации". Эта информация автоматически добавится в форму "Организации" на закладке "Адреса и телефоны".

Для каждой организации может быть определено любое количество банковских счетов для оформления безналичных оплат и любое количество касс для оформления оплаты наличными.

Для организации должна быть определена та система налогообложения, которая применяется в организации. В части розничных продаж организация может быть переведена на применение единого налога на вмененный доход (ЕНВД) .

В программе можно настроить структуру "Интеркампани", то есть определить товары какой организации имеет право продавать другая организация. В программе предусмотрены различные операции по оформлению передачи товаров между организациями: перемещение товаров между организациями, перемещение денежных средств, передача товаров на комиссию другой организации.
