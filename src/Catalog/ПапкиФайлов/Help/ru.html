Справочник "Папки"

Существует возможность хранить файлы без привязки к конкретному объекту. Для этих целей в системе используется список Файлы. Работа со списком организована аналогично работе в Проводнике Windows. Левая часть списка содержит иерархическую структуру папок. Можно создать необходимое количество папок и хранить в них файлы, не привязанные к конкретным объектам системы.
