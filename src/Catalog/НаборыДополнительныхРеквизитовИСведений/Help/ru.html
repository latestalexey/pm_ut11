Справочник "Наборы дополнительных реквизитов и сведений"

Справочник предназначен для хранения списка тех наборов дополнительных свойств, которые могут быть назначены объектам ( различным справочникам и документам) в программе.

В качестве дополнительных свойств в набор свойств могут быть добавлены как дополнительные реквизиты, так и дополнительные сведения. Дополнительные реквизиты добавляются непосредственно в форму объекта, а для дополнительных сведений предназначена отдельная диалоговая форма, которая вызывается при вызове действия "Дополнительные сведения" в форме объекта.

Наборы свойств можно условно разделить на два типа.

  • Предопределенные наборы свойств. Предопределенные наборы свойств задаются на этапе конфигурирования, назначаются справочнику или документу. Пользователь не может изменить название этих наборов свойств и их принадлежность определенному справочнику или документу, но может добавить произвольное количество дополнительных сведений или реквизитов. Примером такого набора свойств может служить набор свойств "Документ.Заказ клиента". Данный набор свойств предназначен для конкретного документа "Заказ клиента". В этот набор могут быть добавлены дополнительные реквизиты, например, канал сбыта, которые появятся в форме заказа клиента.
  • Создаваемые наборы свойств. В процессе работы наборы свойств могут быть созданы автоматически. Наборы свойств создаются автоматически при создании пользователем нового вида номенклатуры или при вводе новой характеристики номенклатуры. Эти наборы свойств автоматически записываются в соответствующие предопределенные группы "Справочник.Номенклатура" и "Справочник.Характеристика номенклатуры". Для каждого вида номенклатуры и для каждой характеристики номенклатуры можно создать свой список дополнительных реквизитов и сведений, который будет показываться только для тех товаров, которые принадлежат данному виду номенклатуры или которые имеют указанную характеристику.
