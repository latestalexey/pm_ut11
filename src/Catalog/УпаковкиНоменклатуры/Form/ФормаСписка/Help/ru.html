Как ввести список упаковок для позиции номенклатуры?

Для позиции номенклатуры можно ввести список тех индивидуальных упаковок, которые назначаются для конкретной позиции номенклатуры. Для ввода упаковок используется справочник Упаковки, который показывается в панели навигации формы. Информация в этом справочнике заполняется пользователем только в том случае, если для вида номенклатуры, к которому принадлежит данная позиция, предусмотрено ведение индивидуального списка упаковок. Если для вида номенклатуры используется общий список упаковок, то он будет показан для просмотра, без возможности его дополнения. Ввод новых упаковок в этом случае производится в справочнике Виды номенклатуры.

Важно. Список упаковок для конкретной позиции номенклатуры вводится только в том случае если для вида номенклатуры, к которому принадлежит данная позиция, предусмотрено ведение индивидуального списка упаковок. Если для вида номенклатуры используется общий список упаковок, то ввод новых упаковок в этом случае производится в справочнике Виды номенклатуры.
