Какую информацию необходимо указать для бизнес-региона?
  • Необходимо указать Наименование бизнес-региона.
  • Если указанный бизнес-регион входит в другой более крупный бизнес-регион, то укажите название того бизнес-региона, в который входит бизнес-регион в поле "Родитель".
  • Возможно указание Основного менеджера, который будет ответственен за работу с клиентами данного региона. Указанное значение будет заполняться как значение по умолчанию при заполнении информации об основном менеджере в карточке нового партнера.
