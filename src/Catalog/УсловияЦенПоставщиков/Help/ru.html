Справочник "Условия цен поставщиков"

Справочник "Условия цен поставщиков" предназначен для хранения списка тех условий , при соблюдении которых поставщик предлагает те или иные цены на товар. Условия цен поставщиков используются в документе "Регистрация цен поставщиков".

Важно. Информация об условиях цен поставщиков регистрируется только в том случае, если в соглашении с поставщиком, по которому оформляется документ "Регистрация цен поставщиков" указано, что цены регистрируются при соблюдении определенных условий (в соглашении с поставщиком установлен флаг "Регистрировать цены по условиям").

Информация об условиях цен заполняется по каждому соглашению с поставщиком. При заполнении иформации об условии цены поставщика можно использовать дополнительные свойства. В карточку условия цены можно добавить список предопределенных дополнительных ревизитов (срок поставки, минимальный объем партии и т.д.). Для каждого поставщика можно ввести свои значения данных реквизитов и затем производить сравнение конкретных условий поставки товаров и предлагаемых цен. Для сравнения цен и условий, при соблюдении которых поставщик предлагает цены, используется отчет "Условия соглашений с поставщиками".
