Как загрузить версию мобильного приложения из макета?

Для загрузки выполните следующие действия.

  • Нажмите кнопку "Выгрузка/загрузка приложений" в панели управления списком.
  • Выберите пункт меню "Загрузить приложение из макета".
  • В появившемся списке предопределенных макетов, которые поставляются с конфигурацией выберите предлагаемый макет мобильного приложения "Торговый представитель".
  • В справочнике будет автоматически создана группа версий приложения "Торговый представитель" и из макета загружена конкретная версия мобильного приложения.

В дальнейшем версии мобильных приложений могут обновляться. В программе будут храниться все версии мобильных приложений, которые загружаются пользователем.

Как выгрузить версию мобильного приложения в файл для последующего использования в другой информационной базе?
  • Установите курсор на версию выбранного мобильного приложения в списке.
  • Нажмите кнопку "Выгрузка/загрузка приложений" в панели управления списком.
  • Выберите пункт меню "Выгрузить приложение в файл".
  • Укажите каталог, в который необходимо сохранить информацию о о мобильном приложении.
  • Имя файла формируется автоматически в соответствии с именем версии мобильного приложения.

Как загрузить информацию о версии мобильного приложения из файла?
  • Нажмите кнопку "Выгрузка/загрузка приложений" в панели управления списком.
  • Выберите пункт меню "Загрузить приложение из файла".
  • Выберите файл, из которого необходимо загрузить приложение.
  • Будет загружена та версия мобильного приложения , которая хранится в файле.

При загрузке версии мобильного приложения проверяется наличие данной версии мобильного приложения в списке доступных версий мобильных приложений. Если данная версия приложения уже имеется в списке, то выдается соответствующее предупреждение.
