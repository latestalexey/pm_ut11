Справочник "Задачи проектов"

Справочник "Задачи проектов" предназначен для регистрации этапов проекта и процессов, используемых в целях планирования и контроля деятельности по проектам.

Важно. Задачи проекта заполняются в плане проекта, который может быть зарегистрирован при оформлении сделки с клиентом. План проект обычно оформляется по долгосрочным сделкам. Задачи проекта могут быть также оформлены при составлении план проекта по маркетинговым мероприятиям.
