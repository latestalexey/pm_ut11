Справочник "Варианты анализа целевых показателей"

В справочнике хранится информация о вариантах анализа, которые используются для анализа целевых показателей. Для каждой цели используется свой список вариантов анализа целевых показателей.

Варианты анализа построены с использованием схем компоновки данных. Пользователь может использовать готовые варианты схем компоновки данных или разрабатывать свои схемы компоновки данных. Используя данный справочник пользователь, имеющий соответствующие права, может добавлять не только новые варианты анализа целевых показателей, но и вводить информацию о новых целях в структуру целей. Для дополнительной детализации данных могут быть также использованы различные варианты стандартных отчетов конфигурации.
