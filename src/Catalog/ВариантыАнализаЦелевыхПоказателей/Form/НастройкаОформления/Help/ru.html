Для чего нужны настройки оформления варианта анализа?

Настройки группы "Цвета линий и точек диаграмм" влияют на цветовое оформление ключевых элементов диаграмм.

Настройки группы "Цвета покомпонентного сравнения":

  • Только цвет основной серии - при оформлении гистограмм сравнения будет использоваться цвет, заданный в "Основной серии". Если настройка отключена, то будет использоваться оформление диаграмм с умолчаниями платформы.
  • Градиент в зависимости от значений - при оформлении гистограммы светлость цвета будет изменяться в зависимости от значений. Чем меньше значение, тем светлее элемент гистограммы.
  • Выделять максимальное значение - максимальное значение будет выделено отличающимся от основного цветом.


Дополнительные настройки:

  • Отображать легенду - при выводе диаграммы будет выводиться легенда.
  • Маркировать точки прогноза - точки прогноза будут маркированы квадратным маркером.
