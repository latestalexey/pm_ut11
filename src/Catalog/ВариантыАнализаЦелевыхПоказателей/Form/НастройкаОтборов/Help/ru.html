Для чего нужны отборы варианта анализа?

Отборы варианта анализа нужны для создания уточненных по различным признакам вариантов анализа.

Например, требуется показывать Выручку по менеджеру отдельным менеджерам, таким образом чтобы каждый менеджер видел только свою выручку.
В указанном случае следует создать варианты анализа выручки с отбором по отдельным менеджерам и соответственно предоставить доступ менеджерам к "их" вариантам анализа.

Какие отборы варианта анализа доступны для настройки?

Для настройки доступны только те параметры и отборы, которые включены в пользовательские настройки шаблона расчета целевого показателя (элемента структуры целей).
