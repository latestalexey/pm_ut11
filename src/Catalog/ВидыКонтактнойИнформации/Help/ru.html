Справочник "Виды контактной информации"

Справочник служит для классификации контактной информации различных объектов системы - например, партнеров, контрагентов, сотрудников и т.д.

Для каждого вида контактной информации определен тип контактной информации, который выбирается из фиксированного списка, и тип объекта, для которого предназначен данный вид контактной информации.

Пример. Для контактной информации типа "Адрес", предназначенной для объектов справочника "Физические лица", можно определить виды контактной информации "Домашний адрес" или "Рабочий адрес", а для объектов справочника "Контрагенты" - "Юридический адрес" или "Почтовый адрес".

В списке справочника виды контактной информации представлены в виде групп, каждая из которых является тем объектом, для которого может быть определена контактная информация: "Организации", "Пользователи", "Физические лица", "Партнеры", Контрагенты", "Контактные лица партнеров".

Для каждой группы определен список предопределенных видов контактной информации, которые отмечены в списке определенным знаком. При необходимости в любую из групп пользователь может добавить любой вид дополнительной контактной информации. После добавления новый вид контактной информации автоматически появится в форме соответствующего объекта.
