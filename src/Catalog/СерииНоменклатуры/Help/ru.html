Справочник "Серии номенклатуры"

В справочнике хранятся сведения о сериях товаров. Серии товаров представляют собой разрез складского учета товаров. Для каждого вида номенклатуры может быть настроена своя политика учета серий: для одних видов товаров необходимо регистрировать информацию о сроке хранения, а для других видов номенклатуры регистрировать серийный номер для каждого экземпляра товара.

Важно. Информация о сериях номенклатуры заполняется в справочнике в момент регистрации серий номенклатуры в документах в соответствии с применяемой политикой учета серий для вида номенклатуры, к которому принадлежит конкретная позиция. Не рекомендуется добавлять информацию о сериях в справочник вручную.
