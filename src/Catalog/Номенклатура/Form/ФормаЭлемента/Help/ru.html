Какую основную информацию необходимо ввести для новой номенклатурной позиции.

В карточку новой позиции номенклатуры вводится следующая информация.

  • Артикул. Артикул позиции номенклатуры.
  • Рабочее наименование. Название товара или услуги, которое используется для быстрой визуальной идентификации номенклатуры в процессе работы с программой.
  • Наименование для печати. Наименование товара и услуги, которое будет печататься во всех документах.
  • Вид номенклатуры. Указывается вид, к которому следует отнести данную позицию номенклатуры. При выборе вида номенклатуры справочно указываются те параметры, которые характеризуют данный вид номенклатуры: тип номенклатуры (товар или услуга), использование характеристик, использование упаковок.
  • Единица хранения. Единица измерения, в которой ведется учет складских остатков товаров и указывается количество номенклатуры в одной упаковке. Для заполнения единицы измерения используется справочник Классификатор единиц измерения.
  • Качество. Можно указать градацию качества вводимой позиции. Для номенклатуры доступно три значения качества: "Новый", "Ограниченно годен", "Не годен". При вводе новой номенклатурной позиции поле "Качество" по умолчанию заполняется значением "Новый". Для каждой степени порчи товара нужно ввести новый элемент справочника "Номенклатура". Программа запоминает связь качественного и бракованного товара. Установить связь можно в списке "Товары другого качества". Перейти к списку можно с помощью команды панели навигации "Подобные товары", далее закладка "Товары другого качества".
  • Вести учет по ГТД. Признак того, что для элемента номенклатуры необходимо вести учет по номерам ГТД.
  • Группа номенклатуры. Группа, в которую входит данная позиция номенклатуры в соответствии с классификацией (иерархией) товаров в справочнике "Номенклатура".
  • Ценовая группа. Ценовая группа, в которую входит позиция номенклатуры, в соответствии с классификацией товаров по ценовым характеристикам.

Как сформировать наименование номенклатурной позиции/характеристики если задан шаблон наименования номенклатуры/характеристики?

Для каждого шаблона в карточке вида номенклатуры можно настроить обязательность его использования с помощью флажков "Запретить редактирование":

  • Если флажок установлен, то наименование в карточке номенклатуры или характеристики становится недоступным для редактирования и переформировывается при каждой записи номенклатуры или характеристики.
  • Если флажок не установлен, то заполнить наименование по шаблону можно с помощью команды "Заполнить" (пиктограмма рядом с полем наименования). При этом если наименование не заполнено, то оно формируется при записи номенклатуры автоматически.

На что влияет установка признака подакцизного товара?

Признак подакцизного товара используется при оформлении розничных продаж, использующих систему налогообложения ЕНВД. Подакцизные товары являются объектами налогообложения в любом случае, независимо от того, применяется ли система налогообложения ЕНВД при розничной торговле. Список подакцизных товаров регламентирован в соответствии со статьей 181 НК РФ.

  • При оформлении продаж с розничного склада, применяющего систему ЕНВД, обобщенному покупателю (документы Чек ККМ, Отчет о розничных продажах) для подакцизных товаров устанавливается ставка НДС, указанная в карточке номенклатуры.
  • Подакцизные товары запрещено продавать в рамках соглашений (документов) с видом налогообложения Продажа облагается ЕНВД.

Как ввести дополнительные сведения для позиции номенклатуры?

Дополнительные сведения по номенклатурной позиции вводятся в отдельном диалоговом окне, которое открывается при нажатии на соответствующую пиктограмму или при выборе действия "Дополнительные сведения". В списке показываются все те дополнительные сведения, которые были определены в наборе свойств для того вида номенклатуры, к которому принадлежит данная позиция номенклатуры. Пользователь должен ввести значения этих дополнительных сведений. В качестве значений могут использоваться различные виды значений: число, дата, текстовая строка, различные справочники. Это определяется тем какой тип значения задан для дополнительного свойства в наборе свойств для данного вида номенклатуры.

Если планируется использовать конкретную позицию номенклатуры в качестве комплекта, то где можно указать комплектующие (спецификацию комплекта)?

Если планируется использовать конкретную позицию номенклатуры в качестве комплекта, то спецификацию комплекта можно указать в списке "Варианты комплектации номенклатуры". Перейти в список можно с помощью команды "Варианты комплектации", в панели навигации карточки номенклатуры.

Как перейти к списку сегментов, в которые входит Номенклатура?

Для открытия списка сегментов перейдите по гиперссылке "Сегменты номенклатуры", расположенной в панели навигации. На закладке "Формировать вручную" есть возможность включения или исключения номенклатуры из сегментов с типом формирования "Формировать вручную".

Как посмотреть место размещения товара на складе?

Информацию о том, где находится конкретный товар можно посмотреть, используя в навигационной панели пункт "Размещение в ячейках" . Информация о размещении товара на конкретном складе, в конкретном помещении и конкретной ячейке задается в момент приема товара на склад кладовщиком.

Как посмотреть информацию по ценам, относящимся к позиции номенклатуры?

Из формы номенклатуры можно посмотреть дополнительную информацию о ценах, которые назначены данной позиции номенклатуры. Для просмотра используется пункт навигационной панели "Цены". Дополнительная информация показана на закладках.

  • Прайс-лист. Здесь можно увидеть цены для текущей позиции номенклатуры в виде настраиваемого отчета.
  • Цены номенклатуры. Показывается список цен, зарегистрированных для данной позиции номенклатуры, документом Установка цен номенклатуры.
  • Цены номенклатуры поставщиков. Список цен поставщиков, зарегистрированных для данной позиции номенклатуры. Информация об этих ценах регистрируется при оформлении документа Установка цен номенклатуры партнера.

Где можно посмотреть и отредактировать параметры управления запасами, относящиеся к данной номенклатуре?

Из формы номенклатуры можно перейти к информации о параметрах управления запасами для данной номенклатуры. Для этого в навигационной панели выберите пункт "Параметры управления запасами". Далее выберите одну из закладок.

  • Методы управления запасами
  • Товарные ограничения
  • Транспортные ограничения
  • Настройка контроля остатков
  • Настройка уровня обслуживания

Где можно посмотреть, куда отнесена по ABC/XYZ классификации данная номенклатура?

Из формы номенклатуры можно перейти к информации об ABC/XYZ классификации. Для этого в навигационной панели выберите пункт "ABC/XYZ классификация". Группа классификации товара определяется для каждого склада (по анализу продаж с конкретного склада) и обобщенно по всем складам.
