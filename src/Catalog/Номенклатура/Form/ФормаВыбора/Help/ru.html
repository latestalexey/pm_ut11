Номенклатура, характеристики

Что можно сделать в форме?


В форме возможно:

  • Подобрать номенклатуру в документ, используя двойной щелчок мыши по нужной позиции списка (или с помощью команды "Выбрать").
  • Осуществить полнотекстовый поиск номенклатуры.
  • Создать новую номенклатурную позицию (с помощью команды "Создать").
  • Установить в списке отбор с помощью фильтра по иерархии или по виду и свойствам.
  • Выполнить ряд других стандартных команд.
