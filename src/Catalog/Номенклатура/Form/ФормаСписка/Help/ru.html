Какая информация показывается в списке позиций номенклатуры?

В справочнике Номенклатура показывается общий список всех позиций номенклатуры. К ним относятся как товары, так и услуги.
Список можно отфильтровать по виду и свойствам номенклатуры, по иерархии и по сегментам.
В списке можно воспользоваться полнотекстовым поиском.
Список позиций номенклатуры может быть представлен в виде списка, в виде иерархического списка (с указанием групп номенклатуры), а также в виде дерева. Порядок представления данных в списке определяется установленным режимом просмотра (Все действия - Режим просмотра).

Как ввести информацию о новой позиции номенклатуры?

  • В списке позиций номенклатуры нажмите на кнопку "Создать".
  • Будет открыт помощник создания номенклатуры.
  • После выбора вида номенклатуры будет определен список тех полей, которые обязательны для заполнения.
  • Следуя указаниям помощника необходимо зарегистрировать данные о номенклатурной позиции.
  • При работе помощника номенклатуры проверяется уникальность созданной номенклатурной позиции.
  • По завершению работы помощника будет создана новая карточка позиции номенклатуры.
  • После записи информации о номенклатурной позиции можно вводить дополнительные данные: характеристики, упаковки, штрихкоды и т.д.

Важно. В программе предусмотрена возможность создания номенклатурной позиции без использования помощника. При таком создании уникальность реквизитов созданной номенклатуры не проверяется. Такая возможность доступна только тем пользователям, которые имеют право на создание номенклатуры без контроля уникальности. Для этих пользователей в списке доступна кнопка "Добавить элемент".

Как из формы списка изменить реквизиты сразу нескольких позиций номенклатуры?
  • Выделите позиции номенклатуры, реквизиты которых нужно изменить.
  • Выберите команду "Изменить выделенные" (меню "Все действия") - откроется форма "Групповое изменение объектов".
  • Измените реквизиты выделенных объектов, следуя инструкциям.
