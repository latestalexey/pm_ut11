Редактирование, открытие и выбор вариантов отчетов.

Перемещаясь по списку "Раздел, Группа" список "Варианты отчетов" автоматически фильтруется по выбранному разделу или группе.
Варианты отчетов можно перетаскивать или добавлять (с зажатой клавишей Ctrl) в другие разделы или группы.

Реквизиты:

  • Наименование
    		•  Представление варианта отчета в списках.
  • Автор
    		•  Автор варианта отчета.
  • Имя отчета
    		•  Имя, под которым отчет зарегистрирован в системе.
  • Отчет
    		•  Представление отчета.
  • Тип отчета
    		•  Внутренний, Дополнительный или Внешний.
    Варианты внешних отчетов не могут размещаться в панели отчетов.
  • Ключ варианта
    		•  Ключ варианта отчета.
  • Пользовательский
    		•  Да - Варианты отчетов, которые были сохранены сохранен пользователями в режиме Предприятия,
    Нет - Варианты отчетов, которые были созданы на этапе разработки отчета в конфигураторе.
  • По умолчанию виден в панелях отчетов всех пользователей
    		•  Значение по умолчанию флажка "Видимость" для всех пользователей в панелях отчетов.

    Команды:

  • Просмотр
    		•  Открыть отчет для формирования и просмотра.
  • Разместить в разделах
    		•  Разместить несколько вариантов отчетов в разделах программы.
  • Сбросить настройки пользователей
    		•  Сбросить пользовательские настройки в панелях отчетов выбранных вариантов отчетов.

     

    См. также: Вариант отчета, Общая информация о формах.
