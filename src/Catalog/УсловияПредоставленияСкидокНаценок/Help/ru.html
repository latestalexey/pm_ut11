Справочник "Условия предоставления скидок (наценок)"

Справочник "Условия предоставления скидок (наценок)" предназначен для хранения списка тех условий , при которых может быть назначена скидка (наценка). Скидка(наценка) может быть назначена при соблюдении одного или нескольких условий.

Условия, по которым может быть назначена скидка( наценка) разделяются на следующие группы (типы условий).

  • За разовый объем продажи.
  • За накопленный объем продаж.
  • За разовую продажу
  • За первую продажу партнеру.
  • За время продажи.
  • За график оплаты.
  • За форму оплаты.

Для каждого типа условий могут быть заданы свои параметры. Например, при оформлении условия "За разовый объем продажи" указывается конкретное количество товара, при котором может быть предоставлена скидка и те товары (сегмент номенклатуры), количество которых будет учитываться при скидке за разовый объем продажи.

В программе также предусмотрены дополнительные условия, которые обычно используются совместно с другими условиями. Например, условие "Ограничение по группе пользователей" позволяет ограничить список тех пользователей, которые будут иметь возможность предоставлять конкретную скидку при выполнении всех тех условий, которые указаны для скидки. Условие "Карта лояльности не зарегистрирована" используется для проверки наличия у клиента карты лояльности.
