Справочник "Контактные лица партнеров"

Справочник "Контактные лица партнеров" предназначен для ведения списка контактных лиц с помощью которых осуществляется связь с партнерами торгового предприятия.

Одно контактное лицо может быть связано с нескольким партнерами. В этом случае в справочник "Контактные лица партнеров" вводится две записи с одним и тем же контактным лицом, но с указанием различных партнеров. В карточке контактного лица регистрируется момент начала связи с партнером. Если контактное лицо перестает работать с данным партнером, то заполняется информация о дате окончания связи с партнером.

Для контактного лица указывается та роль, которое он играет с точки зрения работы с партнером. Список возможных ролей контактных лиц партнеров хранится в отдельном справочнике "Роли контактных лиц партнеров". В общем случае одно контактное лицо может выполнять несколько ролей с точки зрения взаимодействия с партнером.

Важно. Контактные лица партнеров используются при оформлении сделок с клиентами. В рамках сделок с клиентами контактные лица могут выполнять роли, которые регистрируются непосредственно в форме сделки.

На закладке "Адреса и телефоны" указывается информация о том , как связаться с контактным лицом. Непосредственно из карточки контактного лица можно оформить взаимодействие с партнером: написать сообщение по электронной почте, позвонить и т.д. При оформлении взаимодействия автоматически используется та контактная информация, которая указана для партнера.

Пример. При оформлении звонка контактному лицу партнера автоматически заполняется информация о телефоне контактного лица, которая указана в его карточке.

Информация о паспортных данных контактного лица (дата рождения, адрес прописки, удостоверение личности и т.д.) заполняется в справочнике "Физические лица". В карточке контактного лица указывается то физическое лицо, которое соответствует контактному лицу партнера. Связь между контактными лицами партнеров может быть установлен путем установки связи между физическими лицами, которые соответствуют контактным лицам партнеров. Информация о связях между контактными лицами используется при анализе окружения сделки с клиентом.

Для контактных лиц партнеров может быть добавлен произвольный список дополнительных свойств и реквизитов. Для добавления дополнительных свойств и реквизитов используется механизм работы с наборами свойств. В списке наборов свойств для контактных лиц партнеров используется набор свойств "Контактные лица партнеров".
